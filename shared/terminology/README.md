# 术语词汇表

A 股 ECM 业务常用术语的统一说法。目的是减少 skill 之间、skill 与用户之间因为术语不一致产生的误解。

## 命名规范

文件名用 kebab-case，英文：

```
a-share-listing-terms.md
due-diligence-terms.md
corporate-governance-terms.md
```

## 每个术语条目的建议格式

```markdown
### 保荐机构

- **英文**：sponsor
- **别名 / 简称**：主承销商（在承销语境下）
- **定义**：......
- **相关规则**：《证券发行上市保荐业务管理办法》
- **常见误用**：不等同于财务顾问（financial advisor），在 A 股语境下二者职责严格区分
```
