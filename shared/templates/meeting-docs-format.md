# 会议文件排版规范（Single Source of Truth）

本文件是**唯一权威**的中国公司（含拟 IPO 公司和上市公司）**董事会、监事会、股东会**相关文件的文档结构与起草规范；历史文件或客户口语中的"股东大会"按"股东会"处理，并在必要时说明旧称。

本规范既是 `ecm-draft:meeting-docs` 批量生成会议文件的依据，也是 `ecm-qc:meeting-docs-review` 和 `ecm-qc:shareholders-meeting-witness` 反向审查的参照坐标。

---

## 适用范围

涵盖以下三类会议的成套文件：

| 会议 | 法规依据（主要） | 本规范范围 |
|------|----------------|----------|
| 股东会（有限责任公司 / 股份有限公司 / 上市公司；历史文件可称"股东大会"） | 《公司法》（2024）、《上市公司股东会规则》（2025）、《公司章程》 | 通知、议案、决议、记录、签到表、投票表决细则、见证意见所需的附件 |
| 董事会 | 《公司法》（2024）、《公司章程》、证监会 / 交易所信披要求 | 通知、议案、决议、记录、附表 |
| 监事会 | 《公司法》（2024）、《公司章程》 | 通知、议案、决议、记录 |

> **2024 新《公司法》要点**：取消监事会为选配机构（可由审计委员会替代），但设监事会的公司仍按本规范。

## 一、会议文件清单（一次会议的全集）

**股东会**的完整会议文件集包括：

1. **召集通知**（公告 / 书面送达）
2. **议案文本**（每项议案独立一份）
3. **授权委托书模板**（股份公司 / 上市公司必备）
4. **代理投票计票细则**（上市公司）
5. **决议**（签署后的决议文件）
6. **会议记录**（主持人、唱票人、监票人、记录人签字）
7. **签到表**（出席股东 / 代理人签到 + 出席股数 + 占总股数比例）
8. **现场会议人员名单**（中小投资者单独计票时必备）
9. **表决统计表**（逐项议案同意 / 反对 / 弃权票数、股数、占比）
10. **公告公示材料**（上市公司，股东会决议公告、法律意见书；历史公告标题可保留"股东大会"）

**董事会** / **监事会**简化版（不含第 3/4/7/8/10 项）。

### 2.3 通知期限（硬性要求）

| 会议类型 | 最少通知期 | 法规依据 |
|---------|:---------:|---------|
| 上市公司年度股东会 | **现场会议前 20 日** | 《上市公司股东会规则》第 X 条 |
| 上市公司临时股东会 | **现场会议前 15 日** | 同上 |
| 非上市股份公司股东会 | 20 日 | 《公司法》第 112 条 |
| 有限责任公司股东会 | 章程约定；无约定 15 日 | 《公司法》第 60 条 |
| 董事会 | 章程约定；无约定至少 10 日 | 《公司法》第 119 条 |

**meeting-docs 生成通知时必须做日期校验**：会议日期 - 通知日期 ≥ 对应最少通知期。否则在输出顶部打"⚠️ 通知期限不足"标记。

## 三、议案

### 议案通用格式

```
议案 N：关于 xxx 的议案

各位股东 / 各位董事 / 各位监事：

一、议案背景
（事实背景、决策必要性）

二、议案内容
（具体方案）

三、决策依据
（公司章程条款、法律法规引用）

四、本议案需经股东会 / 董事会 / 监事会审议批准；表决方式：{普通决议 / 特别决议}；{是否涉及关联交易/回避表决}；{是否中小投资者单独计票}。

特此议案，请审议。

XX 公司 董事会 / 监事会 / 召集人（盖章）
YYYY 年 MM 月 DD 日
```

### 特别决议识别

以下议案**必须**标注"特别决议"（需 2/3 以上表决通过）：

- 公司合并、分立、解散、清算、变更公司形式
- 修改公司章程
- 增加、减少注册资本
- 发行公司债券
- 上市公司《公司章程》关于表决权、董事会席位的修改

### 关联股东 / 董事回避识别

涉及以下情形的议案**必须**在议案末尾注明"关联股东 / 董事需回避表决"：

- 与关联方的交易议案
- 由关联方担任董事 / 监事 / 高管的提名议案
- 对关联方提供担保或借款议案
- 上市公司章程约定的其他关联交易事项

## 十、meeting-docs 的"一致性校验"

meeting-docs 批量生成会议文件时，必须对各文件做交叉校验：

| 校验点 | 期望 |
|-------|------|
| 通知的"股权登记日" = 签到表对应的"持股基准日" | 一致 |
| 通知的"议案清单" = 议案文本的全集 = 决议审议的议案 = 表决统计表的议案 | 完全一致（编号、名称） |
| 会议开始时间 = 记录开头的时间 | 一致 |
| 议案的"特别决议"标注 = 决议对应议案的"已获得 2/3 以上通过"表述 | 一致 |
| 议案的"关联股东回避"标注 = 决议对应议案的回避说明 | 一致 |
| 总股本 / 总表决权股数 在通知、签到表、决议中保持一致 | 一致 |

输出时在最后附一份**一致性校验报告**，若存在不一致，明确指出并要求律师人工确认。

## 十一、常见踩坑

1. **通知期限按"日历日"还是"工作日"算**：除法规明确写"工作日"外，默认按日历日；meeting-docs 校验时按日历日计算
2. **网络投票与现场投票合并计票**：上市公司决议须将两者合并，并单独标注"网络投票股东……人、……股"
3. **关联股东回避后表决权基数变化**：决议中"占出席股东所持有表决权股份总数"的分母应扣除回避股东的表决权股份，不得用总表决权股数
4. **特别决议未达 2/3**：决议必须明确写"未获通过"，不得模糊为"搁置下次审议"
5. **中小投资者单独计票与合并计票并存**：上市公司部分议案要求单独计票，meeting-docs 在决议里必须同时列出"整体表决结果"和"中小投资者表决结果"
6. **签到表的"占总股本比例"与"占出席股东表决权股份总数比例"混淆**：签到表用前者（基数是总股本），决议用后者（基数是实际出席股东的表决权股份）
7. **董事以通讯 / 视频方式参加会议未在记录中体现**：必须在出席方式列标注，决议开头说明"本次会议以现场与通讯表决结合方式召开"
