# 法规条文节选

跨 skill 复用的法律法规、监管规则、自律规则节选。

## 命名规范

```
<规则名>-<版本年份>.md
```

例：
- `公司法-2024.md`
- `上市公司股东会规则-2025.md`
- `律师事务所从事证券法律业务管理办法-2019.md`

## 每份文件必填元信息

文件顶部用 HTML 注释或 frontmatter 放元信息：

```markdown
---
规则名称: 《中华人民共和国公司法》
发布机构: 全国人大常委会
当前有效版本: 2024 年 7 月 1 日施行
本文节选日期: 2026-04-24
节选范围: 仅摘录第 4 章（股东会）
原文链接: https://example.gov.cn/...
---

# 《中华人民共和国公司法》

## 第 4 章 股东会

### 第 109 条
...
```

## 为什么抽到这里

A 股 ECM skill 大量引用同一批规则条文。如果每个 skill 都维护自己的节选版本，规则更新后维护成本极高。集中存放：

- 规则更新时一次改动，所有引用自动生效
- 可以用 markdown anchor 精确跳转（例：`公司法-2024.md#第109条`）
- 方便人工校对原文链接
