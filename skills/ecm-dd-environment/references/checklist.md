# ecm-dd-environment · 核查要点清单

本清单覆盖环境保护、安全生产和产品质量 / 技术标准三大块。项目现场多个生产
基地 / 多条产品线时，按基地 / 产品线拆分台账逐项核查。

## 一、环境保护

### A. 建设项目环评与验收（"未批先建 / 未验先投"红线）

- [ ] **所有**现役建设项目（含历次扩建 / 改建 / 搬迁）的环评文件（报告书 /
      报告表 / 登记表）齐备，批复机关、批复文号清晰
- [ ] 建设项目实际生产工艺 / 产能 / 选址与环评批复一致（如有变动是否履行
      重新报批或变更程序）
- [ ] **环保"三同时"**落实：防治污染设施与主体工程**同时设计、同时施工、同时投产**
- [ ] 竣工环保验收（2017 年后企业自主验收，须规范编制验收报告、在国家平台公示）
- [ ] 未批先建 / 未验先投的历史情形及整改情况
- [ ] 拟上市公司报告期内新增重大项目的环评手续同步性

### B. 排污许可

- [ ] 依据《排污许可管理条例》是否属于重点 / 简化管理单位
- [ ] 排污许可证的有效期、许可的污染物因子、许可排放浓度 / 排放量
- [ ] 许可证与实际排放（因子 / 浓度 / 总量）匹配性
- [ ] 执行报告（年报 / 季报 / 月报）按期提交
- [ ] 排污口规范化设置

### C. 污染物达标与在线监测

- [ ] 主要大气污染物（SO₂ / NOx / 颗粒物 / VOCs 等）达标情况
- [ ] 主要水污染物（COD / 氨氮 / 总磷 / 重金属等）达标情况
- [ ] 土壤污染风险（尤其历史遗留场地、搬迁后涉及的重点行业）
- [ ] 噪声 / 恶臭达标情况（社区敏感项目重点关注）
- [ ] 电磁辐射 / 电离辐射（如适用）
- [ ] 重点排污单位安装在线监测并联网

### D. 固体废物与危险废物

- [ ] 一般固废的产生 / 贮存 / 转运 / 处置去向
- [ ] 危险废物管理计划、转移联单、贮存设施合规（含暂存点规范化）
- [ ] 危险废物处置单位的经营资质验证
- [ ] 危废自行利用 / 处置的许可

### E. 碳排放与新兴监管

- [ ] 是否纳入全国碳市场重点排放单位
- [ ] 温室气体排放核查报告、配额履约情况
- [ ] ESG / 可持续披露（板块要求不同：港股 / 科创板 / 主板 ESG 要求等）

### F. 应急管理与历史事件

- [ ] 突发环境事件应急预案（编制 / 备案 / 演练）
- [ ] 环境风险评估报告
- [ ] 报告期内突发环境事件历史及处置

### G. 搬迁 / 退城入园 / 产业结构调整

- [ ] 因地方产业政策 / 城市规划调整导致的强制搬迁的程序合法性
- [ ] 涉及募投的"异地搬迁"项目的新址环评

## 二、安全生产

### A. 安全生产许可证（限高危行业）

- [ ] 是否属于危险化学品生产 / 经营 / 使用、矿山、建筑施工、烟花爆竹、
      民爆、金属冶炼、涉氨制冷、建筑施工等需许可行业
- [ ] 安全生产许可证有效期、许可范围与实际一致
- [ ] 危险化学品登记证（如适用）
- [ ] 特种行业经营许可（如民用爆炸物品）

### B. 消防验收 / 备案

- [ ] 厂房 / 办公楼 / 仓库的消防验收或备案意见
- [ ] 与实际使用情况一致（变更用途须重新办理）
- [ ] 消防安全重点单位的年度消防安全自查

### C. 特种设备

- [ ] 特种设备（压力容器 / 压力管道 / 电梯 / 起重机 / 场内专用机动车辆等）
      使用登记证
- [ ] 定期检验证书
- [ ] 作业人员资格证

### D. 职业病危害

- [ ] 职业病危害预评价 / 控制效果评价 / 现状评价
- [ ] 职业病危害项目申报（卫生健康部门）
- [ ] 职业病危害因素检测报告（年度）
- [ ] 个人防护用品发放记录、职业健康监护档案

### E. 重大危险源与应急预案

- [ ] 重大危险源辨识与备案
- [ ] 生产安全事故应急预案备案回执
- [ ] 近 3 年应急演练记录

### F. 生产事故与处罚

- [ ] 报告期内一般 / 较大 / 重大事故清单及处理
- [ ] 事故调查报告结论
- [ ] 处罚决定书（应急管理 / 卫生健康部门）

## 三、产品质量与技术标准

### A. 强制性许可 / 认证

- [ ] **强制性产品认证 CCC**（如适用：电子电器、机动车部件、玩具等）
- [ ] **食品生产许可 SC** / 食品经营许可
- [ ] **药品 GMP / 药品经营质量管理 GSP**（药品行业）
- [ ] **医疗器械注册证 / 生产许可 / 经营许可**（医疗器械行业）
- [ ] **化妆品注册 / 备案 / 生产许可**（化妆品行业）
- [ ] **农药登记证 / 兽药产品批准文号 / 肥料登记证**
- [ ] **国家强制计量许可 CMC / 检测实验室 CNAS**（仪器行业）
- [ ] **压力容器 / 锅炉等制造许可**（特种设备行业）

### B. 体系认证 / 自愿认证

- [ ] ISO 9001 质量管理体系
- [ ] ISO 14001 环境管理体系
- [ ] ISO 45001 职业健康安全体系
- [ ] IATF 16949（汽车）/ ISO 13485（医疗器械）等行业体系
- [ ] 各认证的有效期、范围、复审情况

### C. 技术标准

- [ ] 主要产品适用的国家 / 行业 / 地方 / 团体标准
- [ ] 企业标准备案（如适用）
- [ ] 涉及强制性国家标准 GB 的执行情况
- [ ] 参与行业标准制定（如果有，作为正面信息引用）

### D. 监督抽查与召回

- [ ] 报告期内国家 / 省级 / 市场监管局的产品质量监督抽查结果
- [ ] 抽查不合格时的整改措施和后续抽查结果
- [ ] 产品召回情况（主动 / 被动 / 召回原因 / 召回率）
- [ ] 重大质量事故（涉及消费者人身 / 财产损失、舆情事件）

### E. 知识产权与质量（交叉）

- [ ] 主要产品的专利权属（本 skill 仅标记，权属核查归 `ecm-dd-assets`，BATCH-03）

## 七、特别关注事项

- **"环保督察"常态化**：中央环保督察 / "回头看" / 地方督察过程中暴露的问题，
  应单独评估是否涉及发行人及整改情况
- **"搬迁入园"专项**：化工园区政策导致的搬迁、退城入园，是否构成对募投计划的
  冲突
- **VOCs / 臭氧 / 新污染物**：2021 年后监管对 VOCs、新污染物（微塑料、抗生素）
  关注上升
- **头部行业 ESG 披露**：主板 / 港股 / 科创板特定行业发行人有 ESG 强制披露要求
- **跨境业务的合规辐射**：出口欧美的产品涉及 CE / UL / FDA 等外域合规，虽不
  构成境内发行直接障碍，但需关注对收入持续性的影响
- **关联交易中的 HSE 责任转移**：代工 / 委托加工关系中，环保 / 安全责任如何
  划分（与 `ecm-dd-related-party` 协同核查）
- **"自主验收"合规性**：2017 年《建设项目环境保护管理条例》改革后自主验收，
  仍有专家验收、公示、备案等环节；简化不等于免除
