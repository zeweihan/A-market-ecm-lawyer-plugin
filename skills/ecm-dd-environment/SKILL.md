# ecm-dd-environment

## 定位与边界

本 skill **负责**：

- 环保合规（环评、三同时、排污许可、环保验收、应急预案、污染物达标、固危废
  处置、碳排放报告）
- 安全生产合规（安全生产许可证、特种设备、职业病危害、消防验收、生产事故、
  重大危险源）
- 产品质量与技术标准合规（强制性认证、产品生产许可、体系认证、监督抽查、
  重大质量事故、产品召回）
- 报告期内环保 / 安全生产 / 产品质量领域的**行政处罚**及是否构成"重大违法"
- 建设中 / 新扩建项目的环保 / 安全手续合规（**未批先建、未验先投是高压线**）
- 主管部门出具的合规证明的时效和覆盖完整性
- 出具《编报规则第 12 号》第 14 章对应的 DD Memo

本 skill **不负责**：

- 业务经营类普通许可证（归 `ecm-dd-business`；本 skill 仅涉及
  HSE / 质量类特殊许可）
- 募投项目本身的备案 / 核准及用地（归 `ecm-dd-fundraising`；但募投项目的
  环评手续仍在本 skill 核查）
- 行政处罚决定书的送达 / 复议 / 诉讼程序合法性（归 `ecm-dd-litigation`；
  本 skill 仅判断是否构成重大违法）
- 劳动用工合规（归 `ecm-dd-compliance`；本 skill 只管职业病危害和安全生产）
- 主要财产不动产权属（归 `ecm-dd-assets`；本 skill 仅关注不动产
  是否通过消防验收、环评验收）
- 知识产权 / 专利 / 软著（归 `ecm-dd-assets`）

## 免责声明

本 skill 产出的 DD Memo 为工作底稿，不构成最终法律意见。涉及具体技术 / 工程 /
质量事实的判断建议参考第三方专业机构报告。完整免责声明见
[DISCLAIMER.md](../../DISCLAIMER.md)。

## 本 skill 的实务加固点

- **许可与项目制核查**：环评、排污许可、安评、能评、消防、职业健康、产品质量认证需按项目和经营场所分别核对。
- **处罚闭环**：行政处罚必须核查违法事实、罚款金额、整改完成、复查证明和是否构成重大违法。
- **募投联动**：募投项目涉及新增产能、土地、能耗或危化品时，应单列前置审批和实施障碍。
- **高风险触发器**：未批先建、无证排污、重大安全事故、核心产品质量处罚、整改未完成，应列高风险。

## 核心工作流（五步）

1. **生成核查要点清单 + 生产基地 / 产品线档案**：照 [references/checklist.md](./references/checklist.md)
   生成清单；同时按"生产基地 / 重大生产设施 / 主要产品"三维度建立档案，定位适用
   的环保 / 安全 / 质量许可要求
2. **文件审阅与标注**：对照每个生产基地 / 产品线的必备手续清单逐项审阅，
   重点识别"未批先建、未验先投、证照到期未续、排污许可超范围"等典型风险
3. **走访与访谈建议**（可选）：对现场依赖度高的环保 / 安全事项，在 Memo 里提示
   建议的现场走访 / 主管部门访谈项，以便正式 DD 阶段进一步核实
4. **风险分级**
5. **结论与建议**

## 统一风险分级口径

| 级别 | 定义 | 本 skill 场景的典型例子 |
|------|------|------------------------|
| **高（红）** | 足以构成发行上市实质性障碍 | 重大环保 / 安全事故（人员伤亡、重大污染、生态损害）；受到责令停产停业、吊销许可证、较大数额罚款构成"重大违法"；未批先建且未办理补办；未验先投且关键环保设施未建成；核心产品未取得强制性许可 / 认证而持续销售；主管部门出具否定性评价；重点排污单位连续 / 严重超标 |
| **中（黄）** | 程序 / 文件瑕疵可整改 | 个别小额行政处罚已履行；环保验收报告尚未取得但已申请；排污许可证临近到期未续；应急预案未及时备案；部分体系认证过期；特种设备登记滞后；个别子公司 / 厂区环评批复与实际工艺轻微不符 |
| **低（蓝）** | 形式瑕疵 | 证照印章不清晰；备案回执日期与批复日期存在 minor 差异；合规证明出具时间距申报节点偏早但仍在常规覆盖期；认证证书地址与营业执照存在格式差异 |

**特别约束**：
- 《首次公开发行股票注册管理办法》第 12 条、《上市公司证券发行注册管理办法》等
  将"最近 36 个月内行政处罚情节严重"作为发行条件的消极条件。本 skill 识别出的
  任何行政处罚都应从"是否情节严重 / 是否构成重大违法"角度评估
- "未批先建 / 未验先投"是**红线**，即使主管部门补办了手续，也要在风险分级时
  评估是否仍需披露

## 常见误用 / FAQ

- **Q：发行人不属于"高污染 / 高风险"行业，本 skill 还要触发吗？**
  A：要。即使是轻资产 / 办公类行业，仍涉及消防验收、职业卫生评价（工作场所）、
  体系认证等基础合规；只是台账里只需填写实际适用项目，无需强行覆盖全部门类。

- **Q："未验先投"已经补办验收手续的，还要写高风险吗？**
  A：需要评估是否曾被主管部门处罚、是否触及"重大违法"阈值。若仅形式补办、
  未被处罚，可评为中风险并在披露中说明；若曾被处罚且金额 / 情节较重，仍为高。

- **Q：跨境供应链 / 境外生产基地的环保合规怎么办？**
  A：本 skill 可涵盖"境内母公司对境外子公司 HSE 合规的监督职责"，但具体适用
  境外法律的深度核查建议由当地律师配合；在本 Memo 中标注适用法域并引用当地
  律师意见。
