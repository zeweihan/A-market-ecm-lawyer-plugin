# ecm-dd-compliance · 核查要点清单

本 skill 是编报规则第 17 章"其他合规事项"的兜底核查。**先做分流**——把用户
请求涉及的已归属于其他 DD 章节的事项剥离后，本 skill 按下列 7 大议题组
逐议题展开核查。每一议题在本项目不适用时，必须说明"N/A + 理由"。

## 一、劳动用工合规

### A. 劳动合同

- [ ] 劳动合同签订率（全口径员工，含正式 / 非全日制 / 实习 / 外包等）
- [ ] 劳动合同期限结构（固定期限 / 无固定期限 / 以完成一定任务）
- [ ] 劳动合同续签情况与"连续两次固定期限后订立无固定期限"的法定要求
- [ ] 集体合同（是否签订、涵盖内容、公示 / 备案）
- [ ] 试用期设置合规（期限不超过法定上限、工资不低于本单位相同岗位最低
      工资或劳动合同约定工资的 80%）

### B. 社会保险

- [ ] 五险覆盖：养老、医疗、工伤、失业、生育
- [ ] **缴纳基数**：是否按照职工工资总额（含绩效 / 奖金等）作为基数
- [ ] **缴纳比例**：按当地现行比例
- [ ] **覆盖率**：全体全职员工是否 100% 参保（含试用期）
- [ ] 补充养老（企业年金）/ 补充医疗（如有）
- [ ] 跨地区用工的参保地选择
- [ ] 报告期各年度欠缴 / 漏缴情况及补缴
- [ ] 主管部门出具的**社保合规证明**或**不追究的书面说明**（三件套）

### C. 住房公积金

- [ ] 覆盖率（全体员工是否 100% 缴纳）
- [ ] 缴存基数
- [ ] 缴存比例
- [ ] 补充公积金（如有）
- [ ] 报告期的欠缴 / 漏缴情况及补缴
- [ ] 公积金管理中心出具的合规证明 / 不追究说明

### D. 劳务派遣

- [ ] **劳务派遣比例**（不得超过用工总量的 10%）
- [ ] 派遣岗位是否限于"临时性、辅助性、替代性"
- [ ] 劳务派遣单位的经营许可证
- [ ] 劳务派遣协议合规
- [ ] 被派遣劳动者的"同工同酬"

### E. 非全日制 / 实习 / 外包

- [ ] 非全日制用工：日工作时间 / 周工作时间限制
- [ ] 在校生实习：三方协议 / 实习管理
- [ ] 业务外包与劳务派遣的区分（避免"真派遣假外包"）

### F. 工资与工时

- [ ] 工资支付周期、最低工资遵守情况
- [ ] 加班工资计算基数与支付
- [ ] 综合计算工时 / 不定时工作制的审批
- [ ] 高温津贴 / 夜班津贴 / 其他法定津贴

### G. 工会与职代会

- [ ] 是否设立工会 / 工会登记证
- [ ] 工会经费拨缴（2% 左右，视地方政策）
- [ ] 职工代表大会设置及重大事项审议
- [ ] 民主管理制度（对国企 / 上市公司有额外要求）

### H. 特殊人员保护

- [ ] 女职工保护（三期、工资福利、特殊岗位禁忌）
- [ ] 未成年工保护（是否使用、如有则合规性）
- [ ] 残疾人就业保障金缴纳 / 安置合规

### I. 保密 / 竞业 / 服务期

- [ ] 保密协议（商业秘密范围、期限）
- [ ] 竞业限制协议（人员范围限于高级管理人员 / 高级技术人员 / 其他负有
      保密义务的人员；期限不超过 2 年；合理的经济补偿）
- [ ] 服务期协议（是否有专项培训，违约金合理）

## 二、海关合规

- [ ] 进出口经营权（已备案或通过"互联网 + 海关"注册）
- [ ] 海关企业信用管理等级（高级认证 AEO / 一般认证 / 一般信用 / 失信）
- [ ] 加工贸易手册（如涉及加工贸易）
- [ ] 保税物流中心 / 保税仓 / 出口监管仓（如涉及）
- [ ] 报关事项的归类、估价、原产地合规
- [ ] 进出口许可 / 配额 / 关税配额（如涉及）
- [ ] 进出口**大宗敏感商品**的特别许可（两用物项、战略物资、文物、濒危物种等）
- [ ] 报告期内海关行政处罚情况（事实登记由 `ecm-dd-litigation` 处理，本 skill
      评估是否涉及重大违法)

## 三、外汇合规

- [ ] **FDI 外汇登记**（外商投资企业）
- [ ] **ODI 外汇登记 / 境外投资外汇登记**（境内主体对外投资）
- [ ] **外债登记**（跨境借款）
- [ ] **37 号文登记**（境内居民通过境外特殊目的公司的返程投资）
- [ ] 货物贸易外汇监测（分类等级 A / B / C）
- [ ] 服务贸易 / 非贸易项下付汇合规
- [ ] 跨境资本项下业务（权益变动、利润分配、对境外借款）合规
- [ ] 跨境人民币业务合规
- [ ] 报告期内外汇行政处罚（登记由 `ecm-dd-litigation`，本 skill 评估重大违法）

## 四、反垄断 / 反不正当竞争 / 商业秘密

- [ ] **经营者集中申报**：报告期内是否达到申报标准而申报 / 被反垄断执法
      机构立案
  - 标准依据：《国务院关于经营者集中申报标准的规定》
  - 历史交易是否"未申报先实施"
- [ ] 市场支配地位情形下的合规审视（高市场份额行业）
- [ ] 垄断协议（横向 / 纵向）合规（转售价格维持、分销协议等）
- [ ] 不正当竞争合规（虚假宣传、商业诋毁、侵犯商业秘密等）
- [ ] **商业秘密保护体系**（机密清单、访问权限、泄密应急、员工保密承诺）
- [ ] 公平竞争审查（国企 / 涉及政府补贴情形）
- [ ] 反垄断合规管理制度 / 培训记录

## 五、反洗钱 / 反恐怖融资 / 反腐败

- [ ] **反洗钱合规**（金融行业主体适用；非金融行业视业务判断）
  - 大额交易 / 可疑交易报告
  - 客户身份识别（KYC）
  - 受益所有人识别
- [ ] **反商业贿赂合规**
  - 反商业贿赂制度 / 培训 / 审计机制
  - 回扣 / 佣金 / 礼品费用支付的合规性
  - 经销商 / 代理商合规条款
  - 高风险行业（医药、基建、政府采购、能源等）重点关注
- [ ] **反恐怖融资**（适用主体）
- [ ] **涉外反腐败**（FCPA、UK Bribery Act 等——如有境外业务或境外上市规划）
- [ ] 《企业合规管理体系》ISO 19600 / 37301（如有）
- [ ] 国有股东的廉洁从业制度（国企参股 / 国资控股时）

## 六、数据合规

- [ ] **网络安全等级保护备案**（二级 / 三级）
- [ ] **关键信息基础设施**（CII）身份识别与合规
- [ ] **个人信息保护**：
  - 隐私政策 / 用户协议合规（含 APP 用户信息保护专项）
  - 个人信息处理的合法性基础（同意 / 合同必需 / 法定义务等）
  - 敏感个人信息单独同意 / 儿童个人信息特别规则
  - 自动化决策（算法）告知 / 拒绝机制
  - 个人信息处理者的合规义务（PIA 报告、定期审计）
- [ ] **数据出境**合规：
  - 数据出境安全评估（国家网信办批准）
  - 标准合同备案
  - 个人信息保护认证
- [ ] **数据分类分级**体系建设
- [ ] **APP 移动应用**合规（工信部专项整治口径）
- [ ] **爬虫 / 数据采集**合规（技术合规与法律合规）
- [ ] 数据跨境业务的对等措施 / 安全责任
- [ ] 重大 / 一般网络安全事件报告机制

## 七、出入境管理

## 十一、特别关注事项

- **"应缴尽缴"vs"地方惯例"**：社保 / 公积金在部分地区有"地方宽限"惯例，
  但 A 股发行审核对此口径趋严。没有主管部门书面"不追究"的情形下，应按
  "应缴"保守判断
- **"未申报先实施"经营者集中**：监管越来越重视历史遗留的反垄断申报瑕疵，
  企业可主动补报，但可能面临行政处罚（由 `ecm-dd-litigation` 共同核查）
- **数据合规高变化性**：数据合规法律法规 2021 年起变化剧烈，每 6 个月应
  评估最新审核口径（如 2024 年《促进和规范数据跨境流动规定》的落实情况）
- **"劳务派遣 + 外包"嵌套**：用工形态模糊时，容易被认定为"真派遣假外包"，
  审核中需要看业务实质
- **"合规证明"的三要素**：出具主体适格、覆盖期间完整、出具时间新鲜
  （一般 6 个月内）
- **"员工食堂"等小微合规**：虽小但属于兜底章典型涉及事项——食品生产许可
  不要求（自办食堂属于"内部供餐"），但需要基本的食品安全与消防合规
- **"境内境外"合规口径衔接**：境外业务的合规不等同境内合规，本 skill 仅
  做披露登记，深入合规由当地律师意见承接
