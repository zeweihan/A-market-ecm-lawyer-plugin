# ecm-dd-independence · 核查要点清单

本 skill 在项目中生成的 DD Memo 应照本清单逐条核查。法规依据以 `shared/regulations/` 下条文为准。

独立性核查以**五独立**为主干，每一项都要落实到**事实 + 证据 + 上游 DD skill 的数据来源**。

## 一、资产完整

### A. 经营性资产的完整性

- [ ] 土地使用权、房产所有权登记在发行人 / 子公司名下（由 `ecm-dd-assets` 提供清单）
- [ ] 核心生产设备权属在发行人 / 子公司（由 `ecm-dd-assets` 提供清单）
- [ ] 核心知识产权（专利 / 商标 / 软著 / 非专利技术）权属在发行人 / 子公司
- [ ] 不存在核心资产仍挂靠控股股东 / 实控人 / 关联方名下情形
- [ ] 若存在：资产归集 / 剥离方案、完成时点、工商 / 不动产 / IP 登记变更完成情况

### B. 资产配置的完整性

- [ ] 具备完整的采购 / 生产 / 销售 / 研发链条所需资产
- [ ] 无重要资产"无偿占用"控股股东 / 关联方（如共用无偿办公场所、无偿使用核心设备）
- [ ] 对外租赁控股股东 / 关联方的场地 / 设备的，签署正式租赁合同，价格公允

### C. 资产瑕疵与纠纷

- [ ] 不存在核心资产的重大权属纠纷
- [ ] 不存在核心资产被司法查封 / 行政查封
- [ ] 不存在核心资产被抵押 / 质押后无法正常使用

## 二、业务独立

### A. 业务完整性

- [ ] 发行人具有独立、完整的业务体系（采购 / 生产 / 销售 / 服务链条自主）
- [ ] 无需依赖控股股东 / 实控人 / 关联方即可完成主要经营活动
- [ ] 自有品牌、渠道、客户体系，不依赖关联方品牌 / 渠道
- [ ] 核心技术自主可控，不依赖关联方许可 / 授权（或若授权：价格公允 + 长期稳定 + 不可撤销）

### B. 关联销售 / 采购的重大依赖评价

- [ ] 关联销售占营业收入比例（报告期内逐年分析）
- [ ] 关联采购占营业成本比例
- [ ] 是否构成重大依赖（综合考虑占比、可替代性、行业惯例、交易公允性）
- [ ] 替代方案：如果关联方停止合作，发行人能否继续独立经营

### C. 同业竞争

- [ ] 不存在与控股股东 / 实控人 / 关联方的重大不利影响同业竞争（由 `ecm-dd-related-party` 提供结论）
- [ ] 若存在：已通过注销 / 转让 / 重组等方式解决

### D. 市场能力

- [ ] 直接面向市场独立销售的能力
- [ ] 自主定价能力（非关联方强制定价）
- [ ] 自主获取客户 / 订单的能力

## 三、人员独立

### A. 高管独立（核心）

- [ ] 董事长 / 总经理 / 副总经理 / 财务负责人 / 董事会秘书不在控股股东 / 实控人及其控制的
      其他企业担任除董事、监事以外的职务
- [ ] 上述人员薪酬仅由发行人支付，不存在同时领取控股股东 / 关联方薪酬情形
- [ ] 上述人员社保 / 公积金由发行人缴纳

### B. 董事 / 监事的独立

- [ ] 不同时担任控股股东 / 实控人单位高管（董事 / 监事允许兼任，但不得兼任经营性高管）
- [ ] 独立董事满足独立性标准（不与发行人 / 控股股东 / 实控人存在关联关系）

### C. 员工独立

- [ ] 核心员工（研发 / 销售 / 生产 / 财务）劳动关系在发行人
- [ ] 社保、公积金、个税由发行人代扣代缴
- [ ] 考勤、绩效、晋升独立管理
- [ ] 不存在"一套人马两块牌子"的情形

### D. 历史期人员混同的整改

- [ ] 如历史期存在劳动关系在关联方、工作实际在发行人的情形，已完成劳动关系转移
- [ ] 双重劳动关系已解除
- [ ] 竞业禁止 / 保密协议已重签

## 四、财务独立

### A. 财务制度独立

- [ ] 独立的财务管理制度（资金管理、预算管理、会计核算、财务报告）
- [ ] 独立的财务部门和财务人员（财务负责人不在关联方兼职）
- [ ] 不使用控股股东 / 关联方财务共享系统

### B. 独立核算

- [ ] 独立的会计核算体系（独立建账、独立做账、独立出报表）
- [ ] 独立的成本费用核算
- [ ] 财务数据不与关联方混同

### C. 独立银行账户与资金

- [ ] 拥有独立的基本账户和其他银行结算账户
- [ ] 不与控股股东 / 实控人 / 关联方共用账户、集中收付款
- [ ] 资金收付独立决策，不存在统一调度

### D. 独立纳税

- [ ] 独立办理税务登记
- [ ] 独立申报纳税、独立开具发票
- [ ] 不与关联方共用税控设备、共用纳税账户

### E. 资金往来合规

- [ ] 与关联方资金往来清晰、独立结算
- [ ] 不存在非经营性资金占用（IPO 红线，零容忍）——由 `ecm-dd-debt` 和 `ecm-dd-related-party` 提供证据
- [ ] 不存在为关联方违规担保（IPO 红线，零容忍）
- [ ] 历史期存在的资金拆借已全部清理并支付合理利息

## 五、机构独立

### A. 组织机构独立

- [ ] 独立的法人治理结构（股东会 / 股东大会、董事会、监事会、经理层）
- [ ] 独立的组织架构图（非与关联方"一套班子"）
- [ ] 内部部门设置完整（财务 / 人事 / 法务 / 销售 / 生产 / 研发 / 采购 / 质控等）
- [ ] 不存在关键部门共用、职能混同

### B. 治理结构独立

- [ ] 三会议事规则独立制定和运行（与 `ecm-dd-charter` 协同）
- [ ] 决策流程独立，不受控股股东 / 实控人不当干预
- [ ] 独立董事制度运行有效（上市公司 / 拟上市公司）
- [ ] 不存在关联方对发行人日常经营决策的不当介入

### C. 物理独立

- [ ] 办公场所独立（或租赁控股股东 / 关联方场地但签署正式租赁合同且价格公允）
- [ ] 通讯 / 网络 / 门禁 / 网络地址等基础设施独立
- [ ] 标识独立（门牌 / Logo / 名称）

## 六、历史独立性瑕疵与整改

- [ ] 历史期是否存在重大独立性瑕疵（资金混同、资产混同、人员混同、业务混同）
- [ ] 瑕疵清理方案：时间表、责任主体、完成标志
- [ ] 控股股东 / 实控人出具的承诺函：不再发生 / 不再重复该等瑕疵
- [ ] 若瑕疵发生在报告期内，需披露充分并说明对持续经营无重大不利影响

## 七、关键文件清单（客户需提供）

- [ ] 五独立自查报告（发行人自行出具）
- [ ] 控股股东、实际控制人关于保持发行人独立性的承诺函
- [ ] 董监高、财务负责人兼职情况说明 + 劳动合同
- [ ] 主要资产权属证书（汇集自 `ecm-dd-assets`）
- [ ] 独立的财务管理制度文件
- [ ] 独立的银行账户清单（与银行签订的开户协议）
- [ ] 独立的税务登记证明、近 3 年纳税证明
- [ ] 独立的组织机构图 + 内部部门职责说明
- [ ] 与关联方签订的关联交易协议（汇集自 `ecm-dd-related-party`）
- [ ] 控股股东与发行人共用场地 / 设施的租赁 / 共享协议
- [ ] 关联资金往来清理证明（汇集自 `ecm-dd-debt` 和 `ecm-dd-related-party`）
- [ ] 历史期独立性瑕疵整改文件（如有）

## 八、特别关注事项

- **五独立复合评估**：单维度无瑕疵但多维度存在小问题，仍可能被认定为独立性不足
- **改制重组企业**：由国企 / 集团公司分拆改制的发行人，独立性问题高发，需重点关注历史脐带
- **家族企业**：实控人直接干预、家族成员兼职、资金调配等问题高发
- **大集团下子公司上市**：集团共享采购、共享销售渠道、品牌使用等问题需逐项清理或规范
- **重大客户 / 供应商为关联方**：需专项评价重大依赖（通常金额超 30% 为警戒线，超 50% 被认定
  为重大依赖的可能性极高）
- **独立性与同业竞争的关联**：同业竞争是独立性的一个维度，但二者核查分工不同——本 skill 做
  综合评价，related-party 做具体事实核查
- **历史遗留非经营性资金占用**：哪怕报告期前已清理，也要披露；零容忍的前提是**当下没有**
- **监管审核高频关注点**：核心资产在实控人名下、核心销售依赖关联方、高管大量兼职控股股东
  单位、财务共享、银行账户混同
