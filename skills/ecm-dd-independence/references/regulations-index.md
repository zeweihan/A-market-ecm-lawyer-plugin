## 审核实践中的"独立性"核心命题（供思考）

- **主营业务是否独立**：能否脱离控股股东 / 关联方独立开展主营业务？
- **资产是否完整**：核心经营性资产是否完整、独立配置于发行人？
- **人员是否独立**：核心管理层和员工是否完全隶属于发行人？
- **财务是否独立**：独立的制度、核算、账户、纳税体系？
- **机构是否独立**：独立的组织架构、决策机制、物理空间？
- **对控股股东 / 关联方是否存在实质重大依赖**（业务 / 客户 / 供应商 / 资金 / 资产 / 技术）？
- **历史期瑕疵是否已彻底清理并形成稳定的独立运营**？
