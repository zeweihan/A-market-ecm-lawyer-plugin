# ecm-dd-establishment · 核查要点清单

## 一、发起人 / 股东资格

- [ ] 发起人人数：股份公司设立时 2 人以上 200 人以下，过半数在中国境内有住所（《公司法》第 94 条）
- [ ] 发起人主体资格（自然人身份证明 / 法人营业执照 + 公司决议 / 非法人组织合法存续证明）
- [ ] 发起人不存在《公司法》禁止性情形（公务员、现役军人等若作为自然人股东）
- [ ] 发起人出资来源合法（自有资金 / 合法借款 / 其他合法来源）

## 二、设立文件

- [ ] 发起人协议 / 出资协议（全体发起人签字、日期合法）
- [ ] 设立时章程（全体发起人签字或盖章，通过创立大会的还需有大会决议记载）
- [ ] 设立决议 / 创立大会决议（股份公司）
- [ ] 验资报告（会计师事务所盖章 + 注册会计师签字）
- [ ] 资产评估报告（如涉及非货币出资）
- [ ] 公司设立登记通知书及首次营业执照
- [ ] 发起人对外申报的批准文件（如外商投资 / 国资等特殊情形）

## 三、出资审核

### A. 货币出资

- [ ] 银行进账单 / 汇款回执（验资银行账户）
- [ ] 出资人签章确认
- [ ] 出资期限符合章程（2024 新法："自公司成立之日起 5 年内缴足"）

### B. 非货币出资

- [ ] 评估机构资质、评估日期、评估方法合规
- [ ] 评估报告结论与验资报告一致
- [ ] 权属转移证明：
  - 不动产：过户登记
  - 机动车：过户
  - 股权：工商变更
  - 知识产权：专利局 / 商标局 / 版权局登记
  - 土地使用权：自然资源局过户登记
  - 实物设备：移交清单 + 入账凭证
- [ ] 非货币出资比例（当前法律无上限，但要注意产业政策和行业限制）

### C. 股份公司设立

- [ ] 发起设立：全体发起人认购的股本总额 → 5 年内缴足
- [ ] 募集设立：发起人认购 + 对外募集（需符合公开发行规定）
- [ ] 创立大会的召开（募集设立方式下必须召开）

## 四、整体变更（有限变股份）

仅当发行人是**由有限责任公司整体变更**为股份有限公司时核查：

- [ ] 审计基准日的审计报告（评估机构 + 审计机构出具）
- [ ] 净资产折股方案：净资产总额 ≥ 股本，差额计入资本公积（股份溢价）
- [ ] 整体变更决议（股东会 2/3 以上表决权通过）
- [ ] 股东身份连续性（股东在整体变更前后一致，或已按规定做股权转让）
- [ ] 整体变更工商登记
- [ ] 整体变更时的税务处理（个人股东的个人所得税、企业股东的企业所得税；税务处理是否纳入
      企业所得税递延纳税备案（国税公告 2014 年第 116 号、财税 2015 年 41 号等））
- [ ] 整体变更后**持续经营时间**的起算（目前监管口径："发行人申报前整体变更为股份公司的，
      整体变更前的有限责任公司持续经营期间可连续计算"）

## 五、历史瑕疵与补正

若发现下列情形，需在 Memo 中专项记载瑕疵性质、原因、补正措施和法律后果：

- 货币出资未真实到位（抽逃、过桥、借款回流）
- 非货币出资未办权属转移
- 评估程序瑕疵（未评估、评估机构无资质、评估报告过期）
- 创立大会未召开或决议无效
- 工商登记信息与实际出资不一致

常见补正方式：
1. 补缴出资（含资金来源说明）
2. 补办权属登记
3. 原股东 / 发起人出具承诺函 + 董事 / 监事出具尽职调查报告
4. 会计师事务所出具专项鉴证报告
5. 在招股书专项披露

## 六、关键文件清单

- [ ] 设立时的工商档案（完整版：设立登记申请、章程、发起人协议、股东名册、首次营业执照、董事会 / 监事会成员名单等）
- [ ] 验资报告（含银行回单、入账凭证）
- [ ] 评估报告（如有非货币出资）
- [ ] 创立大会决议、会议记录、签到表（股份公司募集设立）
- [ ] 整体变更时的审计报告、净资产折股方案、工商变更档案
- [ ] 补正证据（如存在瑕疵）

## 七、特别关注

- **2024 年新法对认缴期限的影响**：老法下"章程自定"的长期认缴未到期，新法下需在 5 年内缴足
  （过渡安排见国务院相关规定），DD 中应专项评估
- **持续经营 3 年**：《注册办法》第 8 条要求发行人"依法设立且持续经营 3 年以上"，
  整体变更的起算点、分立合并后的接续问题需要专门论证
- **出资方式变动**：设立后若股东将货币出资改为实物出资（或相反），构成实质性出资方式变更，
  需要履行股东会决议 + 验资 + 工商变更全流程
