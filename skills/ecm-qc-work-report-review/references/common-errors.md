# 律师工作报告 — 常见错误库

---

## A 类 — 基础信息类（事实错）

| 编号 | 典型错误 |
|---|---|
| A1 | 客户简称前后不一致 |
| A2 | 律所名不一致（中英文） |
| A3 | 出具日期三处（封面 / 声明 / 签字页）不一致 |
| A4 | 律所文号命名不规范 |
| A5 | 项目名称跨段不一致 |
| A6 | 报告期跨章描述不一致（某章写"2022-2024"、另一章写"最近三年"但期间不同） |

---

## B 类 — 章节结构类

| 编号 | 典型错误 |
|---|---|
| B1 | 章节号错（"第一部分" 写成 "第一章"） |
| B2 | 章节号断号（"第一、三、三部分"） |
| B3 | 17 章缺章但无占位说明 |
| B4 | 独立性章节位置错（NN=04，位于第四部分） |
| B5 | NN=03 共用目录内"设立"和"历史沿革"顺序错 |

---

## C 类 — 五段式保留类

| 编号 | 典型错误 |
|---|---|
| C1 | 某章缺"一、核查要点清单" |
| C2 | 某章缺"三、风险分级汇总" |
| C3 | 二级标题命名不统一（"一、核查与审阅发现"合并二级标题） |
| C4 | "二、审阅发现"未按"### 发现 N：" 格式 |
| C5 | "五、参考资料"段被删 |

---

## D 类 — 专属表类

| 编号 | 典型错误 |
|---|---|
| D1 | 专属表被改为文字摘要 |
| D2 | 专属表位置错（未在"二、审阅发现"后） |
| D3 | 股权结构图 / 实控人穿透图丢失 |
| D4 | 五独立对照评估表被简化 |

---

## E 类 — 全项目风险分级汇总表类（高频）

| 编号 | 典型错误 |
|---|---|
| E1 | 附件四列定义错（列数不对、顺序不对） |
| E2 | 聚合不完整（漏个别 Memo 的风险行） |
| E3 | 序号按章节重置（应跨章节连续） |
| E4 | 级别出现"严重 / 一般"等别名 |
| E5 | "所属章节"列格式不统一（有的写"第一部分"，有的写"第 1 部分"） |
| E6 | "建议措施"列为空 |

---

## F 类 — 编制说明类

| 编号 | 典型错误 |
|---|---|
| F1 | 存在 DD Memo 违约但报告未加编制说明提示 |
| F2 | 缺章但"未完成章节"清单未列出 |
| F3 | 跨 Memo `skill_version` 不一致但未提示 |

---

## G 类 — 附件类

| 编号 | 典型错误 |
|---|---|
| G1 | 附件一"核查文件清单"未按 17 章分组 |
| G2 | 附件二律师执业证号缺失 |
| G3 | 附件顺序错 |
| G4 | 缺附件四 |

---

## H 类 — 签字页类

| 编号 | 典型错误 |
|---|---|
| H1 | 经办律师 < 2 人 |
| H2 | 律所负责人签字缺失 |
| H3 | 律所盖章位置错 |
| H4 | 签字页日期与封面日期不一致 |
| H5 | 签字律师与引言段"（二）本所及经办律师承办概况"不一致 |

---

## I 类 — 交叉引用 + 方位词类

| 编号 | 典型错误 |
|---|---|
| I1 | 使用"上文 / 下文 / 前述"方位词 |
| I2 | 跨章节引用章节号但章节号已变 |
| I3 | 引用附件但附件名已改 |
| I4 | 引用页码但页码因排版调整已变 |
