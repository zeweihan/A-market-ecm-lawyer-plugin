# 披露文件法律相关章节 — 形式要件审查清单

按**信披文件类型**列出法律相关章节清单与形式要件。内核审查时按类型应用对应章节表。

> 本清单与 `ecm-draft-disclosure-review/references/disclosure-chapter-map.md` 共享章节定义；本 skill 在此基础上补充"形式要件"要求。

---

## 一、招股说明书（IPO）

### 法律相关章节清单

| 节号 | 章节名 | 核心披露要点 |
|------|------|------------|
| 前置 | 重大事项提示 | 工作报告所有"高"级风险事项必须逐项列示 |
| 前置 | 风险因素 | 系统性风险 + 经营性风险 + 法律合规性风险 |
| 第三节 | 发行人及本次发行 | 发行人基本情况、股权结构、董监高 |
| 第四节 | 发行人基本情况 | 历史沿革、股本演变、主要股东、实控人 |
| 第五节 | 业务与技术 | 业务资质、重大合同、行业地位（法律层面） |
| 第八节 | 公司治理 | 股东大会 / 董事会 / 监事会运行 |
| 第九节 | 关联交易和同业竞争 | 关联方清单、关联交易、同业竞争 |
| 第十节 | 主要财产 | 不动产、知识产权、重大设备 |
| 第十一节 | 董监高及其他核心人员 | 花名册、背景、兼职、激励 |
| 第十三节 | 财务会计信息（法律部分） | 主要债权债务 |
| 第十四节 | 募集资金运用 | 募投项目、资金用途 |
| 第十五节 | 诉讼、仲裁、行政处罚 | 已决 / 未决案件清单 |
| 第十七节 | 其他重要事项 | 其他法律披露 |

### 形式要件 10 项

1. **重大事项提示段**必须存在，且至少包含：
   - 全项目风险分级汇总表中的**全部"高"级事项**
   - 特别表决程序（如涉及）
   - 业绩承诺或补偿（如涉及）
   - 减持限制（如涉及）
2. **风险因素段**按类别组织（系统性 / 经营性 / 法律合规性 / 财务性），每类至少 2-3 条具体风险
3. **发行人基本情况**含：中文 / 英文全称、注册资本、设立日期、统一社会信用代码、注册地址、联系方式、董监高姓名与联系方式
4. **历史沿革**以时间线组织（与工作报告股本演变时间线表一致）
5. **股权结构**附表含：股东名称 / 持股数 / 持股比例 / 股份性质（限售 / 非限售 / 内资 / 外资）
6. **实控人认定**：如存在实控人，披露姓名 / 持股链条；如不存在实控人，披露认定依据
7. **关联交易**列表：每项披露关联方 / 交易类型 / 金额 / 占比 / 公允性判断
8. **诉讼仲裁**列表：被告 / 案由 / 标的金额 / 审理阶段 / 预计影响
9. **募集资金运用**每个项目：名称 / 投资金额 / 建设地点 / 建设周期 / 经济效益
10. **其他重要事项**：发行人认为应披露的其他事项（不应留空）

## 二、重大资产重组报告书

### 法律相关章节

| 节号 | 章节名 | 核心披露要点 |
|------|------|------------|
| 前置 | 重大事项提示 | 交易风险 / 业绩承诺 / 股份减持承诺 |
| 前置 | 风险因素 | 交易风险 / 整合风险 / 法律合规风险 |
| 第三节 | 交易对方情况 | 交易对方基本情况、关联关系 |
| 第四节 | 交易标的 | 业务、资产、负债、合规情况 |
| 第五节 | 发行股份情况 | 定价、数量、锁定期 |
| 第八节 | 业绩补偿 | 补偿条件、计算方式、保障措施 |
| 第九节 | 同业竞争和关联交易 | 消除 / 规范措施 |
| 第十节 | 董监高 | 交易后董监高变动 |
| 第十一节 | 诉讼仲裁 | 交易对方 / 标的的诉讼 |
| 第十三节 | 其他重要事项 | 其他法律披露 |

### 形式要件 10 项（重组特有）

1. **交易对方基本情况**：完整披露每个交易对方的注册资本 / 实控人 / 主营业务 / 是否关联方 / 历史合规
2. **标的资产权属**：土地 / 房产 / 知识产权 / 特许经营权 权属清晰，无抵押 / 冻结 / 查封
3. **业绩承诺条款**：明确业绩承诺期、承诺数据、补偿方式、股份锁定
4. **股份锁定期**：交易对方、董监高的锁定期（对应法规：重组办法 / 上市规则）
5. **同业竞争消除**：如交易导致同业竞争，披露整改措施
6. **关联交易规范**：如交易对方为关联方，披露规范化措施
7. **交易批准**：是否取得所需批准（董事会 / 股东大会 / 监管）
8. **重大合同**：标的资产主要合同、未决诉讼
9. **业绩不达标补偿**：计算公式、补偿股份数 / 现金数
10. **过渡期安排**：交易批准至交割的过渡期

## 三、简式 / 详式权益变动报告书

### 法律相关章节

- 收购人基本情况
- 权益变动目的
- 信息披露义务人前 6 个月买卖上市公司股份情况
- 与上市公司之间的重大交易
- 附件（承诺函、授权委托书等）

### 形式要件 6 项

1. **收购人基本情况**：注册资本、实控人、主营业务、董事、监事、高管
2. **权益变动目的**：明确是战略投资 / 财务投资，未来 12 个月继续增持计划
3. **前 6 个月交易情况**：披露买卖股份日期、价格、数量、交易对方
4. **资金来源**：自有资金 / 信贷资金 / 筹资（如为筹资披露债权人、期限）
5. **一致行动关系**：披露一致行动人 / 一致行动协议
6. **承诺函**：不谋求控制权 / 不进行内幕交易等承诺

详式报告书 vs 简式报告书：详式要求更详细（含后续计划、对上市公司的影响）；简式仅满足最低披露要求。按持股比例判断（首次达到 5% 简式；达到 5% 且继续增减变动 5% 详式）。

## 四、收购报告书

法律相关章节：收购人基本情况 / 收购决定 / 收购目的 / 收购方式 / 要约条款 / 资金来源 / 后续计划。

### 形式要件 5 项

1. **要约条款**：要约价格 / 数量 / 期限 / 条件
2. **资金来源证明**：银行保函 / 账户证明
3. **收购后的后续计划**：12 个月的业务调整计划 / 董监高调整
4. **对被收购公司的影响**：法人地位 / 员工安排
5. **一致行动关系披露**

---

## 五、上市公告书

法律相关章节：发行人基本情况 / 本次发行情况 / 上市概况 / 股东及实际控制人 / 董监高 / 重要事项提示。

### 形式要件 5 项

1. **本次发行情况**：发行股数 / 发行价格 / 发行时间 / 募资金额 / 发行费用
2. **上市概况**：上市交易所 / 上市首日参考价
3. **主要股东**：前十大股东（解除限售前后）
4. **董监高**：全部董监高（含执业证号 / 身份证号后四位）
5. **重要事项提示**：发行以来的重大变化（董监高变动、重大合同、诉讼等）

---

## 六、募集说明书（公司债）

法律相关章节：发行人基本情况 / 财务会计信息 / 募集资金运用 / 重要事项提示。

### 形式要件 4 项

1. **发行人合规性**：近三年合规记录
2. **偿债能力分析**（法律层面）：主要债权债务清单、担保情况
3. **募集资金运用**：专项用途、不挪用承诺
4. **重要事项提示**：兑付风险 / 担保措施

---

## 七、年报 / 半年报的法律相关章节

重点审查：
- 公司治理章节（股东大会 / 董事会 / 监事会召开情况）
- 关联交易（报告期内新增）
- 诉讼仲裁（报告期内新增 / 进展）
- 行政处罚（报告期内）
- 重大合同（报告期内新签）

### 形式要件 4 项

1. **公司治理**：三会召开次数、出席率、重要决议
2. **关联交易**：报告期新增，金额、公允性判断
3. **诉讼仲裁**：报告期新增或进展、预计影响
4. **承诺履行**：发行人及股东承诺履行情况

## 执行建议

1. 先识别信披文件类型，切换到对应章节表
2. 按法律相关章节清单逐章过
3. 每章按对应形式要件（6-10 项）勾对
4. 不符合形式要件的走 tracked change 或【必改】批注
