# 披露文件内核审查 — 双向交叉比对矩阵

本矩阵适用于 Step 1.5 Level 1 成功获得律师工作报告 / DD Memo / 法律意见书作为参考坐标时。

双向交叉比对是 disclosure 内核审查的**核心价值**，远比单向（只查披露文件自身形式）重要。

---

## 方向一：披露文件 → 工作报告（"你写的对不对"）

对披露文件法律相关章节的**每一项具体披露**，在工作报告中搜索依据：

| 核对项类别 | 具体字段 | 权威源 |
|-----------|---------|-------|
| 主体资格 | 成立日期 / 注册资本 / 注册地 / 经营范围 | 工作报告第二部分"发行人主体资格" / DD-Memo-主体资格 |
| 历史沿革 | 历次增资 / 减资 / 股权转让 / 改制 | 工作报告第三部分"历史沿革" / DD-Memo-历史沿革 |
| 股权结构 | 股东名单 / 持股数 / 持股比例 / 股份性质 | 工作报告第五部分"股东及实控人" / DD-Memo-股东及实控人 |
| 实际控制人 | 实控人姓名 / 持股链条 / 一致行动关系 | 同上 / 穿透图（DD 专属表） |
| 董监高 | 姓名 / 职务 / 任期 / 背景 / 兼职情况 | 工作报告第十一部分"董监高" / DD-Memo-董监高 |
| 关联方 / 关联交易 | 关联方清单 / 关联交易类型和金额 / 是否公允 | 工作报告第八部分"关联交易" / DD-Memo-关联交易 / 专属表 |
| 同业竞争 | 是否存在 / 如何消除 / 承诺函 | 同上 |
| 主要财产 | 不动产 / 知识产权 / 设备 / 租赁资产 | 工作报告第九部分"主要财产" / 专属表 |
| 重大债权债务 | 借款 / 担保 / 对外担保 / 或有负债 | 工作报告第十部分"重大债权债务" / 专属表 |
| 税务 | 税务合规性 / 税收优惠 / 高新认定 | 工作报告第十三部分"税务" / DD-Memo-税务 |
| 环保 | 环评批复 / 排污许可 / 安全生产 | 工作报告第十四部分"环保" / DD-Memo-环保 |
| 募集资金运用 | 募投项目 / 金额 / 用途 | 工作报告第十五部分"募集资金运用" / DD-Memo-募集资金 |
| 诉讼仲裁 | 已决 / 未决诉讼清单 / 金额 / 性质 | 工作报告第十六部分"诉讼" / DD-Memo-诉讼 |
| 行政处罚 | 处罚种类 / 文号 / 整改情况 | 同上 |

### 标记规则

| 情形 | 标记 | 处理 |
|------|------|------|
| 披露文字能在工作报告找到依据且文字一致 | ✅ 一致 | 无处理 |
| 披露文字能找到依据但措辞不同（同一事实不同措辞） | ⚠️ 表述差异 | 【建议】批注 |
| 披露文字在工作报告找不到依据 | ❌ 无依据 | 【核实】批注（可能起草人新增 or 工作报告未覆盖）|
| 披露文字与工作报告直接冲突（数据 / 日期 / 定性不一致） | 🔴 冲突 | tracked change 改齐 + 【必改】批注 |

### 批注模板（带工作报告证据版）

**🔴 冲突事项**：
> 【必改】本招股书第 {X} 节第 {Y} 段披露 "{披露原文}"，但经核对同项目《律师工作报告》第 {N} 部分 "{工作报告原文}"，两者就 {数据 / 日期 / 定性} 不一致。披露文件应以律师工作报告（工作底稿）为依据，不得脱离底稿披露。请订正；如确需调整，应回到工作报告 / DD Memo 一并修正。

**❌ 无依据**：
> 【核实】本招股书第 {X} 节披露 "{披露原文}"，但经核对同项目《律师工作报告》和相应 DD Memo，未发现该表述的核查依据。请确认：
> - 是起草过程中新增？如是，请落实底稿并在工作报告补充核查
> - 还是工作报告未覆盖？如是，请起草律师核实 / 删除该披露

## 方向二：工作报告 → 披露文件（"你有没有漏披露"）

对工作报告"附件四：全项目风险分级汇总表"的每一条事项，在披露文件中搜索是否已披露。

### 判定规则（基于级别）

| 工作报告的事项 | 披露文件的处理 | 标记 | 处理 |
|--------------|-------------|------|------|
| "高"级风险 | 在"重大事项提示" + "风险因素" + 对应章节都有披露，且披露充分 | ✅ 已披露 | 无处理 |
| "高"级风险 | 披露但不充分（如只在"风险因素"提一句，未在"重大事项提示"列示） | 🔴 重大遗漏（披露不充分） | 【必改】批注 |
| "高"级风险 | 根本未披露 | 🔴 重大遗漏 | 【必改】批注（**触发监管问询 / 行政处罚风险，必须补披露**）|
| "中"级风险 | 在"风险因素"或对应章节有披露 | ✅ 已披露 | 无处理 |
| "中"级风险 | 未披露 | ⚠️ 建议披露 | 【建议】批注 |
| "低"级风险 | 已披露 | ✅ 已披露 | 无处理 |
| "低"级风险 | 未披露 | ℹ️ 可选披露 | 【建议】批注，一般不强制 |

### 批注模板（带工作报告证据版）

**🔴 "高"级风险未披露 / 披露不充分**（本 skill 核心价值点）：
> 【必改】【风险提示】经核对同项目《律师工作报告》附件四"全项目风险分级汇总表"第 {K} 行 "{高级事项描述}"，级别为"高"。按信披规则，此类事项应在本招股书"重大事项提示"段 + "风险因素"段 + 对应章节（第 {N} 节）同时充分披露。当前披露情况：
> - "重大事项提示"段：{已披露 / 未披露 / 披露不充分}
> - "风险因素"段：{已披露 / 未披露 / 披露不充分}
> - 对应章节：{已披露 / 未披露 / 披露不充分}
>
> 请补充披露。披露要素建议包含：（1）{事项概括} （2）{发生原因 / 历史背景} （3）{对发行人的具体影响} （4）{整改进展或消除措施}。漏披露 / 披露不充分可能触发交易所问询或证监会稽查。

**⚠️ "中"级风险未披露**：
> 【建议】经核对《律师工作报告》附件四，第 {K} 行 "{中级事项描述}"（级别="中"）未在本招股书披露。建议在 "风险因素" 段或第 {N} 节对应章节补充披露，措辞可参考 "{Memo 原文}"。

## 方向三：与意见书口径一致性（如有意见书）

披露文件中的法律结论性表述必须与《法律意见书》一致。披露不得比意见书更乐观。

| 核对项 | 方法 | 偏差处理 |
|-------|------|---------|
| 意见书的"特别事项提示" vs 披露文件的"重大事项提示" | 逐条对照 | 意见书有而披露文件没有：【必改】 |
| 意见书的结论 "本所认为 xx 合法有效" vs 披露文件"公司合法合规运营" | 口径比对 | 披露文件更绝对 / 更乐观：【必改】 |
| 意见书的"保留 / 限定意见" vs 披露文件的风险因素 | 保留意见须进风险因素 | 未披露：【必改】 |

**批注模板**：
> 【必改】本招股书第 {X} 节披露 "{披露结论}"，但同项目《法律意见书》第 {N} 部分 "{意见书结论}" 措辞更审慎（如 "除 xx 瑕疵外合法有效" / "保留意见"）。披露文件的结论性表述不得比法律意见书更乐观。请调整披露措辞，与意见书口径保持一致。

## 跨报告期 / 跨版本一致性（如有历史版本）

如起草人提供披露文件的历史版本，做 Diff：

| 核对项 | 方法 |
|-------|------|
| 本版新增法律事实披露 | 核对是否有工作报告 / DD Memo 支撑 |
| 本版删除法律事实披露 | 核对删除原因（事实变更 / 起草人判断）是否合理 |
| 重大事项提示的增删 | 新增要有底稿；删除要有明确法规 / 事实依据 |

变更合理性评估：
- 合理（如事项已消除、整改完成）：记录但不触发批注
- 不合理（如事项仍存在但披露被删）：【必改】批注要求恢复披露
