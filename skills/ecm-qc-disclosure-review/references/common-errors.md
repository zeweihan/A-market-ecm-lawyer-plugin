# 披露文件 — 常见错误库

---

## A 类 — 基础信息类

| 编号 | 典型错误 |
|---|---|
| A1 | 发行人简称前后不一致 |
| A2 | 报告期跨章不一致 |
| A3 | 注册资本 / 总股本披露数据不一致（与工作报告 / 年报 / 工商登记） |
| A4 | 股权结构数据不自洽（表格合计与文字描述冲突） |
| A5 | 实控人持股链条描述与穿透图不一致 |

---

## B 类 — 章节完整性类

| 编号 | 典型错误 |
|---|---|
| B1 | 缺"重大事项提示"段 |
| B2 | "重大事项提示"段未涵盖工作报告的"高"级风险（**核心问题**） |
| B3 | 缺"风险因素"段 |
| B4 | 按信披文件类型的法律相关章节缺章（见 form-requirements） |
| B5 | 章节顺序与信披规则不一致 |

---

## C 类 — 与工作报告 / DD 冲突类

| 编号 | 典型错误 |
|---|---|
| C1 | 披露数据（股数 / 金额 / 比例 / 日期）与工作报告冲突 |
| C2 | 披露的法律定性与工作报告 / DD Memo 冲突（如披露"不存在"某瑕疵，但 DD Memo 识别了该瑕疵） |
| C3 | 披露超出工作报告核查范围（披露事实在工作报告找不到底稿） |
| C4 | 披露结论比意见书更乐观 |
| C5 | 披露遗漏意见书的"特别事项" |

---

## D 类 — 重大遗漏类（触发监管问询 / 处罚风险）

| 编号 | 典型错误 |
|---|---|
| D1 | "高"级风险未披露 |
| D2 | "高"级风险披露位置不对（只在某章节提一句，未在"重大事项提示"段） |
| D3 | 关联交易漏披露（如未披露某关联方的交易） |
| D4 | 诉讼仲裁漏披露（未决 / 预计败诉的重大诉讼） |
| D5 | 行政处罚漏披露（3 年内的所有处罚应披露） |
| D6 | 重大合同漏披露（占营收 10% 以上的客户 / 供应商合同） |
| D7 | 主要股东 / 实控人受行政处罚或刑事案件未披露 |

---

## E 类 — 披露风格类（详见 disclosure-style-checklist）

| 编号 | 典型错误 |
|---|---|
| E1 | 夸大表述（"国内领先"、"行业第一"） |
| E2 | 避重就轻（风险披露段笼统） |
| E3 | 措辞模糊（"基本符合"、"相关规定"） |
| E4 | 结论与事实不符 |
| E5 | 未定义缩写 |
| E6 | 引用过时法规 |

---

## F 类 — 关联交易类

| 编号 | 典型错误 |
|---|---|
| F1 | 关联方清单漏披露（特别是亲属 / 前员工 / 通过员工持股平台控制的企业） |
| F2 | 关联交易金额与工作报告不一致 |
| F3 | 关联交易公允性判断表述模糊（"参考市场价格"但未说明比较基准） |
| F4 | 同业竞争情况披露不全（漏披露兄弟企业经营同类业务） |
| F5 | 关联方之间互相披露不一致（A 方披露对 B 方销售 X 万，B 方披露采购 Y 万，X ≠ Y） |

---

## G 类 — 募集资金类

| 编号 | 典型错误 |
|---|---|
| G1 | 募投项目总投资额与募集金额不匹配（含自筹资金未说明） |
| G2 | 募投项目建设期 / 经济效益预测无充分依据 |
| G3 | 募投项目环评 / 用地 / 能评批复未披露 |
| G4 | 补流 / 还贷 比例超限未说明（按板块规则） |

---

## H 类 — 董监高类

| 编号 | 典型错误 |
|---|---|
| H1 | 董监高任职资格未披露（独立董事的独立性声明） |
| H2 | 董监高兼职披露不全（在关联方的兼职） |
| H3 | 近 3 年董监高变动原因未披露 |
| H4 | 董监高股权激励 / 薪酬披露不全 |

---

## I 类 — 历史沿革类

| 编号 | 典型错误 |
|---|---|
| I1 | 历次增资减资转让与股东名册 / 工商登记不一致 |
| I2 | 改制时的出资到位问题未披露 |
| I3 | 股权代持还原过程未披露 |
| I4 | 实控人认定变化节点未披露 |

---

## 执行建议

优先扫描 **C 类（冲突）+ D 类（重大遗漏）** —— 这是披露内核的最高价值点，直接影响监管反馈 / 问询函数量。
