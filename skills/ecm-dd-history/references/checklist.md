# ecm-dd-history · 核查要点清单

## 一、时间线复原

- [ ] 按日期列出自设立以来的**每一次**股本变动
- [ ] 对每次变动区分类型：增资 / 减资 / 股权转让 / 股份回购 / 员工持股 / 股权激励
- [ ] 变动前后的股东结构（股东名称、持股数量、持股比例）
- [ ] 工商变更时间 vs 决议 / 协议时间（应闭环、不跳跃）

## 二、每次变动的六要素

对每次股本变动，必核下列六项：

### A. 决策程序

- [ ] 股东会 / 股东大会决议（修改章程涉及 2/3 以上表决权通过）
- [ ] 董事会决议（如章程要求）
- [ ] 召集通知、签到、表决结果
- [ ] 关联股东回避表决（如适用）

### B. 协议文件

- [ ] 增资协议 / 股权转让协议签字完整
- [ ] 必备条款齐全（标的、价格、支付方式、过渡期安排、违约责任、争议解决）
- [ ] 附件（股东会决议、评估报告、完税证明）齐全

### C. 定价公允性

- [ ] 定价依据（评估值 / 每股净资产 / 参考 Pre-IPO 投资 / 业绩承诺等）
- [ ] 与历次其他同期定价的对比（无异常价差）
- [ ] 涉及员工持股 / 股权激励的，公允价值差额的会计 / 税务处理

### D. 款项支付

- [ ] 支付凭证（银行流水、汇款回执）
- [ ] 款项支付方向（避免"自己付给自己"的循环支付）
- [ ] 支付时间与协议约定的期限匹配
- [ ] 非货币对价时的资产过户证明

### E. 工商变更

- [ ] 变更通知书
- [ ] 新版营业执照 / 股东名册
- [ ] 变更章程（若涉及）

### F. 税务处理

- [ ] 个人股东股权转让的个人所得税完税证明（财产转让所得 20%）
- [ ] 企业股东股权转让的企业所得税申报
- [ ] 增资扩股时股东应享有的公积金 / 未分配利润转增股本的税务处理
- [ ] 递延纳税备案（非货币出资、股权激励等适用情形）

## 三、涉及国有股权

- [ ] 国有股东的决策程序（职代会 / 党委会 / 董事会 / 股东会 + 国资监管机构）
- [ ] 资产评估（评估机构资质、评估报告备案 / 核准）
- [ ] 进场交易（产权交易所挂牌转让；协议转让需满足例外情形并获批）
- [ ] 国资监管机构的批复文号、批复日期、批复内容与实际交易金额 / 方式一致

## 四、涉及外资股权

- [ ] 外商投资信息报告（2020 年后）
- [ ] ODI / FDI 审批或备案（发改委 + 商务部门 + 银行外汇登记）
- [ ] 37 号文（境内居民境外特殊目的公司股权变动）
- [ ] 10 号令（外国投资者对境内企业的并购）
- [ ] 外汇登记表（银行为办理主体）

## 五、代持情形的排查与清理

- [ ] 梳理是否存在股权代持（委托持股、信托持股、隐名股东）
- [ ] 代持协议、解除协议、解除后工商变更
- [ ] 代持解除时的定价公允性（视同股权转让，有税务处理义务）
- [ ] 解除代持后是否仍有遗留纠纷
- [ ] 员工持股平台（有限合伙 / 有限公司）的历史股东变动

## 六、股权激励 / 员工持股

- [ ] 方案文本（激励对象、行权条件、价格、业绩考核）
- [ ] 股东会决议 / 董事会决议
- [ ] 激励股份来源（增发 / 老股 / 回购库存股）
- [ ] 核算：以权益结算的股份支付费用（会计处理）
- [ ] 个税处理（行权时 / 授予时的纳税时点）
- [ ] 持股平台的 GP / LP 架构（若是有限合伙）

## 七、历史出资瑕疵

若存在以下情形需记载瑕疵性质、原因、补正措施：

- [ ] 货币出资未实际到位（抽逃、过桥、借款回流）
- [ ] 非货币出资未办权属转移
- [ ] 以出资款向股东借款偿还关联方（变相抽逃）
- [ ] 减资未履行债权人通知程序
- [ ] 工商登记与实际股权长期不一致

常见补正：补缴出资 + 完税 + 出具专项核查报告 + 招股书专项披露。

## 八、近 2 年控制权稳定（《注册办法》第 13 条）

- [ ] 报告期近 2 年内控股股东未变更
- [ ] 报告期近 2 年内实际控制人未变更
- [ ] 若已变更，是否符合"发行条件豁免"的特殊情形（证监会 / 交易所问询重点）

## 九、关键文件清单

- [ ] 历次股东会 / 股东大会决议、董事会决议
- [ ] 历次增资协议 / 股权转让协议
- [ ] 历次验资报告（增资时）
- [ ] 历次评估报告（国有 / 非货币出资）
- [ ] 工商变更档案（完整版，覆盖每一次变动）
- [ ] 历次个人 / 企业股东完税证明
- [ ] 历次外汇登记表（如涉及外资）
- [ ] 历次国资批复文件（如涉及国资）
- [ ] 代持及清理文件（如有）
- [ ] 股权激励方案及相关决议（如有）
