# 案例卡片模板（Case Card Template）

本模板供 `ecm-research-case-search` 在抽取单个案例时**对齐字段**使用。Claude 在阅读
判决书、处罚决定书、审议公告全文时，按本模板抽取字段，落到统一输出格式的"2.2 案例清单"段。

---

## 一、通用字段（所有案例类型共用）

| 字段 | 说明 | 示例 |
|------|------|------|
| `id` | 本次检索内部编号 | C-01、C-02…… |
| `case_type` | 案件类型 | 裁判文书 / 行政处罚 / 纪律处分 / 审议意见 / 问询回复 |
| `case_number` | 案号或文书号 | （2019）最高法民终 1524 号；\[2024\] 5 号 |
| `parties` | 当事人 | "某某公司 v. 某某投资"、"某某 IPO 发行人"、"某某董事" |
| `issuing_authority` | 作出机构 | 最高人民法院；证监会；上交所；深交所科创板上市委 |
| `issue_date` | 文书日期 | YYYY-MM-DD |
| `status` | 效力状态 | 终审 / 一审已生效 / 审理中 / 已生效 / 撤销 / 变更 |
| `source_url` | 官方来源 URL | 裁判文书网首页或具体文书链接；证监会处罚专栏链接等 |
| `access_date` | 本次访问/核对日期 | YYYY-MM-DD |
| `reliability` | 可靠性等级 | A / B / C / D（见 shared/templates/research-output-format.md） |

## 二、裁判文书字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `court_level` | 审级 | 最高人民法院 / 高级人民法院 / 中级人民法院 / 基层人民法院 |
| `case_cause` | 案由 | 股权转让纠纷 / 公司决议效力确认纠纷 / 证券虚假陈述责任纠纷 |
| `procedure` | 程序 | 一审 / 二审 / 再审 / 执行复议 |
| `facts` | 事实摘要（2–4 句） | "2018 年 X 投资与 Y 公司签署对赌协议……" |
| `issue` | 争议焦点（1–2 句） | "投资人与目标公司的对赌条款是否有效，是否违反公司资本维持原则" |
| `holding` | 裁判要旨（2–4 句） | "法院认为，投资人与目标公司对赌……如未履行减资程序……不得请求回购" |
| `applicable_law` | 援引法条 | 《公司法》第 35 条、第 166 条；《合同法》第 52 条 |
| `conclusion` | 判决主文要点 | "驳回 X 投资要求 Y 公司回购股权之诉讼请求" |
| `precedent_value` | 先例价值 | 公报案例 / 指导性案例 / 典型案例 / 一般案例 |
| `relevance` | 对本项目的启示（2–3 句） | "本案为'与公司对赌必须履行减资程序'的代表性先例，可作为 IPO 申报前
  清理对赌条款的论证依据" |

### 写作示例

```markdown
**[C-01]** [裁判文书] （2019）最高法民终 1524 号（A 投资 v. B 公司，对赌条款效力纠纷）[终审]

- **审级**：最高人民法院 二审
- **案由**：股权转让纠纷
- **案件事实**：2018 年 A 投资与 B 公司签订《增资协议》，约定 B 公司在 2020 年前未实现上市时，
  A 投资有权要求 B 公司按原投资价加年化 8% 回购其持有的股权。B 公司未能如期上市，A 投资
  提起诉讼请求履行回购义务。
- **争议焦点**：投资人与目标公司签订的股权回购条款是否有效；若有效，能否直接支持股权回购请求。
- **裁判要旨**：最高法认为，投资人与目标公司之间的股权回购条款本身不当然无效，但公司履行
  回购义务必须以履行减资程序为前提。本案中 B 公司未履行减资程序，A 投资请求公司回购股权
  不能得到支持。
- **援引法条**：《公司法》第 35 条、第 166 条
- **结论**：驳回 A 投资要求 B 公司回购股权的诉讼请求
- **先例价值**：最高法指导性案例第 XX 号
- **对本项目的启示**：本案确立了"与公司对赌必须履行减资程序"的核心裁判规则，印证了 IPO 申报前
  对赌条款必须彻底清理（减资完成 OR 无条件放弃回购权）的审核口径。

出处：中国裁判文书网 wenshu.court.gov.cn（案号检索）| 访问日期：2026-04-24 | 可靠性：A
```

## 六、字段抽取的常见坑

1. **案号格式多样**：最高法用"（年份）最高法 XX 字 X 号"；证监会用"\[年份\] X 号"或"证监罚字
   \[年份\] X 号"；交易所用"沪/深/京 + 监函/纪律/上市〔年份〕X 号"。抽取时保留**原文标点**，
   不要擅自统一成半角全角。

2. **当事人脱敏**：裁判文书网上部分文书对自然人姓名脱敏（如"张某某"），对法人不脱敏但有时
   用简称。**引用时保持原文**，必要时在"parties"字段加注释说明脱敏状况。

3. **争议焦点 vs 裁判要旨**：
   - 争议焦点是**双方分歧**（"X 主张 A，Y 主张 B"）
   - 裁判要旨是**法院的说理**（"本院认为……")
   - 不要混淆。

4. **援引法条**：抽取时必须写完整条款编号（"第 X 条"，不仅写"公司法"）。若判决援引了多条，
   全部列出；若援引了已修订或已废止的旧法，特别标注（"1993 年公司法第 XX 条，对应现行公司法第 XX 条"）。

5. **审议结果的精确表述**：
   - "通过"≠"过会"；注册制下还需证监会注册
   - "暂缓审议"是**事实上的被否**，但走的是审慎程序（允许补充说明）
   - "终止审核"分为申请人主动撤回和审核机关主动终止，区别较大
   - 抽取时按**官方公告原文**写，不得简化

6. **问询函的回复是否公开**：沪深主板 IPO 问询回复在审核专区公开；部分重组问询回复只公开
   交易所问询函，不公开重组标的回复。抽取时明示"问询函已公开 / 回复已公开 / 仅问询公开"。

---

## 七、抽取后的正文写法要求

- 每条案例卡片独立成段，**不用表格**，便于后续被引用
- 严格按"事实 → 争点 → 要旨 / 处罚理由 → 结论 / 处罚 → 启示"顺序
- 每条案例末尾必有"出处 / 访问日期 / 可靠性"三项
- "对本项目的启示"段**必须写**，不能省略——这是本 skill 相对于裸案例检索的核心增值
