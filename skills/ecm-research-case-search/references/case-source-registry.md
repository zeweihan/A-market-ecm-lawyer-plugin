# 案例数据源登记表（Case Source Registry）

本文件登记 `ecm-research-case-search` 在检索时会用到的**所有权威案例数据源**：覆盖范围、入口 URL、查询方式、可靠性等级、已知坑。

---

## 一、司法裁判类

### 1.1 中国裁判文书网

- **入口 URL**：https://wenshu.court.gov.cn
- **覆盖范围**：理论上覆盖全国各级法院的生效裁判文书；实际上 2021 年后上网率下降，2024 年起部分文书改为非公开
- **可靠性等级**：A（官方原始）
- **查询方式**：
  - 关键词检索（支持"与/或/非"逻辑）
  - 按案号精确检索
  - 按法院、案由、审理程序、审理日期筛选
- **已知坑**：
  1. 改版频繁，具体文书 URL 易失效——**优先记录案号本身**，引用时写 "案号 + 裁判文书网首页链接"
  2. 2021 年后裁判文书上网率显著下降，**重要判决可能查不到全文**
  3. 国家安全、涉外、未成年人案件不公开——这类主题不应期望在本数据源找到
  4. 搜索引擎有频率限制，短时间内反复查询会被临时封禁——如遇 403 不强行重试
- **调用建议**：对**最高法、各高院**的判决优先使用本数据源；中院以下案件若查不到，转用"最高法公报"或各法院官网

### 1.2 最高人民法院公报案例

- **入口 URL**：https://www.court.gov.cn → 司法公开 → 公报案例
- **覆盖范围**：最高法筛选的**有指导意义**的案件
- **可靠性等级**：A
- **查询方式**：按年份、主题分类浏览
- **特点**：每年数量有限（通常 50–100 件），但每件都经过最高法编辑润色，**作为论证依据的权威性非常高**
- **调用建议**：论证主流裁判口径时优先引用公报案例

### 1.3 最高人民法院指导性案例

- **入口 URL**：最高法官网 → 司法案例 → 指导性案例
- **覆盖范围**：最高法定期发布的指导性案例（编号形如"指导性案例第 XX 号"）
- **可靠性等级**：A
- **特点**：对下级法院有**事实上的约束力**，是最高权重论据
- **调用建议**：若研究主题涉及民商事一般问题（合同效力、公司治理、股东权利），应优先查指导性案例

### 1.4 类案检索数据库（法信、法答网等）

- **入口 URL**：https://www.faxin.cn（法信）；答问类通过最高法官网进入
- **覆盖范围**：最高法主导的类案检索工具，含判决、司法政策、答复等
- **可靠性等级**：A
- **特点**：官方立场，集中度高
- **调用建议**：复杂主题的类案梳理优先用法信

## 二、证监会行政处罚类

### 2.1 中国证监会行政处罚决定书

- **入口 URL**：http://www.csrc.gov.cn → 信息公开 → 行政处罚
- **覆盖范围**：证监会本级、派出机构（各省证监局）的行政处罚决定
- **可靠性等级**：A
- **查询方式**：
  - 按年份、文号检索
  - 按当事人名称检索
  - 部分支持关键词全文检索
- **典型文号格式**：`[2024] 5 号`、`证监罚字 [2024] 99 号`、`京证监罚〔2024〕11 号`（派出机构）
- **已知坑**：
  1. 早期处罚决定（2010 年以前）上网不全
  2. 派出机构处罚在本级官网聚合不完整，需同时查当地证监局官网
- **调用建议**：检索"虚假陈述"、"内幕交易"、"操纵市场"、"保荐失职"、"中介机构责任"等主题时优先

### 2.2 证监会市场禁入决定书

- **入口 URL**：同 2.1 → 市场禁入
- **覆盖范围**：对董监高、从业人员的市场禁入决定
- **可靠性等级**：A
- **调用建议**：涉及董监高任职资格、中介机构人员合规的主题使用

## 三、证券交易所纪律处分类

### 3.1 上海证券交易所

- **入口 URL**：http://www.sse.com.cn → 监管 → 监管信息公开 → 违规处理
- **覆盖范围**：上交所主板、科创板的纪律处分、监管措施
- **文号格式**：`上证公监字〔2024〕X 号`（上市公司）、`上证纪律〔2024〕X 号`
- **可靠性等级**：A

### 3.2 深圳证券交易所

- **入口 URL**：http://www.szse.cn → 监管信息公开 → 自律监管措施 / 纪律处分
- **覆盖范围**：深交所主板、创业板的纪律处分、监管措施
- **文号格式**：`深证上〔2024〕X 号`、`深证监函〔2024〕X 号`
- **可靠性等级**：A

### 3.3 北京证券交易所

- **入口 URL**：http://www.bse.cn → 监管信息公开
- **覆盖范围**：北交所上市公司、新三板创新层/基础层挂牌公司的纪律处分
- **可靠性等级**：A

### 3.4 全国中小企业股份转让系统（新三板）

- **入口 URL**：http://www.neeq.com.cn → 信息披露 → 自律监管措施
- **覆盖范围**：新三板挂牌公司、主办券商
- **可靠性等级**：A

**共性调用建议**：研究 IPO / 再融资 / 重大资产重组项目时，应同时查发行人拟挂牌板块的
交易所纪律处分数据库，识别同行业公司的**信息披露、分红违规、承诺未履行、关联交易未履行**等
典型处分事由。

## 四、发行审核案例类

### 4.1 证监会并购重组审核委员会（重组委）

- **入口 URL**：证监会官网 → 上市公司并购重组管理 → 并购重组委工作动态
- **覆盖范围**：上市公司重大资产重组（借壳、发股购买资产）的审议意见
- **典型产出**：审议会议结果公告、问询函、反馈意见函
- **可靠性等级**：A
- **调用建议**：研究重组方案时核心数据源

### 4.2 上交所科创板上市审核委员会

- **入口 URL**：http://www.sse.com.cn → 科创板 → 发行上市审核 → 上市委审议公告
- **覆盖范围**：科创板 IPO 审议
- **产出**：审议结果公告（通过 / 暂缓审议 / 终止审核）、上市委审议意见
- **可靠性等级**：A
- **调用建议**：科创板 IPO 项目的**重点数据源**

### 4.3 深交所创业板上市委

- **入口 URL**：http://listing.szse.cn（创业板审核专区）
- **覆盖范围**：创业板注册制 IPO 审议
- **可靠性等级**：A

### 4.4 北交所上市委

- **入口 URL**：http://www.bse.cn → 公开发行并上市审核 → 审核动态
- **覆盖范围**：北交所公开发行并上市审议
- **可靠性等级**：A

### 4.5 证监会发行审核委员会（主板发审委，历史）

- **注**：2023 年 2 月 17 日全面注册制改革后，沪深主板 IPO 审核职能转至交易所，但历史发审委审议案例仍具参考价值
- **入口**：证监会官网历史公告
- **可靠性等级**：A
- **调用建议**：研究 2023 年以前主板 IPO 审核口径时使用

### 4.6 沪深交易所主板上市委（注册制后）

- **入口**：上交所主板 IPO 审核专区 / 深交所主板 IPO 审核专区
- **覆盖范围**：注册制下沪深主板 IPO 审议
- **可靠性等级**：A

## 五、问询函 / 反馈函类

### 5.1 沪深交易所问询函数据库

- **入口**：各板块审核专区下的"问询函"或"审核问询函"子栏目
- **覆盖范围**：IPO、重组、再融资、日常信息披露的问询函及回复
- **可靠性等级**：A
- **调用建议**：**论证监管关注口径的首选数据源**；同一行业同一问题的问询 + 回复，可以直接给本项目做披露参考

### 5.2 证监会反馈意见（历史）

- **注**：注册制前证监会 IPO 反馈意见
- **可靠性等级**：A
- **调用建议**：研究历史上证监会对同类问题的口径

## 六、商业数据库（C/B 级，用户订阅依赖）

### 6.1 北大法宝

- **入口 URL**：https://www.pkulaw.com
- **覆盖范围**：几乎全部官方案例 + 法规 + 学术 + 评注
- **可靠性等级**：B（编辑加工过，但源头追溯到官方）
- **特点**：需付费订阅；案号、裁判要旨标注规范；可做交叉关联
- **调用建议**：用户若已订阅，作为 P0 源的**补充**使用；未订阅则不默认尝试

### 6.2 威科先行、LexisNexis、万律、无讼案例

- 类似北大法宝，使用原则相同

### 6.3 无讼 / Alpha 案例库

- **入口**：无讼案例 wusongtech.com 等
- **可靠性等级**：C（商业聚合，需回溯原文核对）
- **调用建议**：作为辅助线索，**不作为最终引用**

## 七、非权威来源（不作为引用依据）

以下来源**仅作为线索发现**，**绝不作为正式引用**：

- 百度文库、道客巴巴等 UGC 内容站
- 个人博客、公众号（除非是官方或权威律所公众号，且仍需回溯原文）
- 微博、知乎等社交媒体讨论
- 新闻报道（可作为背景，但具体案号必须核对官方数据源）

**硬性规则**：若检索发现某案例仅在以上来源出现，在官方数据库未查到，**不写入输出**，而在
"3. 时效性与局限性"中说明"有线索但未能在官方数据源核实"。

## 八、快速检索路径图（Cheat Sheet）

按常见研究主题选择优先数据源的速查表：

| 研究主题类型 | 优先数据源（P0） | 次优（P1） |
|-------------|----------------|-----------|
| 对赌协议效力 | 裁判文书网 + 最高法公报 | 北大法宝 |
| 实际控制人认定 | 证监会审核问询 + 交易所上市委审议意见 | 证监会处罚（实控人认定错误的处罚案例） |
| 同业竞争认定 | 交易所上市委审议意见 + 上市公司重组委 | 证监会反馈 |
| 对赌清理 | 交易所 IPO 审核问询 | 上市委审议公告 |
| 股权代持 | 裁判文书网 + 交易所审核问询 | 证监会处罚（IPO 申报隐瞒代持） |
| 虚假陈述民事赔偿 | 裁判文书网（示范判决） + 最高法公报 | 证监会处罚（前置程序） |
| 内幕交易 | 证监会处罚 + 刑事判决（裁判文书网） | 最高检公布案例 |
| 信息披露违规 | 证监会处罚 + 交易所纪律处分 | 上市委审议意见 |
| 保荐责任 | 证监会处罚（保荐代表人）+ 交易所纪律处分（保荐机构） | — |
| 并购重组方案 | 重组委审议 + 交易所重组审核问询 | 裁判文书（并购纠纷） |
| 红筹架构 | 境外上市备案案例（证监会公告）+ 交易所审核问询 | — |
| 境外上市备案 | 证监会《境内企业境外发行证券和上市备案管理办法》相关公告 + 已完成备案案例 | — |
