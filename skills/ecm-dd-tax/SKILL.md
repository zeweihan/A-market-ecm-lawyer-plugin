# ecm-dd-tax

## 定位与边界

本 skill **负责**：

- 报告期内发行人及其重要子公司适用的**主要税种、税率及其变化**
- **纳税合规性**（按期申报、足额缴纳、无欠税、纳税信用等级）
- **税收优惠**的合法性（高新企业、研发费用加计扣除、即征即退、两免三减半、小型微利、
  西部大开发、地方财政返还等）的合法依据、批复文件、持续满足优惠条件、到期安排
- **税务处罚**情况（是否存在重大税务违法、是否构成发行上市的实质性障碍）
- 主管税务机关出具的**合规证明**的时效性与出具规范性
- 与历史沿革 / 股权激励 / 整体变更相关的税务处理（仅从合规角度，不做税务筹划）
- 出具《编报规则第 12 号》第 13 章对应的 DD Memo

本 skill **不负责**：

- 关联交易转让定价合理性判断（归 `ecm-dd-related-party`，本 skill 仅核查申报合规）
- 地方财政返还 / 奖励协议的民商法效力（本 skill 仅核查返还在税收合规角度的风险；
  协议效力核查归 `ecm-dd-debt`）
- 重大资产重组的税务筹划意见（超出 DD 范围；需要时由专项税务意见出具）
- 税务处罚决定书的程序合法性（归 `ecm-dd-litigation`，本 skill 仅从"是否属于重大
  违法"角度做结论）
- 募投项目本身的可行性和备案（归 `ecm-dd-fundraising`，本 skill 仅核查募投项目用地 /
  厂房涉及的税务瑕疵，例如土地使用税、房产税的缴纳）
- 股东个人转让股权的个税（归 `ecm-dd-shareholders` 的穿透核查，本 skill 仅列示）

## 本 skill 的实务加固点

- **税种全覆盖**：企业所得税、增值税、个税、印花税、土地增值税、契税及行业特定税费需按业务事实筛查。
- **优惠合规性**：高新技术企业、软件企业、研发加计扣除、地方返还等优惠必须核对资格、期间和备案材料。
- **历史交易税务**：股权转让、改制、重组、资产划转和代持还原必须核查纳税或税务说明。
- **高风险触发器**：重大欠税、偷逃税处罚、优惠资格不实、报告期内大额补税且解释不足，应列高风险。

## 核心工作流（四步）

1. **生成核查要点清单 + 主要税种 / 优惠档案**：照 [references/checklist.md](./references/checklist.md)
   输出本项目的核查清单，同时建立"主要税种台账"和"税收优惠台账"两张表，逐税种 /
   逐优惠项列示适用依据、税率、报告期金额、合规状态
2. **文件审阅与标注**：对照清单审阅税务合规证明、纳税申报表、税收优惠批文、
   税务处罚决定书；重点做三类比对：
   - 纳税申报表与审计报告是否一致（调用 `ecm-dd-data-verify` 做数值比对）
   - 税收优惠条件（研发费比例、高新技术领域、员工学历占比等）是否持续满足
   - 税务合规证明的出具日期是否覆盖招股说明书披露的完整报告期
3. **风险分级**：按下方"统一风险分级口径"
4. **结论与建议**：汇总税务合规结论；明确高风险事项的披露 / 整改路径；
   对税收优惠可持续性提出评估

## 统一风险分级口径

| 级别 | 定义 | 本 skill 场景的典型例子 |
|------|------|------------------------|
| **高（红）** | 足以构成发行上市的实质性障碍 / 监管重点反馈 / 可能被否 | 报告期存在偷税、抗税、骗税或其他重大税收违法行为且未 36 个月届满；主要税收优惠被追缴或明显不符合享受条件；地方财政返还比例过高、构成对持续盈利能力的重大依赖；重大欠税未结清 |
| **中（黄）** | 程序 / 文件瑕疵可整改 | 部分税收优惠备案文件缺失但实质合规；个别期间纳税申报逾期但已补缴及滞纳金；地方财政返还政策依据不够明确需补取证；税务合规证明出具时间过早（超过 6 个月）需重新取得；转让定价调整尚在税务沟通中 |
| **低（蓝）** | 形式瑕疵，不影响实质 | 个别纳税申报表与财务数据存在小额差异（可调节）；税务证明印章不清晰；申报表历史文件未编号归档 |

**特别约束**：根据《首次公开发行股票注册管理办法》第 12 条和各板块上市规则，
**报告期内受到重大税务行政处罚是发行条件的重大瑕疵**——本 skill 中只要识别出
"重大"税务处罚（金额标准 / 性质标准），必须评为高风险并提示可能触发发行条件问题。

## 常见误用 / FAQ

- **Q：地方财政返还属于税务合规问题还是关联交易 / 债权债务问题？**
  A：本 skill 从"税务合规角度"核查（是否构成税收优惠的违法性、是否可持续），
  协议民商法效力归 `ecm-dd-debt`；是否构成对地方政府的重大依赖（持续经营能力）
  由 `ecm-dd-business`补充评估。

- **Q：整体变更中未分配利润转增资本是否要触发本 skill？**
  A：要。整体变更的税务处理（企业所得税、股东个税）是本 skill 的核查范围——具体见
  `ecm-dd-establishment` 的变更合规性和本 skill 的税务合规性两侧共同覆盖。
