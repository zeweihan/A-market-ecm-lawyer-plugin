# ecm-dd-tax · 核查要点清单

本 skill 在项目中生成的 DD Memo 应照本清单逐条核查。法规依据以 `shared/regulations/` 下
条文或现行有效的法律法规为准。

## 一、主要税种台账（逐税种核查）

### A. 企业所得税

- [ ] 所得税税率（25% / 15% 高新 / 10% 小型微利 / 其他特殊）适用依据
- [ ] 报告期内每年汇算清缴表是否完成、是否与审计报告一致
- [ ] 跨年度损益调整是否履行主管税务机关备案（如适用）
- [ ] 特别纳税调整情况（转让定价、资本弱化等）
- [ ] 报告期内有无重大所得税稽查 / 补税 / 处罚

### B. 增值税 / 消费税 / 关税

- [ ] 增值税一般 / 小规模身份、适用税率是否正确
- [ ] 进项抵扣合规（不得抵扣项目未混入）
- [ ] 期末留抵退税是否合规申请（2022 年后扩大范围）
- [ ] 消费税（如适用）税目、税率、应税对象
- [ ] 进出口涉及的关税、代征增值税 / 消费税是否合规

### C. 附加税费

- [ ] 城建税（7% / 5% / 1% 因地而异）、教育费附加（3%）、地方教育附加（2%）
- [ ] 与增值税、消费税同期申报缴纳
- [ ] 跨区经营的附加税费划分

### D. 个人所得税（代扣代缴义务）

- [ ] 工资薪金、劳务报酬、股息红利、股权转让的代扣代缴
- [ ] 股权激励涉及的个税扣缴（员工持股、限制性股票、股票期权）
- [ ] 累计未分配利润转增资本、资本公积转增（部分情形需缴个税）

### E. 其他税种（视业务需要）

- [ ] 房产税、城镇土地使用税（自有不动产）
- [ ] 土地增值税（涉及不动产转让时）
- [ ] 契税（取得不动产或股权时）
- [ ] 印花税（合同、账簿、产权转移书据）
- [ ] 环境保护税、资源税（如适用）
- [ ] 车船税、耕地占用税（如适用）

## 二、税收优惠（逐项核查）

### A. 高新技术企业优惠

- [ ] 高新技术企业证书（含编号、发证机关、有效期）
- [ ] 享受 15% 优惠税率的年度是否在证书有效期内
- [ ] **核心指标持续满足**：
  - 研发费用占销售收入比例（按规模分档）
  - 科技人员占职工总数比例（≥ 10%）
  - 主营业务属《国家重点支持的高新技术领域》
  - 近一年科技成果转化能力
- [ ] 复审 / 重新认定的时间安排

### B. 研发费用加计扣除

- [ ] 研发项目立项文件（科技部门备案 / 自行立项）
- [ ] 研发费用归集口径与税务文件要求一致
- [ ] 辅助账、研发支出明细账完整
- [ ] 加计扣除比例（75% → 100% → 可能进一步调整，以现行政策为准）
- [ ] 报告期年度汇算清缴表附表已正确填报

### C. 小型微利企业优惠（如适用）

- [ ] 从业人数、资产总额、应纳税所得额是否符合标准
- [ ] 是否同时适用其他不可叠加的优惠

### D. 即征即退 / 两免三减半 / 西部大开发 / 软件企业等

- [ ] 享受优惠的法律依据（政策文号）
- [ ] 享受起止期间
- [ ] 批复 / 备案文件及执行情况
- [ ] 享受金额的年度审核 / 备案

### E. 增值税即征即退 / 留抵退税

- [ ] 对应政策及批复
- [ ] 实际收到的退税金额与申报是否一致
- [ ] 纳入政府补助的会计处理是否合规

### F. 地方财政返还 / 奖励（**重点风险区**）

- [ ] 政策依据（有权机关、层级、是否合法——注意税收法定原则）
- [ ] 返还协议 / 文件的合法性
- [ ] 返还占利润比重（过高构成重大依赖）
- [ ] 是否存在政策清理风险（历年国务院清理地方招商政策的影响）
- [ ] 报告期是否持续、本次发行后是否可持续
- [ ] 是否按政府补助准则规范会计处理

## 三、纳税合规性

- [ ] 税务登记证（"三证合一" / "多证合一"后由营业执照承载）
- [ ] 报告期各年度纳税申报记录（按期 / 足额）
- [ ] 纳税信用等级（A / B / M / C / D，拟上市通常要求 B 级以上）
- [ ] 主管税务机关出具的"合规证明"或"无重大违法说明"
  - 出具日期建议在申报前 6 个月内
  - 覆盖完整报告期
  - 抬头、印章规范
- [ ] 纳税义务履行涉及的**主要业务场景**（特别是跨区经营、异地项目部、派遣用工、
      出口退税等）的合规性

## 四、税务处罚与争议

- [ ] 是否被税务机关采取过"偷税、抗税、骗税"认定（**高压红线**）
- [ ] 报告期内受到的税务行政处罚（罚款、滞纳金、补税）清单
- [ ] 对应处罚决定书是否生效、是否已履行
- [ ] 按《注册办法》第 12 条和板块上市规则，是否属于"重大违法"
  （判断标准：金额 / 性质 / 情节 / 社会影响 / 是否构成犯罪）
- [ ] 正在进行的税务稽查 / 税务行政复议 / 税务行政诉讼
- [ ] 主管税务机关是否开具"无重大违法"证明

## 五、历史沿革与整体变更中的税务处理

- [ ] 整体变更为股份公司时的企业所得税处理（累计未分配利润 / 资本公积 / 盈余公积
      转增股本的计税口径）
- [ ] 整体变更时自然人股东的个税（累计未分配利润转增部分，主管税务机关有不同处理）
- [ ] 报告期内的资产划转、股权划转涉及的特殊性税务处理（财税 [2014] 109 号、
      财税 [2009] 59 号等，现行政策为准）
- [ ] 历次增资 / 转股的税务申报完整性

## 六、股权激励的税务处理

- [ ] 限制性股票 / 股票期权 / 员工持股计划的股权取得日、行权日、解禁日
- [ ] 是否适用递延纳税政策（财税 [2016] 101 号，现行政策为准）
- [ ] 扣缴义务是否履行
- [ ] 与审计、劳动、内控的交叉校验

## 七、关键文件清单（客户需提供）

- [ ] 报告期各年度企业所得税汇算清缴表 + 附表
- [ ] 报告期各年度增值税申报表（主表 + 主要附表）
- [ ] 完税凭证（报告期重要税种）
- [ ] 主管税务机关出具的合规证明 / 无重大违法证明
- [ ] 纳税信用等级查询记录
- [ ] 税收优惠批复 / 备案文件（全部）
- [ ] 高新技术企业证书 + 复审记录（如适用）
- [ ] 研发费用加计扣除的辅助账 / 项目立项材料
- [ ] 税务处罚决定书（如有）及履行凭证
- [ ] 地方财政返还协议 + 地方政府批文 + 实际到账记录
- [ ] 整体变更的税务处理文件（汇算表附注 / 税务机关意见等）
- [ ] 股权激励的税务处理记录（扣缴凭证）
- [ ] 重要税种的最近 1 期纳税证明（公司和重要子公司）

## 八、特别关注事项

- **税收法定 vs 地方返税**：地方"即征即退"式财政返还在 2014 年国发 62 号文以来反复被
  清理，项目中应评估政策层面是否仍具合法依据。对 IPO 项目，返还若构成对利润的
  重大依赖，容易触发"持续盈利能力"问题
- **"重大"税务处罚的判定**：各板块上市规则对"重大违法行为"有不同表述，基本口径
  包括：金额较大、情节严重、涉及偷抗骗税、公司核心业务涉税等。同一处罚是否
  "重大"需结合 3 年比较评估
- **高新技术企业的"持续满足"**：发行人报告期享受高新税率但在审核期间出现指标
  下降（如研发费占比跌破阈值）的，应评估是否触发反馈问询或需补税
- **过渡期 IPO 项目的转让定价**：关联交易比例较高的发行人，要特别关注转让定价
  管理文件是否齐备（同期资料、国别报告等）
- **股权代持还原、一致行动协议签署**中的个税处理：若未完整扣缴，整改时应在本
  skill 台账中体现
