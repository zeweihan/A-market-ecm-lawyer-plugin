# ecm-dd-litigation

## 定位与边界

本 skill **负责**：

- 发行人及其**重要子公司**涉及的诉讼、仲裁、行政处罚
- **控股股东、实际控制人**涉及的诉讼、仲裁、行政处罚及刑事犯罪记录
- **现任董事、监事、高级管理人员**涉及的诉讼、仲裁、行政处罚及刑事犯罪记录
- 重大未决诉讼 / 仲裁的金额、进展、潜在影响、财务计提情况
- 已结案的诉讼 / 仲裁判决书 / 裁定书 / 裁决书的履行情况
- 所有行政处罚是否属于"重大违法"（含是否属于"情节严重"、是否落入
  《注册办法》第 12 条发行条件的消极条件范围）
- 证监会立案 / 处罚 / 警示函、交易所自律处分的特别敏感事项
- 股东之间 / 发起人之间的纠纷（股权代持纠纷、对赌纠纷、一致行动纠纷等）
- 多渠道**公开信息检索汇总**（裁判文书网、信用中国、执行公开网、企业信用公示、
  征信报告、第三方工商查询）
- 出具《编报规则第 12 号》第 16 章对应的 DD Memo

本 skill **不负责**：

- 各类行政处罚所涉及的**实体合规性**（税务处罚的税务合规归 `ecm-dd-tax`；
  环保 / 安全 / 质量处罚的合规背景归 `ecm-dd-environment`；劳动 / 社保
  仲裁的合规背景归 `ecm-dd-compliance`）——本 skill 只登记处罚事实、核查
  程序合法性、判断是否构成"重大违法"
- 知识产权诉讼涉及的**权属实质**（归 `ecm-dd-assets`，BATCH-03；本 skill
  仅登记诉讼事实与进展）
- 业务合同纠纷的**业务属性与履行实质**（归 `ecm-dd-business` 或 `ecm-dd-debt`，
  BATCH-03；本 skill 仅登记纠纷事实）
- 股东身份合法性和穿透核查（归 `ecm-dd-shareholders`）
- 董监高任职资格（归 `ecm-dd-directors`；本 skill 将识别到的董监高涉诉 / 处罚
  情况反向推送给 directors）

## 免责声明

本 skill 产出的 DD Memo 为工作底稿，不构成最终法律意见。关于各具体诉讼 /
仲裁的实体判断、胜诉可能性、执行可能性等，需由发行人的**诉讼专项代理律师**
或其他专业律师出具独立意见；本 skill 仅从发行上市法律核查角度汇总披露。
完整免责声明见 [DISCLAIMER.md](../../DISCLAIMER.md)。

## 核心工作流（五步）

1. **识别核查对象的完整清单**：发行人本体 + 重要子公司 + 控股股东 + 实际
   控制人（含一致行动人）+ 全部现任董监高 + 报告期内离任的董监高（判断是否
   需入列）
2. **多渠道公开检索**（逐对象）：
   - 裁判文书网（wenshu.court.gov.cn）
   - 信用中国（creditchina.gov.cn）
   - 中国执行信息公开网
   - 国家企业信用信息公示系统 / 企查查 / 天眼查
   - 证监会行政处罚决定公告、各证监局行政处罚决定公告
   - 沪深北交易所纪律处分公告、上市委 / 并购重组委审议结果
   - 行业主管部门的处罚公告（生态环境、应急管理、市场监管、税务、海关等）
   - 中国人民银行征信（个人征信需取得本人授权）
3. **文件审阅与登记**：对清单中每一个事项建立档案，照 [references/checklist.md](./references/checklist.md)
   逐条核查
4. **风险分级**
5. **结论与建议**

## 统一风险分级口径

| 级别 | 定义 | 本 skill 场景的典型例子 |
|------|------|------------------------|
| **高（红）** | 足以构成发行上市实质性障碍 | 报告期内受到中国证监会行政处罚且在 36 个月内；受到其他行政机关处罚且**情节严重**（数额 / 性质 / 社会影响构成"重大违法"）；实际控制人 / 控股股东 / 董监高被采取证监会市场禁入；董监高存在《公司法》第 146 条消极条件；重大未决诉讼涉案金额巨大且可能导致核心资产被查封 / 执行 / 拍卖；主体涉嫌刑事犯罪被立案 / 采取强制措施 |
| **中（黄）** | 程序 / 披露瑕疵可补正，或事项对发行不构成障碍但需披露 | 报告期内存在多起一般行政处罚已履行完毕；涉案金额较大但已有充分预计负债 / 保险覆盖；股东之间纠纷已和解但未充分披露；交易所公开谴责但 12 个月已届满；近亲属 / 实控人历史刑事记录年代较早且已依法消灭 |
| **低（蓝）** | 不影响实质的事项 | 小额合同纠纷已一次性结案；查询记录仅涉及同名同姓的排查（已排除）；个别行政处罚为"警告 / 轻微违法"且年代较早；检索记录的形式瑕疵 |

**特别约束**（高压要点）：
- **证监会 / 交易所的处罚**特殊敏感，不论金额，**36 个月内**受到证监会行政处罚
  即构成发行条件消极条件（《注册办法》第 12 条）
- 其他行政机关处罚的"情节严重 / 重大违法"有不同判断标准（参考《首发业务若干
  问题解答》《监管规则适用指引——发行类第 5 号》及主管部门的专项认定）
- 董监高中任何人存在"36 个月内受证监会处罚"或"12 个月内受交易所公开谴责"，
  触发任职资格障碍，结果反向推送给 `ecm-dd-directors`

## 常见误用 / FAQ

- **Q：发行人涉案金额较小的诉讼（比如 10 万以下）要不要汇总披露？**
  A：需要登记，但在"重大诉讼"披露中可按"小额 / 不重大"处理。**对金额重大性的
  判断**应结合发行人净资产 / 净利润、所涉业务金额、诉讼数量、案件集中度等
  因素综合。

- **Q：已经和解撤诉的案件还要披露吗？**
  A：报告期内发生的重大诉讼 / 仲裁即便已结案，一般仍需在招股说明书中披露
  （内部可选择性披露取决于重大性）。本 skill 应完整登记不遗漏。

- **Q："36 个月"从什么时间起算？**
  A：《注册办法》的"最近 36 个月"一般从处罚决定书的**决定日或生效日**起算，
  截至发行上市审核时点（但具体以现行审核口径为准）。

- **Q：实控人曾受"重大行政处罚"是否等同发行人受处罚？**
  A：不同。《注册办法》第 12 条将"发行人 / 控股股东 / 实际控制人"分层规定，
  本 skill 按主体维度拆分台账，不混用。
