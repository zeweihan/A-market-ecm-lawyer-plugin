# ecm-dd-litigation · 核查要点清单

本清单按"主体清单 → 公开信息检索 → 单案 / 单处罚登记 → 重大性判断 → 披露
要求"五阶段展开。项目中每一个应核查的主体都应完整过一遍。

## 一、核查对象清单（建立主体名单）

- [ ] **发行人**
- [ ] **重要子公司**（合并范围内的境内外子公司；控制型联营合营视情况）
- [ ] **控股股东**（法人股东）
- [ ] **实际控制人**（自然人 / 法人 / 无实控人情形需专门说明）
- [ ] 实际控制人的**一致行动人**
- [ ] 现任**董事、监事、高级管理人员**（含独立董事、董事会秘书）
- [ ] 报告期内**已离任**的董监高（视重要性决定是否列入）
- [ ] **核心技术人员**（科创板 / 创业板适用，依板块披露规则）
- [ ] **法定代表人**（通常与董监高重合，单独列示确保不遗漏）

## 二、多渠道公开信息检索（逐对象）

### A. 司法信息

- [ ] 中国裁判文书网（wenshu.court.gov.cn）
- [ ] 中国执行信息公开网（失信被执行人、限高、被执行人）
- [ ] 中国法院网 / 全国法院诉讼服务网（开庭公告、案件进展）
- [ ] 人民法院报（重大案件公告）
- [ ] 涉外仲裁裁决检索（中国国际经济贸易仲裁委员会、贸仲网等）
- [ ] 裁判文书网无法覆盖时，借助企查查 / 天眼查 / 威科先行等第三方数据库

### B. 信用与处罚信息

- [ ] 信用中国（creditchina.gov.cn）
- [ ] 国家企业信用信息公示系统（gsxt.gov.cn）——行政处罚公示
- [ ] 中国证监会处罚决定公告及各证监局处罚决定公告
- [ ] 沪深北交易所纪律处分公告、新三板全国股转系统处分公告
- [ ] 税务、海关、海事、外汇、生态环境、应急管理、市场监管、药监、
      交通、消防等行业监管部门的处罚公告（国家和地方两级）
- [ ] 公安部失信人信息（如适用）
- [ ] 中国人民银行征信报告（需主体授权）

### C. 主管部门 / 第三方证明

- [ ] 公安机关出具的**无犯罪记录证明**（董监高）
- [ ] 发行人所在地主要主管部门出具的**合规证明**
  - 生态环境 → `ecm-dd-environment` 交叉
  - 应急管理 → 同上
  - 税务 → `ecm-dd-tax` 交叉
  - 市场监管 → 本 skill + business / assets 交叉
  - 海关、外汇、人社、公安、消防等（按实际业务）
- [ ] 法律服务机构出具的**涉诉 / 处罚专项报告**（如通过专项法律调查）

### D. 发行人内部说明

- [ ] 发行人关于报告期涉诉 / 涉仲 / 受处罚情况的**自查说明函**
  （加盖公章、法定代表人签字）
- [ ] 控股股东、实际控制人、董监高**个人声明与承诺函**

## 三、诉讼 / 仲裁单案登记

对每一起应登记的诉讼 / 仲裁，逐项填报：

- [ ] 案件编号（案号）、案由
- [ ] 原告 / 被告 / 第三人（标注发行人方身份：原告 / 被告 / 被申请人等）
- [ ] 管辖法院 / 仲裁委
- [ ] 立案日期、开庭 / 审理进展、是否已出判决 / 裁决
- [ ] 涉案金额（主张金额 / 被诉金额）
- [ ] 已作出的司法 / 仲裁结论
- [ ] 是否已履行或进入执行程序
- [ ] 是否已计提预计负债 / 披露或有负债
- [ ] 案件对发行人业务、资产、财务的影响评估
- [ ] 是否需要披露、披露章节
- [ ] 案件类型分类：**合同纠纷 / 侵权纠纷 / 公司纠纷 / 劳动争议 /
      知识产权 / 反垄断 / 反不正当竞争 / 行政诉讼 / 行政复议 / 涉税 /
      股东之间 / 其他**
- [ ] 对应的实体核查 skill（用于跨 skill 协同）
- [ ] 相关证据材料索引

## 四、行政处罚单案登记

对每一个应登记的行政处罚，逐项填报：

- [ ] 处罚决定书文号、决定机关（具体司局 / 分局 / 支队）、层级
- [ ] 受处罚主体（发行人 / 子公司 / 控股股东 / 实控人 / 董监高）
- [ ] 处罚事由（违法事实概述）
- [ ] 处罚依据（具体条款）
- [ ] 处罚措施：警告 / 罚款金额 / 没收 / 吊销 / 责令停产停业 / 市场禁入 等
- [ ] 是否已生效、是否已履行
- [ ] 是否已申请行政复议 / 提起行政诉讼（进展）
- [ ] 是否属于"重大违法 / 情节严重"——判断要素：
  - 金额（罚款 / 没收金额）
  - 性质（故意 / 重大过失 / 过失；单位犯罪判定）
  - 情节（主观恶性、持续时间、社会影响、是否被通报）
  - 是否涉及公众健康、生态安全、重大经济损失
- [ ] 是否落入《注册办法》第 12 条或板块规则的"消极条件"
- [ ] 发行人是否取得主管部门"不属于重大违法"的认定函（常见口径）
- [ ] 所涉实体合规归属的 DD skill（交叉协同）

## 五、刑事犯罪 / 监察记录

- [ ] 控股股东、实际控制人、董监高的刑事记录
- [ ] 近 5 年内是否受过刑事处罚
- [ ] 是否正被监察机关立案调查、留置、采取强制措施
- [ ] 是否涉嫌行贿 / 受贿 / 洗钱 / 证券犯罪等与资本市场相关的犯罪
- [ ] 即便执行完毕 5 年已过，是否仍存在对任职资格的影响（《公司法》第 146 条）

## 六、证券监管特别事项

- [ ] 控股股东 / 实控人 / 董监高是否被**中国证监会**立案调查
- [ ] 是否在 36 个月内受到证监会行政处罚
- [ ] 是否被采取**证券市场禁入**措施（终身 / 5 年 / 3 年等）
- [ ] 是否在 12 个月内受到证券交易所公开谴责
- [ ] 是否被采取出具警示函、监管谈话、责令改正、公开批评等**监管措施**
- [ ] 是否涉及保荐机构 / 会计师 / 律师 / 资产评估机构被处罚（对发行中介
      衔接的影响）
- [ ] 是否涉及上市公司公开谴责、通报批评、采取暂停或限制证券交易等纪律处分

## 七、股东 / 发起人之间的纠纷（特别章节）

- [ ] 股权代持还原过程中的纠纷
- [ ] 股权转让 / 增资协议纠纷
- [ ] 对赌协议 / 回购条款纠纷
- [ ] 一致行动协议纠纷
- [ ] 家族成员之间的财产继承 / 离婚财产分割
- [ ] 与早期投资人的清算 / 退出纠纷
- [ ] 以上任一纠纷对股权清晰性的影响（与 `ecm-dd-shareholders` / `ecm-dd-history`
      交叉）

## 八、重大性判断与披露

- [ ] **单案重大性**：按涉案金额相对发行人最近一期净资产 / 净利润的比例判断
      （常见口径：净资产或净利润 10% 以上）
- [ ] **累计重大性**：即便单案不重大，但同类案件数量多、涉案累计金额高，
      仍需披露
- [ ] **性质重大性**：关涉核心资产、核心业务、核心人员或商誉声誉的案件，
      不拘金额应当披露
- [ ] **持续经营影响**：是否对持续经营构成不确定性
- [ ] 对应披露章节：招股说明书 / 律师工作报告 / 律师意见书的"诉讼仲裁"章节

## 九、主管部门合规证明

- [ ] 公安、工商、税务、海关、外汇、环保、应急、市场监管、人社等主要
      部门出具的"报告期内无重大违法行为"证明
- [ ] 证明覆盖报告期完整区间（通常跨 3 年或 2 年 + 1 期）
- [ ] 证明出具时间距申报节点不超过 6 个月（常见惯例）
- [ ] 抬头、印章规范、主管部门层级与监管范围匹配
- [ ] 多地经营时，须覆盖主要经营地

## 十、关键文件清单

- [ ] 发行人报告期内全部诉讼 / 仲裁的起诉书 / 应诉材料 / 判决书 / 裁定书 /
      调解书 / 裁决书 / 执行文件
- [ ] 发行人报告期内全部行政处罚决定书及履行凭证
- [ ] 主管部门合规证明 / 守法证明
- [ ] 控股股东 / 实控人 / 董监高的无犯罪记录证明
- [ ] 控股股东 / 实控人 / 董监高的个人征信报告（建议取得）
- [ ] 发行人自查说明函
- [ ] 第三方检索报告（裁判文书网 / 信用中国 / 执行公开网 / 企信公示）
- [ ] 发行人披露的或有负债和预计负债计算底稿（与审计交叉）
- [ ] 历史股东之间和解协议 / 仲裁裁决 / 法院判决的全文

## 十一、特别关注事项

- **"一裁终局"类仲裁**：仲裁裁决原则上一裁终局，影响评估需特别关注
- **跨境诉讼 / 仲裁**：境外诉讼适用外国法律，境内承认与执行程序独立，
  评估需结合境外执业律师意见
- **集体诉讼 / 代表人诉讼**：2020 年《证券法》修订后的特别代表人诉讼机制，
  对上市公司 / 公众公司特殊敏感
- **刑事与行政重叠**："以刑代行政"或"以行政代刑事"情形下的定性
- **警示函 / 监管谈话**：虽非行政处罚，但多次累积可能被审核认定为"违法
  情节严重"的辅助证据
- **失信被执行人信息**：发行人及关键主体的失信记录影响重大，务必纳入核查
- **与交易所问询的衔接**：诉讼 / 处罚披露的完整性是问询高频点，缺漏是审核
  重要雷区
