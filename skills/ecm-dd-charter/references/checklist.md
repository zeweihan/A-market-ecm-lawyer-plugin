# ecm-dd-charter · 核查要点清单

## 一、公司章程本体

### A. 章程事项完整性

- [ ] 《公司法》第 46 条（有限）/ 股份公司章程事项齐全
- [ ] 现行章程记载事项与营业执照、工商登记一致
- [ ] 章程签字页有全体股东（设立）/ 法定代表人（变更）签章
- [ ] 章程修订时间与股东会决议日期闭环

### B. 条款合规性

- [ ] 股东权利义务条款不违反强制性规定
- [ ] 股东会职权、董事会职权条款与《公司法》第 56 / 66 条一致（或在法定范围内章程自定）
- [ ] 表决权分配（是否存在特别表决权 / 类别股 / 同股不同权安排及合规性）
- [ ] 利润分配条款（分红政策）
- [ ] 解散清算条款
- [ ] 若为上市公司或拟上市公司：对照《上市公司章程指引》逐条比对

### C. 章程修订程序

报告期内每次章程修订核查：
- [ ] 股东会 / 股东大会决议（2/3 以上表决权通过）
- [ ] 修订条款新旧对照表
- [ ] 工商变更登记

## 二、三会议事规则

- [ ] 股东大会议事规则
- [ ] 董事会议事规则
- [ ] 监事会议事规则
- [ ] 独立董事工作制度（拟上市 / 上市公司）
- [ ] 董事会审计委员会 / 战略委员会 / 提名委员会 / 薪酬与考核委员会实施细则
- [ ] 关联交易决策制度
- [ ] 对外担保决策制度
- [ ] 对外投资 / 募集资金管理制度
- [ ] 信息披露管理制度（拟上市 / 上市公司）
- [ ] 总经理工作细则

## 三、近 3 年（报告期内）三会召开规范性

### A. 股东大会

- [ ] 年会召开（每年至少一次）及时间合规
- [ ] 通知期限（年会 20 日前 / 临时会议 15 日前）
- [ ] 审议事项完备（与章程和议事规则一致）
- [ ] 出席情况（股东名单、持股比例、出席方式：现场 / 委托 / 网络投票）
- [ ] 签到表、授权委托书齐全
- [ ] 表决程序合规（普通决议 / 特别决议、关联方回避）
- [ ] 中小投资者单独计票（若为上市公司或需遵循的拟上市情形）
- [ ] 决议内容明确，有董事长或主持人签字
- [ ] 上市公司股东大会法律见证意见（若涉及，调用 `ecm-qc:shareholders-meeting-witness`）

### B. 董事会

- [ ] 每年至少召开两次会议
- [ ] 通知期限（章程约定，通常 10 日前）
- [ ] 审议事项齐全（年度预算、年度报告、重大交易等）
- [ ] 独立董事事前认可意见（涉及关联交易、对外担保、募投变更等）
- [ ] 董事亲自出席或书面委托
- [ ] 决议签字、会议记录归档

### C. 监事会

- [ ] 每年至少召开一次会议
- [ ] 核查财务报告、重大事项、董事高管履职情况
- [ ] 决议签字、会议记录归档

## 四、独立董事 / 董事会专门委员会

仅适用于拟上市 / 上市公司：

- [ ] 独立董事人数符合要求（占董事会 1/3 及以上）
- [ ] 独立董事资格符合（非关联、任期、兼职数量）
- [ ] 独立董事工作报告（年度述职）
- [ ] 审计委员会：主任由独立董事担任、审计委员会会议记录、对年报的意见
- [ ] 提名委员会：提名董事高管的程序
- [ ] 薪酬与考核委员会：薪酬方案、考核结果
- [ ] 战略委员会（如章程设置）

## 五、内部控制制度

- [ ] 关联交易决策制度、权限、审议程序
- [ ] 对外担保决策制度（严守《公司法》第 15 条、章程限制）
- [ ] 对外投资决策制度
- [ ] 募集资金专户管理制度（上市公司 / IPO 后适用）
- [ ] 财务管理制度（资金拆借、费用报销、资产盘点）
- [ ] 信息披露管理制度（上市公司 / 拟上市公司）
- [ ] 投资者关系管理制度
- [ ] 董事、监事、高管薪酬管理制度
- [ ] 内审制度

## 六、特别关注

- **新《公司法》（2024）衔接**：老章程须按新公司法于 2025 年 6 月 30 日前完成修订
  （董事会人数下限、监事会设置、股东会职权等调整）
- **股份公司特别决议事项**：修改章程、增减资、合并分立解散、变更公司形式需 2/3 以上表决权
- **对外担保重大情形**：为股东 / 实控人 / 关联方提供担保必须经股东大会决议

## 七、关键文件清单

- [ ] 现行章程 + 历次修订版章程（全部）
- [ ] 三会议事规则 + 独立董事制度 + 专门委员会实施细则
- [ ] 内控制度汇编（含关联交易、对外担保、对外投资、募资管理、信息披露等）
- [ ] 近 3 年三会会议通知、议案、决议、会议记录、签到表
- [ ] 独立董事近 3 年述职报告 + 专项意见（涉及关联交易等）
- [ ] 各专门委员会近 3 年会议记录
- [ ] 律师见证意见 / 法律意见（若上市公司召开股东大会时曾聘请律师）
