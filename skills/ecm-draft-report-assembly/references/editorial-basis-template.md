# 引言段 / 编制依据 标准模板片段

本文件提供 `ecm-draft-report-assembly` 起草律师工作报告"引言"段时的标准措辞，按项目类型（IPO / 再融资 / 并购 / 重组 / 新三板挂牌）分组。

---

## 一、IPO 项目（首次公开发行）

### （一）本次发行概况

```
{客户全称}（以下简称"发行人"/"{客户简称}"）拟向中国证券监督管理委员会 / 上海证券交易所 /
深圳证券交易所 / 北京证券交易所首次公开发行人民币普通股股票（A 股）（以下简称"本次发行"），
并申请在{主板 / 科创板 / 创业板 / 北交所}上市。

本次发行拟发行股票种类：人民币普通股（A 股）
发行数量：不超过 {XXX} 万股（具体以中国证监会 / 交易所批准为准）
每股面值：人民币 1.00 元
发行方式：{网下向投资者询价配售与网上向社会公众投资者定价发行相结合}
发行对象：{符合条件的询价对象和已开立 A 股账户的投资者}
承销方式：{主承销商 XXX 包销}
```

### （二）本所及经办律师承办概况

```
{律所全称}（以下简称"本所"）系根据中华人民共和国法律设立并依法存续的律师事务所。
本所持有中华人民共和国司法部颁发的"律师事务所执业许可证"（许可证号：{XXX}）
及中国证监会颁发的"律师事务所从事证券法律业务资格证书"（证书号：{XXX}）。

受发行人委托，本所指派律师 {律师 A}、{律师 B} 担任发行人本次发行的专项法律顾问，
为本次发行出具法律意见书和律师工作报告。
```

### （三）编制依据（IPO 项目标准清单）

```
本所律师出具本律师工作报告的依据包括但不限于：

1. 《中华人民共和国公司法》（2023 年修订）
2. 《中华人民共和国证券法》（2019 年修订）
3. 《首次公开发行股票注册管理办法》（证监会令第 205 号）
4. 《公开发行证券公司信息披露的编报规则第 12 号——公开发行证券的法律意见书和律师工作报告》
5. 《律师事务所从事证券法律业务管理办法》（证监会令第 159 号）
6. 《律师事务所证券法律业务执业规则（试行）》
7. 《上海证券交易所股票发行上市审核规则》/《深圳证券交易所股票发行上市审核规则》/
   《北京证券交易所股票发行上市审核规则》（依交易所确定）
8. 《上海证券交易所科创板股票上市规则》 / 《深圳证券交易所创业板股票上市规则》
   （依板块确定）
9. 《中国证监会关于支持资本市场平稳健康发展的若干政策措施》等中国证监会规范性文件
10. 发行人截至本律师工作报告出具之日已有的营业执照、公司章程、三会决议、出资证明、
    工商登记档案、历次增资及股权转让协议、股东情况、董事监事高管情况、业务资质、
    重大合同、重大债权债务、诉讼仲裁处罚、税收、环保、安全生产、募投项目及其他合规性资料
11. 发行人出具的说明与承诺，以及发行人控股股东、实际控制人、董事、监事、高级管理人员
    出具的有关声明、说明与承诺
12. 相关政府主管部门出具的证明或批复、行业主管部门出具的意见
13. 其他与本次发行有关的法律法规、规范性文件
```

### （四）关于本报告的使用（IPO 项目标准措辞）

```
本所律师同意将本律师工作报告作为发行人本次发行申请的必备法律文件，
随其他材料一同上报，并依法对本律师工作报告承担相应的法律责任。

本所律师同意发行人在其为本次发行所编制的《招股说明书》（或其他信息披露文件）中
引用本律师工作报告的内容，但发行人做上述引用时，不得因引用导致法律上的歧义或曲解。

本律师工作报告仅供发行人本次发行之目的使用，不得用作任何其他目的。本所同意本律师
工作报告作为公告文件，并依法对本律师工作报告承担相应法律责任。
```

## 二、再融资项目（定增 / 配股 / 可转债）

### （一）本次发行概况

```
{客户全称}（股票简称 {XXX}，股票代码 {XXXXXX}；以下简称"发行人"）是一家在{上海 / 深圳 / 北京}
证券交易所{主板 / 科创板 / 创业板 / 北交所}上市的股份有限公司。

发行人拟向{特定对象 / 原股东 / 社会公众}发行{股票 / 可转换公司债券 / 优先股}（以下简称"本次发行"）。
……
```

### （三）编制依据（再融资项目标准清单）

```
1. 《公司法》《证券法》
2. 《上市公司证券发行注册管理办法》（证监会令第 206 号）
3. 《公开发行证券公司信息披露的编报规则第 12 号》
4. 《律师事务所从事证券法律业务管理办法》及《执业规则》
5. 依发行品种适用的交易所业务规则
6. 发行人上市以来有效的营业执照、公司章程、三会决议、定期报告与信披文件、
   再融资方案、本次发行决议等
7. 发行人承诺与说明；董监高承诺；控股股东、实际控制人承诺等
8. 其他相关法规、规范性文件
```

## 三、并购 / 重组项目

### （一）本次交易概况

```
{上市公司全称}（股票简称 {XXX}，股票代码 {XXXXXX}；以下简称"上市公司"/"收购人"）
拟{以发行股份及支付现金方式购买 / 以现金方式收购} {标的公司全称}{XX%}股权，
同时拟向 {特定对象} 发行股份募集配套资金不超过 {XXX} 万元（以下简称"本次交易"）。
……
```

### （三）编制依据（并购 / 重组项目标准清单）

```
1. 《公司法》《证券法》
2. 《上市公司重大资产重组管理办法》（证监会令第 230 号，2025 年修订）

4. 《关于规范上市公司重大资产重组若干问题的规定》
5. 《公开发行证券公司信息披露的编报规则第 12 号》
6. 《公开发行证券公司信息披露的编报规则第 26 号——上市公司重大资产重组》
7. 《律师事务所从事证券法律业务管理办法》及《执业规则》
8. 上市公司与标的公司的公司章程、三会决议、重组协议、业绩承诺补偿协议等
9. 其他相关法规、规范性文件
```

## 五、其他项目类型

项目类型超出上述范围时，report-assembly 会回退到最通用的编制依据（《公司法》《证券法》《编报规则第 12 号》《律师业务管理办法》及《执业规则》），并在编制说明中提示用户人工补充行业 / 项目专属依据。
