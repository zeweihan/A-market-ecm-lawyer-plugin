# 通用字段抽取清单

当律师没有给出具体章节的 `references/checklist.md` 时，用本通用字段清单打底。按文件"类型
特征"去抽，而不是盯着具体章节。

## 一、法人实体类文件

### 营业执照
- 企业名称
- 统一社会信用代码
- 注册资本
- 成立日期
- 法定代表人
- 注册地址
- 经营范围
- 登记机关
- 有效期（如有）

### 公司章程
- 公司名称、注册资本、住所地
- 股东人数与出资额
- 经营范围
- 公司机构设置（股东会 / 股东大会、董事会、监事会 / 监事）
- 法定代表人条款（董事长 / 执行董事 / 经理）
- 股东会议事规则（通知期限、表决比例、特别决议事项）
- 董事会议事规则（人数、召集、表决比例）
- 监事会议事规则
- 利润分配政策
- 章程修订条款
- 通过日期 + 股东 / 股东会签章

### 组织机构代码证（历史档案）
- 代码编号
- 有效期
- 颁发机关

### 税务登记证（历史档案）
- 税号
- 登记机关
- 纳税人类型

> 2015 年 10 月"三证合一"后，营业执照已合并前述三证。之前的老档案仍需分别核对。

## 二、决议类文件

### 股东会决议 / 股东大会决议
- 会议届次（如"2024 年第一次临时股东（大）会"）
- 会议日期
- 召集人
- 会议地点（含现场 + 网络投票）
- 股权登记日（股份公司）
- 通知发出日期
- 出席股东人数 + 所持表决权股份数 + 占总股本比例
- 决议事项清单
- 每项决议的表决结果（同意 / 反对 / 弃权，及占出席股东所持表决权的比例）
- 特别决议标注（如适用，应特别标明"本议案为特别决议事项"）
- 关联回避说明（如涉及）
- 中小投资者单独计票（如涉及）
- 见证律师签字 / 律所盖章（如律师见证）

### 董事会决议
- 会议届次（如"第二届董事会第八次会议"）
- 会议日期 + 地点
- 通知发出日期 + 通知方式
- 出席董事人数 / 应出席董事人数（含委托出席）
- 决议事项清单
- 每项决议的表决结果
- 董事签字（含委托书）
- 独立董事事前认可意见（如涉及关联 / 重大事项）

### 监事会决议
- 会议届次、日期、地点
- 通知
- 出席监事人数
- 决议事项 + 表决结果
- 监事签字

## 三、出资 / 股权类文件

### 发起人协议 / 出资协议
- 发起人清单
- 各发起人认缴出资额
- 出资方式（货币 / 实物 / 知识产权 / 土地使用权）
- 出资时间
- 发起人的权利义务
- 签署日期 + 签字盖章

### 验资报告
- 出具机构（会计师事务所）
- 验资基准日
- 验资金额 + 币种
- 各股东实缴情况
- 非货币出资的评估基础与作价
- 注册会计师签字 + 事务所盖章 + 文号

### 评估报告（股权 / 资产评估）
- 评估机构
- 评估基准日
- 评估目的
- 评估对象 + 范围
- 评估方法（资产基础法 / 市场法 / 收益法）
- 评估结论金额
- 备案情况（如涉及国有资产）
- 评估师签字 + 机构盖章 + 文号

### 股权转让协议
- 转让方 + 受让方
- 标的股权（公司、比例、对应出资额）
- 转让对价 + 定价依据
- 支付方式 + 支付时间
- 先决条件（CP）
- 过户时间
- 违约责任
- 签署日期 + 签字盖章

## 四、主要人员类文件

### 任职文件（聘任决议）
- 任职人员姓名
- 职务（董事 / 监事 / 总经理 / 副总经理 / 财务负责人 / 董秘）
- 任期（起止日期）
- 决议文号

### 个人简历
- 姓名、性别、出生年月、学历
- 工作经历（近 5 年详细）
- 现任其他公司职务
- 持股情况

### 无犯罪记录证明 / 个人诚信声明
- 出具机关 / 出具日期
- 覆盖时间段
- 涉及人员

### 承诺函（董监高）
- 承诺主体
- 承诺事项（不与公司同业竞争 / 不损害公司利益 / 减持限制 / 诚信等）
- 承诺期限
- 签字 + 日期

## 五、政府批文 / 许可类文件

### 发改委立项批复
- 文号
- 项目名称
- 项目主体
- 批准的投资总额 + 建设内容
- 有效期 / 开工期限

### 商务部门批复（外商投资）
- 文号
- 批准事项（设立 / 变更 / 转让）
- 批准日期
- 有效期（对外投资证书有效期 2 年）

### 国资委 / 国资监管机构批复
- 文号
- 批准事项
- 批准日期
- 备案层级（国务院国资委 / 省级 / 市级）

### 行业许可证（业务资质）
- 许可证名称
- 许可事项
- 许可范围
- 有效期
- 颁发机关

### 环评批复 / 排污许可证
- 文号
- 项目名称
- 批准事项 / 允许排污种类 + 量
- 有效期

### 安全生产许可证
- 证号
- 许可事项
- 许可范围
- 有效期

### 行政处罚决定书
- 决定机关
- 文号
- 被处罚主体
- 处罚事由
- 处罚依据（引用法条）
- 处罚结果（罚款金额 / 没收 / 吊销 / 警告 / 责令停产停业）
- 履行情况

## 六、重大合同类文件

### 重大业务合同（客户 / 供应商）
- 合同双方
- 合同标的
- 合同金额
- 合同期限
- 付款条件 + 交货条件
- 违约责任
- 争议解决条款
- 签署日期 + 签字盖章
- 履行情况（已执行部分）

### 借款合同
- 贷款人 / 借款人
- 贷款金额 + 币种
- 利率
- 期限
- 还款方式
- 担保方式
- 违约责任

### 担保合同
- 担保人 / 被担保人 / 债权人
- 担保主债权
- 担保方式（保证 / 抵押 / 质押）
- 担保范围 + 担保期限
- 抵押物 / 质押物

### 劳动合同 / 服务协议（关键员工）
- 员工姓名
- 岗位 + 职责
- 期限（固定期 / 无固定期）
- 薪酬
- 竞业限制
- 保密条款

## 七、诉讼仲裁类文件

### 起诉书 / 仲裁申请
- 法院 / 仲裁机构
- 案号
- 原告 / 被告（申请人 / 被申请人）
- 诉讼请求
- 事实与理由
- 日期

### 判决书 / 仲裁裁决书
- 案号
- 当事人
- 审理过程 + 事实认定
- 判决 / 裁决结果
- 生效时间
- 审理法官 / 仲裁员签章

## 如何选用

1. 如果律师已经给出 `references/checklist.md`——优先按它；本文件仅作兜底
2. 如果是扫描到未分类目录——按"文件类型特征"匹配本文件的对应章节
3. 不在列表里的特殊文件——在摘要里明确标"文件类型未收录，仅做原样摘要"
