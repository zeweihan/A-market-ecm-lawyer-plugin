# 跨文件一致性核查场景库

ECM DD 中高频出现的"同章节下多份文件彼此对照"场景。本文件登记 **15 类典型核查链路**，
每类给出应当一致的字段 + 常见不一致原因 + 建议处理。

上层业务 DD skill 可以把本文件的场景 ID 作为"审阅发现"的引用来源。

---

## 公司基础面类

### C-01：章程 vs 营业执照 vs 工商登记簿
- **应一致的字段**：企业名称、注册资本、法定代表人、注册地址、经营范围
- **常见不一致原因**：
  - 章程修订后未及时办理工商变更登记（2024 新《公司法》下注册资本认缴改革后更频繁）
  - 工商系统未同步到章程最新版本
  - 同名多版本章程存档混乱
- **建议处理**：索取最新章程 + 最新工商登记簿；如工商未更新，列入"待补办工商变更"清单

### C-02：设立时章程 vs 现行章程
- **应一致的字段**：本次发行 / 上市涉及的条款历史连续性（可以不同，但应说明演变原因）
- **常见差异**：
  - 注册资本变化→对应历次增资 / 减资
  - 股东人数变化→对应历次股权转让
  - 经营范围扩展 / 缩减→反映业务转型
- **建议处理**：把每次章程版本变化的"决议 / 协议 / 工商变更"配齐，形成历史沿革时间线

### C-03：股东册 vs 工商登记股东 vs 章程
- **应一致的字段**：股东名单、各股东持股比例、认缴 / 实缴出资额
- **常见不一致原因**：
  - 股东册未同步最新转让
  - 代持未在工商登记
  - 三者口径差异（章程通常只写发起人不写后续股东）
- **建议处理**：要求客户提供最新"股东册 + 出资证明"；对代持情况要求出代持清理文件

### C-04：历次增资协议 vs 工商变更通知书 vs 验资报告
- **应一致的字段**：增资金额、新增股东 / 原股东继续增资、增资前后注册资本、增资价格
- **常见不一致原因**：
  - 实际付款金额 ≠ 协议约定（增资价格调减 / 股东部分出资未到位）
  - 工商登记滞后于实际交易
- **建议处理**：按"决议 / 协议 / 付款凭证 / 验资 / 工商"五件套逐次核对

### C-05：股权转让协议 vs 工商变更通知书
- **应一致的字段**：转让方、受让方、转让比例、转让日期
- **常见不一致原因**：
  - 工商变更日期 ≠ 协议约定过户日期（正常情况下可差几天，超过 30 天应询问）
  - 名义受让方 ≠ 工商登记受让方（代持 / 借名持股风险）
- **建议处理**：关注日期差异和主体一致性；代持嫌疑须获客户说明书

## 三会运作类

### C-06：股东（大）会通知 vs 决议 vs 法律见证
- **应一致的字段**：会议届次、日期、地点、股权登记日、议案清单、表决方式
- **常见不一致原因**：
  - 通知发出日期晚于应通知期限（股份公司年会 20 日、临时会 15 日）
  - 议案清单有变化（加议案 / 撤议案）但未补充通知
  - 特别决议事项未在通知中特别标注
- **建议处理**：算准通知期限；议案变化必须在开会前补充通知（否则形成瑕疵）

### C-07：董事会决议 vs 独立董事事前认可意见
- **应一致的字段**：涉及关联交易 / 对外担保等重大事项时，独立董事事前认可
- **常见不一致原因**：
  - 决议有该议案但无事前认可（高风险瑕疵）
  - 独立董事事前认可日期晚于董事会召开日期（形同虚设）
- **建议处理**：凡涉及关联、重大对外担保、财务资助、超出日常经营范围的重大交易，
  独立董事事前认可必须前于董事会；否则列入"需补正"

### C-08：近 3 年三会决议清单 vs 章程议事规则
- **应一致的字段**：决议事项是否属于章程规定的各该会议权限；召集程序、表决比例
- **常见不一致原因**：
  - 章程要求特别决议（2/3）的事项仅过了普通决议（过半）
  - 本应股东会决议的事项仅经董事会
- **建议处理**：对每一决议反向比对章程权限清单；越权决议需通过追认决议补正

## 历史沿革类

### C-09：验资报告 vs 发起人协议 vs 章程（设立时）
- **应一致的字段**：发起人出资额、出资方式、注册资本
- **常见不一致原因**：
  - 非货币出资评估值较协议约定调减（导致实际验资金额少于约定）
  - 协议与章程表述不一致（章程为法定文件应以章程为准）
- **建议处理**：重点检查非货币出资的评估报告→验资附件的金额是否连贯

### C-10：非货币出资评估报告 vs 权属转移证明
- **应一致的字段**：评估对象、权属证明对应
- **常见不一致原因**：
  - 评估对象描述笼统（如"机器设备一批"）无法与具体权属证明对应
  - 知识产权评估后未办理实际权属转移
- **建议处理**：要求评估报告附清单 + 逐一对应权属转移凭证；知识产权要求查
  专利局 / 商标局变更证明

### C-11：整体变更时净资产折股 vs 最近一期审计报告
- **应一致的字段**：净资产数额（作为折股基准）
- **常见不一致原因**：
  - 审计报告 vs 变更方案采用不同基准日
  - 净资产折股比例非 1:1（产生溢价时未作资本公积转增处理）
- **建议处理**：严格审计报告基准日 = 整体变更基准日；溢价折股须资本公积入账

## 外部审批类

### C-12：股东会 / 董事会决议 vs 发改委批复 vs 商务部批复
- **应一致的字段**：项目金额、项目内容、交易结构
- **常见不一致原因**：
  - 内部决议调整后未同步办理外部变更
  - 实际交易规模超出批复
- **建议处理**：超出批复部分需补办审批；不能补办的须列为实质障碍

### C-13：国资审批批复 vs 交易作价 vs 评估报告
- **应一致的字段**：批复金额、评估价值、实际作价
- **常见不一致原因**：
  - 实际作价低于评估价（国有资产流失嫌疑）
  - 未在有效期内完成交易
- **建议处理**：交易作价原则上不低于评估价的 90%；超期需重新评估

## 业务 / 资产类

### C-14：重大业务合同 vs 董事会决议
- **应一致的字段**：合同金额、合同对方是否关联、是否属于重大对外交易
- **常见不一致原因**：
  - 金额超过章程规定的董事会权限但未经股东会
  - 关联交易未履行审议程序
- **建议处理**：按章程权限表反向复核；关联交易须补充关联方审议回避

### C-15：不动产权证 vs 土地评估 vs 采购合同
- **应一致的字段**：权属人、地块 / 楼栋标识、面积
- **常见不一致原因**：
  - 过户未完成导致证载权属人仍为原业主
  - 实际使用面积大于证载
- **建议处理**：完整权属链条（合同 + 发票 + 完税凭证 + 权属证书）

## 如何新增场景

在本文件末尾加一个 `### C-XX` 编号的新条目，含：

1. **应一致的字段**
2. **常见不一致原因**
3. **建议处理**

并在业务 DD skill 的 `references/checklist.md` 里回引本场景 ID。
