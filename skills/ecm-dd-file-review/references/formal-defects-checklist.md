# 形式瑕疵检查清单

每份文件审阅完后过一遍本清单，把命中的项记到摘要的"形式瑕疵"段。

## 一、签字

- [ ] 应签字的人全部都签了（对照抬头 / 签署页提示的主体列表）
- [ ] 签字位置正确（签署页的姓名栏而非其他空白处）
- [ ] 若是自然人，签字可辨认（签字对应姓名，而非画押）
- [ ] 若是法人，法定代表人签字与营业执照上一致
- [ ] 委托出席的董事 / 股东，附有**委托书**（含委托事项、授权范围、期限）
- [ ] 独立董事意见 / 见证律师意见另有签字位，且签字完整

**常见瑕疵**：
- 只有公司盖章没有法定代表人签字（某些文件需要两者）
- 委托书缺失或授权范围过窄
- 签字页"共 N 页 第 M 页"不完整（只签了最后一页，可能有增页风险）

## 二、盖章

- [ ] 主要章（公章、合同章）齐备
- [ ] 骑缝章完整（多页文件是否骑缝盖章，骑缝章位置合理）
- [ ] 附件页盖章（图纸、财务报表等附件是否都有章）
- [ ] 钢印（如涉及，常见于行政审批文件）
- [ ] 决议性文件的"会议记录 + 决议 + 签到表"三件套的章是否一致
- [ ] 章模糊程度可接受（关键要素如公司名称、章中编号仍可辨认）

**常见瑕疵**：
- 决议文件仅盖合同章未盖公章
- 涉外文件缺翻译件 / 公证件 / 认证件上的章
- 历史档案章模糊导致真伪存疑

## 三、日期

- [ ] 主要日期完整（"年月日"而非仅"年月"）
- [ ] 日期之间逻辑合理：
  - 决议日期 ≥ 通知发出日期 + 法定通知期限
  - 决议日期 ≥ 股权登记日（股份公司股东大会）
  - 决议日期 ≤ 工商变更日期
  - 合同签署日期 ≤ 履行日期
  - 批复日期 ≥ 申请日期
- [ ] 非连续日期填写规范（如"2025 年 3 月 12 日"而非"25/3/12"）
- [ ] 签字日期与文件日期相符（特别是多位签字人的，是否有人明显晚签数月）

**常见瑕疵**：
- 通知发出日期与决议日期不足法定期限（股份公司股东大会 20 日、临时会 15 日）
- 日期颠倒（工商变更日期早于决议日期）
- 决议追溯性生效而未说明理由

## 四、页数与完整性

- [ ] 文件首页 / 目录 / 末页齐全
- [ ] 附件清单（如有）与实际附件一致
- [ ] 页码连续，无缺页
- [ ] "共 N 页"与实际 N 页一致
- [ ] 扫描件：是否有倒置、歪斜、压角遮住关键信息

**常见瑕疵**：
- 首页目录列 5 个附件，实际只附 3 个
- 关键页（签字页 / 盖章页 / 金额页）缺失或模糊

## 五、扫描质量（针对扫描件 PDF / 图片）

- [ ] 关键文字可辨认
  - 公司名称、统一社会信用代码完整
  - 金额数字清晰（注意"0"和"6"、"1"和"7"易混）
  - 日期数字清晰
  - 签字可辨认
  - 公章中字可读
- [ ] 分辨率够用（推荐 ≥ 300 DPI）
- [ ] 彩色章是否保留（黑白扫描可能丢失防伪）

**常见瑕疵**：
- 使用手机拍照替代扫描，有阴影 / 反光
- 扫描件分辨率过低导致 OCR 失败
- 重要页仅有黑白扫描无彩色件备查

## 六、版本

- [ ] 文件为最新版（标注"最新"/"终稿"或日期最新）
- [ ] 历次修订版本齐全（能看到演变）
- [ ] 修订记录明确（修订日期、修订原因、修订内容）
- [ ] 不同版本间的差异合理（不存在"倒退式"修订）

**常见瑕疵**：
- 只有过程稿无最终版
- 最终版未体现过程中提出的修改意见
- 多份"最终版"存在冲突

## 七、文件真实性的初步判断

本 skill 不做司法鉴定，但可以识别一些初步的可疑迹象：

- [ ] 字体一致性（同一文件不同段落字体跳变是否合理）
- [ ] 章的边缘是否自然（后期 PS 的章边缘可能不齐）
- [ ] 签字笔迹在文件内是否一致
- [ ] 签字与章的关系（签字压章 / 章压签字是否符合常规）
- [ ] 文件是否"过于干净"（历史档案通常会有折痕、虫蛀等自然痕迹）

**发现可疑迹象时**：记入"疑点记录"但不直接下判断；由业务 DD skill 或项目律师决定是否
要求提供原件验核、做司法鉴定或客户出具情况说明。

## 如何使用本清单

- 本清单是**兜底**；对应章节有更具体的形式要求时以专章清单为准
- 每份文件检查完，把"发现的"项归为"形式瑕疵"段；"检查过但无问题的"不用列
- 形式瑕疵不是业务风险分级——上层业务 DD skill 会根据其性质决定是否升级为"中 / 高"
