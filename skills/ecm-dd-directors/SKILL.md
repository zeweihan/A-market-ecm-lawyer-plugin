# ecm-dd-directors

## 定位与边界

本 skill **负责**：
- 现任董事、监事、高级管理人员（含独立董事）的任职资格核查
- 董监高的兼职情况及合规性（《公务员法》、与控股股东及关联方的任职关系）
- 董监高的对外投资情况（是否构成同业竞争）
- 报告期内（近 3 年）董监高变化情况及对公司稳定性的影响
- 董监高近亲属关系（对关联交易、一致行动的潜在影响）
- 与《公司法》第 146 条、第 147 条、第 180–184 条的对应核查

本 skill **不负责**：
- 三会运作和议事规则（归 `ecm-dd-charter`）
- 独立董事制度本体（归 `ecm-dd-charter`，但独立董事个人任职资格仍归本 skill）
- 股东身份穿透（归 `ecm-dd-shareholders`）
- 薪酬和股权激励计划（归 `ecm-dd-history` 的股权激励段 + 其他 DD skill 的薪酬税务部分）
- 关联交易 / 同业竞争本体（归 `ecm-dd-related-party`，BATCH-03）
- 董监高个人的处罚和诉讼（本 skill 仅记"是否存在"；深核归 `ecm-dd-litigation`，BATCH-04）

## 核心工作流（四步）

1. **生成核查要点清单 + 董监高花名册**：逐人列示现任董监高基本信息，建立核查表
2. **文件审阅与标注**：按每个董监高逐项核查任职资格、兼职、对外投资、近亲属关系、变化情况
3. **风险分级**
4. **结论与建议**

## 统一风险分级口径

| 级别 | 定义 | 本 skill 场景的典型例子 |
|------|------|------------------------|
| **高（红）** | 实质性障碍 | 董监高存在《公司法》第 146 条消极条件、被采取证监会市场禁入、36 个月内受证监会行政处罚或 12 个月内受交易所公开谴责、报告期近 2 年内核心人员（控股股东派出董事、董事长、总经理、财务负责人）发生重大不利变化 |
| **中（黄）** | 瑕疵但可补正 | 多名高管在控股股东处兼任高管导致独立性存疑、个别任职资格证明文件缺失、对外投资清单不完整、离任原因说明不充分 |
| **低（蓝）** | 形式瑕疵 | 简历格式不统一、任职文件日期存在小差异、身份证扫描件不清 |
