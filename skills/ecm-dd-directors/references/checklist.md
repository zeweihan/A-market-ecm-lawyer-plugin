# ecm-dd-directors · 核查要点清单

## 一、董监高花名册（逐人建立）

对每位现任董事、监事、高级管理人员（含独立董事、董事会秘书），逐人建立档案：

- 基本信息：姓名、性别、出生日期、身份证 / 护照、户籍、婚姻状况
- 教育背景：最高学历、专业
- 工作简历：近 5 年（至少覆盖报告期）
- 本公司职务：职务、任期起止、聘任决议
- 其他身份：是否同时为股东 / 实控人 / 控股股东派出 / 关联方高管

## 二、任职资格核查

### A. 《公司法》第 146 条消极条件（硬性）

- [ ] 非无民事行为能力或限制民事行为能力
- [ ] 无贪污、贿赂、侵占 / 挪用财产、破坏社会主义市场经济秩序犯罪记录（或执行期满已逾 5 年）
- [ ] 未因犯罪被剥夺政治权利（或执行期满已逾 5 年）
- [ ] 非破产清算公司的董 / 经理对破产负有个人责任（或 3 年已过）
- [ ] 非被吊销营业执照公司的法定代表人且负有个人责任（或 3 年已过）
- [ ] 未有数额较大的到期债务未清偿（失信被执行人）

### B. 证券市场特殊消极条件

- [ ] 未被证监会采取市场禁入措施（或期限届满）
- [ ] 36 个月内未受证监会行政处罚
- [ ] 12 个月内未受交易所公开谴责

### C. 《公司法》其他任职限制

- [ ] 非公务员（严格限制兼职）
- [ ] 非现役军人
- [ ] 监事不得由本公司董事、高管兼任
- [ ] 党政机关领导干部不得违规在企业兼职（2013 年 18 号文等）

### D. 独立董事特别要求（拟上市 / 上市公司）

- [ ] 独立性：非关联方，持股不超过 1% 且非前 10 名股东的亲属
- [ ] 任期：连续任期最多 6 年
- [ ] 兼职数量：同时兼任独董不超过 3 家上市公司
- [ ] 专业背景（至少一名会计专业独董）

## 三、兼职情况

- [ ] 列示董监高的**全部**外部兼职（公司、职务、薪酬、任期）
- [ ] 外部兼职是否与本公司业务构成同业竞争（另由 `ecm-dd-related-party` 深入核查，
      本 skill 先做初步标记）
- [ ] 高管在控股股东 / 实际控制人单位的任职情况（影响独立性，关联 `ecm-dd-independence`）
- [ ] 董事在其他公司任职对履行勤勉义务的影响

## 四、对外投资

- [ ] 董监高**直接 / 间接**持有其他企业股权的完整清单（穿透至本人及近亲属）
- [ ] 所投企业是否与本公司构成同业竞争（初步标记，深核归 `ecm-dd-related-party`）
- [ ] 所投企业是否与本公司存在关联交易
- [ ] 所投企业是否存在重大违法违规、诉讼、处罚

## 五、近亲属关系

- [ ] 梳理董监高之间是否存在近亲属关系（父母、配偶、子女、兄弟姐妹 + 相应配偶）
- [ ] 董监高与控股股东 / 实控人 / 其他主要股东的近亲属关系
- [ ] 近亲属对公司持有的股份、任职情况
- [ ] 近亲属与公司的交易（潜在关联交易）

## 六、报告期内变化（近 3 年）

### A. 离任情况

对报告期内离任的每位董监高：

- [ ] 离任时间和原因（正常换届 / 个人原因 / 被免 / 辞职）
- [ ] 离任时的公司财务状况 / 业务状况
- [ ] 离任后与公司是否仍有关联（任职 / 股权 / 交易）
- [ ] 是否披露（上市公司适用）

### B. 新任情况

对报告期内新任的每位董监高：

- [ ] 新任的聘任决议
- [ ] 新任人员简历及任职资格核查
- [ ] 是否参加董事会必要培训（上市公司）

### C. 核心人员稳定（《注册办法》第 13 条）

- [ ] 近 2 年内董事、高级管理人员未发生重大不利变化
- [ ] 董事长、总经理、财务负责人、核心技术人员（若有披露）的连续任职情况
- [ ] 若发生变化，是否构成重大不利影响（监管问询重点）

## 七、董监高承诺事项

- [ ] 不存在《公司法》第 146 条消极条件的承诺
- [ ] 避免同业竞争的承诺
- [ ] 减少或规范关联交易的承诺
- [ ] 股份锁定承诺（控股股东 / 实控人 36 个月；其他董监高 12 个月 + 离职后 6 个月 + 每年 25% 限售）
- [ ] 稳定股价承诺（上市公司适用）
- [ ] 关于招股书虚假陈述的连带赔偿承诺

## 八、关键文件清单

- [ ] 现任董监高名单（按职务分类）
- [ ] 每位现任董监高的简历
- [ ] 每位现任董监高的身份证明
- [ ] 每位现任董监高的任职文件（董事 / 监事：股东会决议；高管：董事会决议）
- [ ] 每位现任董监高的无犯罪记录证明 + 个人征信报告（建议）
- [ ] 每位现任董监高的对外投资 / 兼职情况说明 + 承诺函
- [ ] 报告期内离任人员的离任文件和现状说明
- [ ] 报告期内新任人员的聘任决议
- [ ] 董监高关于任职合规、避免同业竞争、规范关联交易、股份锁定等承诺函（签字原件）

## 九、特别关注

- **核心人员稳定性**：监管近年对"核心人员 2 年内重大不利变化"问询很严格，变化需充分说明原因
- **独立董事违规**：上市公司独董曾被处罚的，可能触发上市标准调整
- **实控人派驻高管**：若发行人的实际控制人大量派驻高管，会影响独立性判断
- **近亲属关系披露**：招股书需披露董监高近亲属持股及任职情况，DD 中应完整摸排
