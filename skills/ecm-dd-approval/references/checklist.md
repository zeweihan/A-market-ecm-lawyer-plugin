# ecm-dd-approval · 核查要点清单

本 skill 在项目中生成的 DD Memo 应照本清单逐条核查。法规依据以 `shared/regulations/` 下条文为准。

## 一、内部决议（董事会 / 股东会 / 股东大会）

### A. 召集与通知

- [ ] 召集主体适格（董事会召集股东大会；董事长主持，董事长不能履职时按章程规定递补）
- [ ] 股东大会通知期限：年会 20 日前、临时会议 15 日前（股份公司，《公司法》第 119 条）
- [ ] 通知方式符合章程（公告 / 函告 / 电子送达）
- [ ] 通知载明会议时间、地点、方式、审议事项、股权登记日、投票程序
- [ ] 临时提案符合章程（通常要求单独或合并持股 3% 以上的股东，会议召开 10 日前提出）

### B. 召开程序

- [ ] 出席股东持股数量达到章程规定的会议召开最低比例
- [ ] 现场会议的签到表、授权委托书、身份证明齐全
- [ ] 网络投票的流程和数据记录（涉及上市公司 / 拟上市公司需要特别核查）
- [ ] 会议有律师现场见证（如涉及上市公司股东大会，必要时调用
      `ecm-qc:shareholders-meeting-witness`）
- [ ] 独立董事的事前认可意见已出具（涉及关联交易、对外担保等事项时）

### C. 表决与决议

- [ ] 普通决议：出席会议股东所持表决权过半数通过（《公司法》第 130 条）
- [ ] 特别决议事项及 2/3 以上比例（修改章程、增减注册资本、合并分立解散、变更公司形式）
- [ ] 关联董事 / 关联股东回避表决情形已按规执行
- [ ] 中小投资者单独计票（涉及上市公司 / 拟上市公司的重大事项时）
- [ ] 表决结果与决议内容一致，决议文件有董事长或主持人签字
- [ ] 决议内容明确、具体，授权范围覆盖实际方案（发行数量、价格、时间安排等）

### D. 授权有效期

- [ ] 发行相关授权通常为 12 个月（特殊情形除外），核查是否在有效期内
- [ ] 授权内容涵盖后续配套安排（调整发行数量、价格底价、承销安排、超额配售等）

## 二、外部批准（视项目性质按需核查）

### A. 发改委 / 商务部门

- [ ] 境外投资（ODI）：发改委立项批复 / 备案通知书 + 商务部门 ODI 证书
- [ ] 外资并购 / 投资：外商投资信息报告（2020 年新规后以备案制为主，部分领域仍需核准）

### B. 国资监管

- [ ] 国有股东的决策程序（职代会 / 党委会 / 董事会 / 股东会 + 国资监管机构审批或备案）
- [ ] 国有产权交易进场（涉及国有股权转让时）
- [ ] 评估程序（国有资产评估报告 + 核准或备案通知书）

### C. 行业主管 / 特殊审批

- [ ] 特殊行业（金融、医疗、教育、文化、军工等）的行业主管部门批复
- [ ] 反垄断经营者集中申报（涉及并购重组时，按《反垄断法》第 26 条和国务院规定的门槛）
- [ ] 国家安全审查（涉及外资并购关键领域时）

### D. 外汇 / 税务（关联）

- [ ] 涉及跨境对价支付的外汇登记（ODI / FDI / 37 号文 / 10 号令等）
- [ ] 重大股权交易的税务筹划和申报（纳税证明，细项归 `ecm-dd-tax`，本 skill 只需记录是否履行）

## 三、关键文件清单（客户需提供）

- [ ] 本次发行 / 交易的董事会决议（含签字页 / 盖章页）
- [ ] 本次发行 / 交易的股东会 / 股东大会决议（含召集通知、签到表、表决结果、决议正文）
- [ ] 独立董事事前认可意见 / 独立意见（如涉及关联交易、对外担保、募投变更等）
- [ ] 发改委 / 商务部 / 国资委等外部批复文件（如适用）
- [ ] 反垄断申报回执及审查决定书（如适用）
- [ ] 国家安全审查通知（如适用）
- [ ] 外商投资信息报告回执 / 外汇登记表（涉及跨境时）

## 四、特别关注事项

- **发行决议的范围**：若实际发行方案与决议授权范围不一致（如数量超授权上限、价格低于底价），
  属于高风险，需重新履行决议程序
- **决议时间与工商变更时间**的逻辑关系：增资决议→验资→工商变更的时间链要闭环
- **决议的证据留痕**：建议客户保存完整的会议录音 / 录像（涉及后续被举报或监管问询时有据可查）
- **国资特批**：涉及国有控股企业的，务必核查国资监管机构的具体文号、批复日期、批复内容
  是否与发行方案匹配
