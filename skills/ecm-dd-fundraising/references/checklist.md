# ecm-dd-fundraising · 核查要点清单

本清单按"单个募投项目"的颗粒度展开。发行人每一个募投项目（含补流 / 还贷等
无具体投向的项目）都应走完 A—F 六组核查。产业政策适用情况（G）与募集资金
管理制度（H）是**总体层面**的核查。

## 一、募投项目总体情况（总表）

- [ ] 募投方案董事会决议、股东会 / 股东大会决议（完整性、授权有效期、与
      `ecm-dd-approval` 核查结果一致）
- [ ] 投资总额、拟使用募集资金金额、占比，各项目加总与发行方案披露一致
- [ ] 各项目建设期、进度安排
- [ ] 与主营业务的相关性说明（上游 / 下游 / 横向延伸 / 主业同领域）
- [ ] 募投项目选址（是否位于发行人现有基地 / 异地 / 新设）
- [ ] 同业竞争影响分析（若涉及收购 / 异地产能，需结合 `ecm-dd-related-party`
      BATCH-03 一并评估）

## 二、单项目核查（逐项目展开）

### A. 备案 / 核准 / 审批

- [ ] 项目类型（内资 / 外资 / 境外 / 收购）
- [ ] 备案或核准层级（省 / 市 / 区县发改委；特殊行业的行业主管审批）
- [ ] 备案证 / 核准批复的**文号、批复机关、批复日期**
- [ ] 批复内容与可行性研究报告一致
- [ ] 涉及外商投资的：外商投资信息报告、商务部门备案回执、外汇登记
- [ ] 涉及国有股东 / 国有资产的：国资备案 / 审批

### B. 前置要件

- [ ] 环境影响评价批复（或环评报告表 / 登记表备案）
  - 环评结论与项目建设内容一致
  - 环评批复是否在有效期内
- [ ] 能源评估意见（高耗能项目）
- [ ] 安全生产许可前置（高危行业新建项目）
- [ ] 消防 / 施工前置（一般在施工许可阶段，但与募投进度相关）
- [ ] 特殊行业前置（药品、医疗器械、食品、危化等）

### C. 用地 / 用房

- [ ] 新增用地：
  - 土地出让合同或划拨决定书
  - 土地使用权证或不动产权证
  - 用地规划许可证 / 用地预审意见
  - 土地规划用途（工业 / 仓储 / 商办）与项目内容一致
- [ ] 利用现有土地 / 厂房：
  - 现有不动产权证
  - 与项目新用途是否一致（如需变更需重新规划审批）
- [ ] 租赁用地 / 用房：
  - 租赁合同（期限覆盖建设期 + 运营期）
  - 出租方权属
  - 消防 / 环评合规
- [ ] 涉及收购项目：目标公司用地 / 用房情况

### D. 项目内容与产能

- [ ] 主要建设内容与工艺路线
- [ ] 新增产能数量与测算依据
- [ ] 现有产能利用率（支撑新增产能合理性）
- [ ] 产品规格 / 目标客户 / 应用领域
- [ ] 对上下游及同业竞争的影响

### E. 效益测算口径（法律合规角度）

- [ ] 可行性研究报告的假设（价格 / 毛利率 / 市场占有率）与历史数据、行业
      水平对照；在合规角度判断是否构成误导性披露风险
- [ ] 测算的"达产年效益"基础是否充分
- [ ] 测算与发行人现有财务表现的衔接
- [ ] 若测算高度乐观 / 与同行业显著偏离，需提示披露风险

### F. 涉及的特殊情形

- [ ] 境外募投 / 跨境投资（ODI 备案 / 核准、外汇合规）
- [ ] 收购项目：标的公司股权 / 资产清晰性，评估报告合规，交易对方关联性
- [ ] 投资基金：投资方向合规、是否构成"财务性投资"
- [ ] 数字化 / 智能化改造：边界与内涵，不同于研发投入
- [ ] 补充流动资金 / 偿还银行贷款（见下）

## 三、补充流动资金 / 偿还银行贷款专项

- [ ] 拟补充流动资金 / 偿还贷款金额
- [ ] 占募集资金总额比例（各板块、各发行类型有不同口径上限）
  - IPO：通常要求较低比例，具体看板块
  - 定增 / 可转债：再融资口径有专门规定
- [ ] 拟偿还贷款清单（银行、合同金额、余额、到期日、利率）
- [ ] 补流的必要性测算（运营资金缺口、应收应付期限等）
- [ ] 与大额分红 / 财务性投资的联动合规性（若上市前大额分红又募资补流，
      需关注监管反馈）

## 四、超募资金 / 节余资金 / 置换预先投入

- [ ] 超募资金使用方案的董事会 / 股东大会决议
- [ ] 节余资金（项目完成后）的使用安排
- [ ] **置换 pre-IPO 投入**：
  - 董事会 / 股东大会决议
  - 审计机构专项鉴证报告
  - 置换范围仅限募投项目前期已投入资金，且经鉴证
  - 置换完成后及时披露
- [ ] 超募 / 节余 / 置换的使用限制（禁止用于财务性投资、房地产业务等）

## 五、募投变更 / 终止（报告期内发生的）

- [ ] 变更 / 终止的决策程序（董事会 / 股东大会）
- [ ] 变更 / 终止原因说明与客观性
- [ ] 剩余募集资金 / 尚未使用资金的新用途合规性
- [ ] 公告 / 披露的完整性（上市公司再融资场景）

## 六、产业政策适用（总体层面）

- [ ] 项目类型与《产业结构调整指导目录》的对照（鼓励类 / 允许类 / 限制类 /
      淘汰类）
- [ ] 《外商投资准入负面清单》对照（涉及外资时）
- [ ] 《市场准入负面清单》对照
- [ ] 特定行业的准入政策（汽车、互联网、数据、医疗等）
- [ ] 区域性产业政策（自贸区 / 海南自贸港 / 粤港澳大湾区等）

## 七、募集资金管理制度

- [ ] 《募集资金管理制度》文本合规性
- [ ] **募集资金专户**设置（专户开立银行、账号）
- [ ] **三方监管协议**（发行人 / 专户银行 / 保荐机构）
  - 协议条款完整
  - 限额提款规定
  - 销户条款
- [ ] 涉及并购募投的，与独立财务顾问的监管协议
- [ ] 募集资金使用的审批权限和流程
- [ ] 使用情况的季度 / 半年 / 年度披露机制
- [ ] 募集资金内部审计与监事会监督机制

## 八、关键文件清单

- [ ] 全部募投项目的可行性研究报告（或立项报告）
- [ ] 项目备案证 / 核准批复
- [ ] 项目环评批复 / 报告
- [ ] 用地文件（出让合同、土地证 / 不动产权证、规划许可、租赁协议）
- [ ] 用房权属文件
- [ ] 市场分析报告 / 行业研究报告（第三方）
- [ ] 对效益测算的第三方意见（如有）
- [ ] 收购类募投的标的股权 / 资产权属文件、评估报告、交易协议
- [ ] 董事会 / 股东大会关于募投方案的决议（含说明、附件）
- [ ] 《募集资金管理制度》
- [ ] 募集资金专户开户文件、三方监管协议
- [ ] 报告期内发生的募投变更 / 终止决议和披露文件
- [ ] 置换 pre-IPO 投入的董事会 / 股东大会决议和审计鉴证

## 九、特别关注事项

- **"轻资产 / 数字化"募投的披露**：数字化改造、IT 系统建设、智能化投入等
  非实体募投项目，监管对"是否属于信息化投入不构成固定资产购置"的口径不断
  调整，应按最新发行审核问答更新披露口径
- **"研发中心"募投**：需明确研发中心建设内容、研发方向、与主营业务的关联、
  研发投入资本化 / 费用化口径
- **"跨主业"募投**：明显偏离主业的募投是重大审核障碍；如发行人希望进入新领域，
  需以收购或补流形式辅助，并以主业稳定为前提
- **"再融资募投"与原募投的衔接**：再融资时原 IPO 募投未完成的，要说明原因
- **"补流 + 分红"联动**：上市前高比例分红又大比例募资补流，典型问询点
- **"财务性投资"剥离**：有财务性投资的发行人，监管要求先剥离或纳入限制
- **"土地 / 房产未落实"的高风险**：项目选址未定或用地手续严重缺失的项目，
  建议在申报前完成，否则风险上调到高
- **"房地产业务"限制**：各板块对房地产业务募投均有严格限制，结合项目定位评估
- **"大股东补血"嫌疑**：并购收购型募投涉及关联方时，核查交易对价合理性和
  是否存在变相输送利益
