# ecm-dd-shareholders · 核查要点清单

## 一、股东身份核查

### 自然人股东

- [ ] 身份证 / 护照（有效期内）
- [ ] 户籍信息、婚姻状况（配偶可能构成关联方或一致行动人）
- [ ] 近亲属关系（父母、子女、兄弟姐妹 + 其他股东 / 董监高）
- [ ] 未被列入失信被执行人名单
- [ ] 未被采取证监会市场禁入措施
- [ ] 近 36 个月 / 近 3 年内无《公司法》第 146 条任职消极条件（若同时为董监高）

### 法人股东

- [ ] 营业执照 / 合法存续证明
- [ ] 公司章程、内部决议（股东投资决议、授权签字人决议）
- [ ] 法定代表人身份证明
- [ ] 控制关系穿透至最终受益人
- [ ] 出资款决议及资金来源（自有资金 / 合法借款 / 不得使用公司资金变相持股）

### 合伙企业股东（含员工持股平台）

- [ ] 合伙协议（GP / LP 身份、出资比例、利润分配、事务执行权）
- [ ] 合伙人身份穿透至自然人或法人
- [ ] GP 是否为 PE / VC 管理人（涉及私募基金需登记备案）
- [ ] 员工持股平台穿透核查（员工名单、岗位、持股比例、锁定期）

### 三类股东（契约型私募 / 资管计划 / 信托计划）

涉及 IPO 时必核：

- [ ] 产品登记备案情况（中基协登记证明）
- [ ] 管理人登记与资格
- [ ] 穿透核查产品背后的最终投资人
- [ ] 产品存续期覆盖锁定期（不能在发行上市后到期）
- [ ] 与发行人、控股股东、实控人、董监高、本次发行中介机构的利益冲突
- [ ] 定期披露机制

## 二、股权清晰

- [ ] 股东名册与工商登记一致
- [ ] 股权权属无重大争议（诉讼 / 仲裁）
- [ ] 股权不存在重大质押 / 冻结 / 司法查封
  - 质押：核查登记证明、担保主债权、质押期限、质押比例
  - 冻结 / 查封：核查原因、金额、解除前景
- [ ] 员工持股平台 / 股权激励的锁定安排合规

## 三、实际控制人认定

### A. 认定路径

- [ ] 直接控制：单一自然人 / 法人持股比例最高且超过 30% / 章程约定的控制权
- [ ] 协议控制：通过一致行动协议 / 投票权委托实现共同控制
- [ ] 间接控制：逐层穿透至最终控制主体
- [ ] 无实际控制人：股权高度分散且不存在协议控制，须符合监管认定口径

### B. 一致行动人协议

- [ ] 一致行动范围（表决、提案、选举等）
- [ ] 一致行动期限（应覆盖锁定期和申报期）
- [ ] 争议解决机制（以谁为准）
- [ ] 协议签署时间（不应临近申报前突击签署）

### C. 国资系统控制

- [ ] 逐层穿透至国务院国资委 / 地方国资委 / 财政部 / 其他国有主体
- [ ] 判断是否构成同一实际控制人（监管有特殊豁免，但需论证）

### D. 近 2 年稳定性（《注册办法》第 13 条）

- [ ] 控股股东近 2 年未变
- [ ] 实际控制人近 2 年未变
- [ ] 若变更过，是否符合豁免情形（如家族内部调整、一致行动人内部变动等，需专项论证）

## 四、代持情形排查与清理

- [ ] 所有股东是否为真实持股（问询确认函）
- [ ] 若存在代持：
  - 原因（规避身份限制？代持他人？）
  - 代持协议原文
  - 清理方案（还原真实持股 / 解除代持）
  - 清理后的工商变更
  - 清理时的定价公允性和税务处理
  - 控股股东 / 实控人出具不再存在代持的承诺函

## 五、出资来源合法

对持股 5% 以上股东（及前 10 名股东）：

- [ ] 出资资金来源说明（自有资金 / 银行贷款 / 亲友借款 / 其他）
- [ ] 资金流水证据
- [ ] 若为借款：借款协议、利率、还款来源（不得使用发行人或其关联方资金）
- [ ] 若用发行人分红：合规；若用发行人工资 / 奖金：合规（员工激励）

## 六、关键文件清单

- [ ] 现行股东名册（加盖公章）
- [ ] 前 10 名股东 / 持股 5% 以上股东的身份证明文件
- [ ] 法人股东章程、决议、营业执照
- [ ] 合伙企业股东的合伙协议
- [ ] 三类股东的登记备案文件及穿透文件（如涉及）
- [ ] 实控人穿透图（含每一层的持股比例）
- [ ] 一致行动协议（如有共同实控）
- [ ] 股权质押登记 / 解除证明
- [ ] 股权冻结 / 解除证明（如有）
- [ ] 代持协议及清理文件（如有）
- [ ] 控股股东、实际控制人、主要股东出具的承诺函（股权清晰、无代持、无一致行动以外的其他约定等）
- [ ] 员工持股平台的员工名单和份额表

## 七、特别关注

- **"三类股东"红线**：IPO 申报前应尽可能清理或至少确保产品存续期可覆盖锁定期 +3 年
- **对赌条款**：带对赌协议 / 回购条款的股东须在申报前清理（《监管规则适用指引——发行类第 4 号》），
  具体按监管问询口径处理
- **突击入股**：申报前 12 个月新增股东须延长锁定期并专项披露出资来源
- **国资股东特殊要求**：国有股东持有的股份是否划转社保（视具体情形）
