# ecm-dd-debt · 核查要点清单

本 skill 在项目中生成的 DD Memo 应照本清单逐条核查。法规依据以 `shared/regulations/` 下条文为准。

## 一、重大借款 / 授信 / 融资

### A. 银行借款与授信

- [ ] 授信额度总览（分主要授信银行）：已用 / 未用、到期日、担保方式
- [ ] 重大单项借款合同：债权人、金额、期限、利率、用途（流动资金 / 项目贷款 / 并购贷款）
- [ ] 担保方式：保证（连带 / 一般）/ 抵押 / 质押 / 保证金 / 信用
- [ ] 限制性条款（"保护性条款"）：
  - 禁止转让 / 禁止重组 / 禁止出售重大资产
  - Change of control（控制权变更触发加速到期）
  - 禁止提前清偿 / 提前清偿罚息
  - 交叉违约（cross-default）
  - 财务契约（资产负债率 / 流动比率 / 利息保障倍数等财务指标维持义务）
  - 对外担保限制（总额或单项金额上限）
  - 优先债权安排 / 次级债权安排
- [ ] 是否需要贷款人事先同意才能进行本次发行 / 并购（"同意函"策略）
- [ ] 逾期情况、不良征信记录

### B. 债券 / 其他债务融资工具

- [ ] 公司债 / 企业债 / 中票 / 短融 / 超短融 / PPN / 永续债 / ABS / ABN 等
- [ ] 债券受托管理协议、募集说明书中的承诺事项
- [ ] 回售条款、赎回条款、加速到期条款
- [ ] 评级变化对利率 / 回售的触发
- [ ] 涉及发行人上市后对债务融资工具条款影响的分析

### C. 融资租赁 / 保理 / 其他非银融资

- [ ] 融资租赁：出租人、租赁物、租金结构、未到期金额、提前终止
- [ ] 保理融资：有追索权 / 无追索权、应收账款标的、保理费率
- [ ] 应收账款质押 / 股权质押 / 动产抵押 / 存货质押等融资安排
- [ ] 信用贷款、供应链金融安排
- [ ] 关联方借款（交 `ecm-dd-related-party` 协同）

### D. 其他债务承诺

- [ ] 股东承诺注入资金但未到位的承诺（或有债务）
- [ ] 对第三方的财务承诺（维护约定的资产负债率等）
- [ ] 对员工的股权激励行权义务、现金类长期奖励
- [ ] 与政府产业引导基金的回购义务 / 绩效对赌

## 二、对外担保

### A. 担保分类

- [ ] 对子公司担保（内部担保，合规性相对可控）
- [ ] 对合并报表范围外主体担保（格外关注）
- [ ] 对关联方担保（涉及红线，详见 `ecm-dd-related-party`）
- [ ] 对第三方担保（通常商业合理性较弱，重点核查）

### B. 核查要点（对每项对外担保）

- [ ] 担保类型：保证（连带 / 一般）/ 抵押 / 质押
- [ ] 被担保债权：主债权人、主合同、金额、期限、利率
- [ ] 担保金额、担保期限
- [ ] **决策程序**：
  - 按章程和《公司法》第 15 条（2024 年新法将担保和对外投资决策权授予董事会 / 股东会），
    判断应由董事会还是股东会（大会）审议
  - 上市公司 / 拟上市公司的担保：按《上市规则》担保规模超过净资产一定比例（常见为 50%）
    需股东大会审议
  - 关联担保：关联股东需回避表决
- [ ] 反担保：有无、形式、覆盖比例
- [ ] 担保余额与发行人净资产比例（超过 50% 通常构成高风险）
- [ ] 是否已披露（上市公司已披露；拟上市公司在招股书中披露）

### C. 违规担保识别（IPO / 重组红线）

- [ ] 历史上是否存在为控股股东 / 实控人 / 关联方违规提供担保
- [ ] 清理时点、清理方式、款项凭证
- [ ] 清理后是否再次发生（零容忍）
- [ ] 控股股东 / 实控人是否出具"不再违规担保"承诺函

## 三、重大应收 / 应付账款

### A. 应收账款

- [ ] 前五大客户应收账款余额、账龄、占比
- [ ] 账龄超过一年的应收账款（可能意味着争议或坏账）
- [ ] 坏账准备计提是否充分（会计准则层面，律师提示但不深入财务）
- [ ] 单项金额大的应收账款（是否存在争议、诉讼）
- [ ] 关联方应收（往期历史，提示 `ecm-dd-related-party` 协同）
- [ ] 应收账款保理 / 质押情况

### B. 应付账款

- [ ] 前五大供应商应付账款余额、账龄、占比
- [ ] 长期挂账应付（是否存在实际不支付情形）
- [ ] 存在争议的应付（争议金额、可能结果）
- [ ] 关联方应付

### C. 其他应收 / 其他应付

- [ ] 对员工、股东、董监高的往来款（非经营性，敏感）
- [ ] 对供应商 / 客户的非经营性往来
- [ ] 存在金额大、账龄长、对方不明的往来款——重点核查

## 六、关键文件清单（客户需提供）

- [ ] 重大借款合同 + 授信协议（金额排序前 20 或超过净资产 5% 的合同）
- [ ] 债券募集说明书 + 受托管理协议（如有债券）
- [ ] 融资租赁 / 保理 / 其他非银融资合同
- [ ] 对外担保合同 + 反担保安排 + 决策文件
- [ ] 应收账款明细表 + 账龄分析（会计师提供）
- [ ] 应付账款明细表 + 账龄分析
- [ ] 其他应收 / 其他应付明细表
- [ ] 或有负债清单（未决诉讼 + 对外担保 + 回购 / 补偿 + 其他）
- [ ] 银行授信确认函、对账单、不良征信记录查询
- [ ] 非经营性资金占用自查及清理证明（涉及历史占用）
- [ ] 重大合同 change of control / 交叉违约 / 财务契约条款清单

## 七、特别关注事项

- **IPO 红线之非经营性资金占用**：控股股东 / 实控人及其关联方对发行人资金占用构成红线；
  必须在申报前完整清理；需要控股股东 / 实控人出具承诺函
- **IPO 红线之违规担保**：发行人为控股股东 / 实控人 / 关联方违规提供担保构成红线；必须清理
- **上市公司担保审议阈值**：上市公司提供担保超过净资产 50% / 连续 12 个月担保超过 30% /
  对资产负债率超过 70% 的单位担保等情形，需股东大会审议（视交易所上市规则细则）
- **change of control 的隐蔽风险**：重大合同的 control 条款分散在不同文件，容易漏；
  核查时应专项清查
- **交叉违约传染**：一项融资违约可能触发其他融资的交叉违约，应做风险图谱
- **2024 年新《公司法》担保规则**：
  - 公司为公司股东或实控人提供担保，须经股东会 / 股东大会决议
  - 关联股东回避表决
  - 担保合同需经董事会或股东会决议，否则效力待定
- **对赌协议清理**：与股东的对赌协议（涉及发行人或创始人义务）应在申报前清理或恢复（依监管指引）
- **永续债与优先股**：金融工具分类对资产负债率的影响，法律上关注其回售 / 赎回义务
