# ecm-dd-assets · 核查要点清单

本 skill 在项目中生成的 DD Memo 应照本清单逐条核查。法规依据以 `shared/regulations/` 下条文为准。

## 一、不动产（房产 + 土地使用权）

### A. 土地使用权

- [ ] 土地性质：国有出让 / 国有划拨 / 集体土地
- [ ] 出让土地：《国有土地使用证》或《不动产权证书》、出让合同、出让金缴纳凭证、出让期限
- [ ] 划拨土地：划拨决定书、划拨用途限制、是否可以出租 / 抵押 / 转让（需补缴出让金）
- [ ] 集体土地：是否属于企业合法取得的集体经营性建设用地（"三块地"改革后可入市）；
      集体土地上不得新建工业厂房用于经营（历史遗留处理另作评估）
- [ ] 宅基地：不得作为经营性资产使用
- [ ] 土地使用权剩余年限是否覆盖经营周期
- [ ] 土地取得方式：原始取得（出让 / 划拨）、继受取得（转让 / 出资 / 划转 / 并购）——历史来源完整

### B. 房产（建筑物）

- [ ] 《不动产权证书》或历史的《房屋所有权证》+《国有土地使用证》
- [ ] 房产证载明的"房屋用途"（工业、商业、办公、住宅）与实际使用用途是否一致
- [ ] 建筑物合法性：
  - 规划许可：《建设用地规划许可证》《建设工程规划许可证》《建设工程施工许可证》
  - 竣工验收：《竣工验收报告》《不动产权证》或《房屋所有权证》
  - 消防验收：消防验收合格证明（由 `ecm-dd-compliance` 交叉）
- [ ] 违章建筑：是否存在未取得建设工程规划许可证的自建建筑（按《城乡规划法》第 64 条）
- [ ] 临时建筑：是否到期、是否办理延期
- [ ] 已建成但未办证的建筑：原因、办证可能性、时间表
- [ ] 建筑物面积与实际使用面积的差异（阳台 / 加建 / 夹层）

### C. 土地与房产的合规性

- [ ] 无历史闲置土地被收回风险
- [ ] 土地储备金、新增建设用地有偿使用费等欠费情况
- [ ] 涉及集体土地 / 宅基地的"小产权房"问题
- [ ] 涉及军事、文物、生态红线等特殊管控区域（视项目排查）
- [ ] 不动产权属纠纷（诉讼 / 仲裁 / 行政复议）——案件标的、进展、对权属的影响

### D. 权利受限

- [ ] 抵押：抵押权人、担保主债权、抵押期限、抵押比例
- [ ] 查封：司法查封 / 行政查封、原因、解除前景
- [ ] 冻结：案号、金额、解除可能性
- [ ] 优先权：建设工程价款优先受偿权等
- [ ] 租赁：涉及"买卖不破租赁"的承租人优先购买权

### E. 异地办公 / 生产基地

- [ ] 分公司 / 子公司所在地的不动产核查（合并纳入本 skill 范围）
- [ ] 注册地址 / 办公地址 / 生产经营地址的一致性说明

## 二、知识产权

### A. 专利

- [ ] 发明专利、实用新型、外观设计（分别核查）
- [ ] 专利证书、专利登记簿副本、年费缴费凭证
- [ ] 专利保护期：发明 20 年 / 实用新型和外观设计 10 年（自申请日起）
- [ ] 专利稳定性：是否被提起无效宣告、是否被复审、是否有授权后修改
- [ ] 专利权人与发行人名称一致性（若历史存在差异，是否完成变更登记）
- [ ] 职务发明权属：发明人、设计人的职务发明权属争议（重要技术人员入职离职时间线）
- [ ] 与科研院所 / 高校的联合研发专利的分配
- [ ] 专利是否处于质押、转让、许可状态
- [ ] 专利侵权：被诉侵权 / 提起侵权诉讼情况
- [ ] 核心专利的支撑作用（支撑主营业务的关键专利清单）

### B. 商标

- [ ] 注册商标证书、核定使用商品 / 服务类别、有效期（10 年 / 续展）
- [ ] 商标权人与发行人名称一致性
- [ ] 商标稳定性：撤三（连续三年未使用撤销）、无效宣告、异议程序
- [ ] 驰名商标 / 著名商标 / 地理标志（如涉及）
- [ ] 商标许可 / 转让 / 质押
- [ ] 近似商标抢注、商标异议、商标维权情况
- [ ] 商标覆盖类别与主营业务匹配（建议覆盖主业所在尼斯分类 + 防御性分类）

### C. 著作权 / 软件著作权

- [ ] 软件著作权登记证书、著作权人、开发完成日期
- [ ] 作品著作权登记证书（美术 / 文字 / 视听作品等，如适用）
- [ ] 职务作品权属：开发人员（员工）vs 发行人（法人）的权属安排
- [ ] 外包开发的著作权归属约定（合同条款是否明确归属发行人）
- [ ] 开源代码使用（GPL / MIT / BSD 等）的合规性和发行人自有代码的隔离
- [ ] 第三方软件许可（商业软件的 License 合规，涉及 Microsoft / Oracle / Adobe 等）

### D. 域名

- [ ] 主要域名清单（主域名 + 常见备用域名 + 核心产品域名）
- [ ] 域名注册商、注册人、注册日期、到期日期
- [ ] 域名注册人与发行人名称一致性
- [ ] 域名是否处于争议状态（UDRP / 仲裁）

### E. 商业秘密 / 非专利技术

- [ ] 重要技术秘密清单（配方、工艺、算法、模型、客户名单等）
- [ ] 保密制度（保密协议、竞业禁止协议与核心员工签订情况）
- [ ] 接触商业秘密的员工流动管理
- [ ] 与客户 / 供应商 / 合作方的保密条款
- [ ] 商业秘密被泄露 / 被侵犯的历史

### F. 特许经营权 / 行政授权

- [ ] 授予文件、有效期、授权范围
- [ ] 行政许可的持续维持条件
- [ ] 授权可转让性 / 续期可能性

### G. 知识产权瑕疵

- [ ] IP 权属登记在关联方 / 创始人个人名下未转回发行人（高风险）
- [ ] 核心技术来源于员工职务发明但未办理权属转移（历史遗留）
- [ ] 核心 IP 被提起无效 / 撤销 / 侵权诉讼
- [ ] 涉及开源软件使用的 License 违规风险

## 三、重大设备

### A. 权属与价值

- [ ] 重大设备清单（按账面价值排序，覆盖生产主设备 / 检测设备 / 运输设备 / IT 设备等）
- [ ] 采购合同、付款凭证、发票、入库单
- [ ] 权属登记（适用机动车 / 特种设备）：车辆登记证、特种设备使用登记证
- [ ] 权利人名称与发行人一致

### B. 特殊取得方式

- [ ] 融资租赁：未到期金额、承租人名下是否登记为所有权人、提前终止风险
- [ ] 经营租赁：租金、期限、续租
- [ ] 保留所有权买卖：已付款比例、风险转移节点
- [ ] 关联方出资设备：评估报告、权属转移完成情况

### C. 权利受限

- [ ] 设备抵押登记（动产抵押登记系统）
- [ ] 设备被查封、扣押
- [ ] 设备处于维修 / 闲置的比例

## 四、承租资产

### A. 租赁房屋

- [ ] 租赁合同（完整合同文本，含附件）
- [ ] 出租方的产权证明（避免"二房东"转租无权限）
- [ ] 租赁备案（住建 / 不动产登记机构）：备案证明
- [ ] 租金支付凭证
- [ ] 租赁期限是否覆盖发行人经营需要（至少覆盖上市后合理期间，通常建议 3 年以上）
- [ ] 装修投入折旧与租赁期匹配
- [ ] 分租 / 转租的合法性（需出租人同意）
- [ ] 关联方房屋租赁的公允性（交 `ecm-dd-related-party` 协同）

### B. 租赁土地 / 设备 / 其他

- [ ] 土地租赁合同（涉及集体土地 / 农村 / 海域等）的合法性
- [ ] 设备经营租赁合同的关键条款
- [ ] 其他重大租赁（商标 / 技术 / 特许使用权）

## 五、关键文件清单（客户需提供）

- [ ] 不动产权证书 / 历史房产证 + 土地使用证（全量）
- [ ] 规划许可 + 施工许可 + 竣工验收文件（重要建筑）
- [ ] 专利证书、专利登记簿副本、年费缴费凭证（全量）
- [ ] 商标注册证书（全量，含续展记录）
- [ ] 软件著作权证书、作品登记证书（核心 IP）
- [ ] 域名注册证书 / 注册商后台截图
- [ ] 保密协议 + 竞业禁止协议（核心员工签署样本）
- [ ] 重大设备采购合同 + 发票 + 付款凭证
- [ ] 融资租赁 / 经营租赁合同
- [ ] 租赁合同 + 租赁备案证明（承租方）
- [ ] 抵押登记 / 质押登记 / 查封通知书 / 解除证明
- [ ] 权属纠纷诉讼文件 + 判决书
- [ ] 资产评估报告（涉及资产出资、划转、关联交易时）
- [ ] 不动产登记中心 / 国家知识产权局查询记录

## 六、特别关注事项

- **集体土地 + 自建厂房**：是历史遗留常见问题，监管敏感。处理路径：办理集体经营性建设用地入市手续、
  土地性质变更、或搬迁到合法用地
- **划拨土地的出让金**：将划拨土地用于非公益性用途或转让、出资时，应补缴出让金并办理出让手续
- **职务发明的权属转移**：核心技术人员入职前已有的 IP，入职后是否以合理对价转让给发行人；
  离职后一段时间内的 IP 仍可能被认定为职务发明
- **开源软件合规**：GPL 类 copyleft 开源许可可能传染发行人自有代码，需专项评估
- **商标撤三**：连续三年未使用可被他人申请撤销；注意保留使用证据
- **外资企业不动产限制**：历史上部分外资企业持有特定类型不动产受限（2020 年后已放宽）
- **跨境 IP 安排**：红筹架构下的"IP 装在境外主体还是境内主体"对所得税、权属、VIE 有重大影响，
  与 `ecm-design:cross-border` 协同
- **历史遗留房产办证**：老国企改制、历史沿革复杂的发行人经常存在历史房产未办证，需逐栋评估
