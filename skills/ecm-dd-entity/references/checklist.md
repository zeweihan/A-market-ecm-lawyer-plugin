# ecm-dd-entity · 核查要点清单

## 一、工商存续状态

- [ ] 营业执照合法有效（统一社会信用代码、成立日期、营业期限、注册资本、法定代表人、住所、经营范围）
- [ ] 企业信用信息公示系统 / 企查查 / 国家企业信用信息公示系统的信息与营业执照一致
- [ ] 未列入经营异常名录（原因：年报异常、住所失联、公示信息隐瞒 / 虚假等）
- [ ] 未列入严重违法失信企业名单（黑名单）
- [ ] 未被吊销、注销、撤销、清算中
- [ ] 税务登记状态正常（非正常户、走逃失联户须特别关注）
- [ ] 海关、外汇、社保、公积金等涉及监管部门的合规状态（"多证合一"后可从公示信息 / 信用中国查询）

## 二、基本工商信息一致性

- [ ] 公司名称（工商、章程、申报文件、合同主体四者一致）
- [ ] 住所（注册地址、实际经营地）
- [ ] 法定代表人（最新工商记载与章程、聘任决议一致）
- [ ] 注册资本（认缴 / 实缴数额、币种、出资期限，2024 新法下认缴 5 年内缴足的约束）
- [ ] 营业期限（临近到期需要提前延期）
- [ ] 经营范围（覆盖报告期内全部主营业务；涉及前置许可的业务须有对应许可证）
- [ ] 股东信息（与股东名册 / 章程一致；历史变更与 `ecm-dd-history` 的审阅结果交叉）

## 三、特殊身份核查

- [ ] 外商投资企业：批准证书（2020 年前） / 外商投资信息报告（2020 年后）；行业是否在负面清单
- [ ] 上市公司控股子公司：关联方定义、同业竞争情形
- [ ] 国有企业：国有股东背景、国资监管层级
- [ ] 特殊行业基础资质（作为"主体身份"层面最低限度的许可，不涉及具体业务深度核查）：
  - 金融：金融许可证、支付业务许可证、小额贷款 / 融资担保 / 融资租赁的经营许可
  - 医疗：医疗机构执业许可证
  - 教育：办学许可证
  - 出版：出版经营许可证
  - 烟草：烟草专卖许可证
  - 危化品：危险化学品经营许可证
  - 建筑：建筑业企业资质
  - 电信增值：ICP 许可证 / EDI 许可证 / IDC 许可证 / CDN 许可证

## 四、《注册办法》第 10 条 · 消极条件（36 个月内）

- [ ] 未擅自公开或变相公开发行证券（回溯最近 36 个月）
- [ ] 未因违反法律、行政法规、规章或交易所业务规则被交易所公开谴责（近 36 个月）或其他纪律处分（近 12 个月）
- [ ] 未因涉嫌违法违规被证监会立案调查或司法机关立案侦查（截至 DD 时点尚未结论的）

## 五、关键文件清单

- [ ] 营业执照正副本（最新版）
- [ ] 工商登记档案（完整版：设立登记 + 全部变更登记）
- [ ] 近 3 年企业年报
- [ ] 企业信用信息公示报告（国家企业信用信息公示系统截图 / 下载件，DD 时点前 1 个月内有效）
- [ ] 第三方背景核查报告（企查查 / 天眼查 / 启信宝等）
- [ ] 近 3 年国地税合并后的税务登记状态证明
- [ ] 外商投资批准证书 / 信息报告（如适用）
- [ ] 特殊行业基础许可证复印件（如适用）
- [ ] 控股股东 / 实际控制人书面确认函（承诺 36 个月内无《注册办法》第 10 条消极情形）

## 六、特别关注

- **工商字段与实控路径**：同样的字段（如"住所"）同时出现在工商、章程、租赁合同中，任何不一致都要在 Memo 记载并建议整改
- **经营范围的前置许可匹配**：经营范围中的"许可经营项目"必须有对应的有效许可证；缺许可证又有该收入的，属于高风险
- **历次年报异常**：即使已移出经营异常名录，仍应在 Memo 中记载移出原因，便于招股书"发行人被列入经营异常名录情况"专项披露
- **外资身份变化**：报告期内是否发生外资→内资或相反的变更，若发生需专项披露
