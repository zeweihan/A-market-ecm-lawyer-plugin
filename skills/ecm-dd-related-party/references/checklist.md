# ecm-dd-related-party · 核查要点清单

本 skill 在项目中生成的 DD Memo 应照本清单逐条核查。法规依据以 `shared/regulations/` 下条文为准。

## 一、关联方范围的完整性

### A. 关联自然人

- [ ] 控股股东、实际控制人及其近亲属（配偶、父母、子女及其配偶、兄弟姐妹及其配偶、
      配偶的父母、配偶的兄弟姐妹、子女配偶的父母）
- [ ] 董事、监事、高级管理人员及其近亲属
- [ ] 持有发行人 5% 以上股份的自然人股东及其近亲属
- [ ] 过去 12 个月内曾为上述主体者（按会计准则和交易所规则认定期）

### B. 关联法人 / 其他组织

- [ ] 控股股东、实际控制人直接或间接控制的除发行人以外的其他法人 / 组织
- [ ] 被上述关联自然人直接或间接控制的法人 / 组织
- [ ] 对发行人或其子公司施加重大影响的其他主体（合营企业 / 联营企业 / 重大投资者）
- [ ] 董监高、5% 以上股东担任董事、高管的其他法人（潜在关联）
- [ ] 过去 12 个月内曾为上述主体者

### C. 认定方法与交叉核对

- [ ] 以企查查 / 天眼查 / 国家企业信用信息公示系统对实控人 / 董监高 / 主要股东做对外投资与任职穿透
- [ ] 比对发行人财务报表附注的"关联方及关联交易"披露是否完整
- [ ] 问询发行人核心管理层（CFO / 法总）确认无遗漏
- [ ] 要求控股股东、实控人、董监高、主要股东出具**关联方确认函**

## 二、关联交易核查

### A. 关联交易分类

- [ ] 经常性关联交易（销售 / 采购 / 接受或提供劳务 / 租赁 / 许可使用 / 技术开发）
- [ ] 偶发性关联交易（资产购买 / 出售、担保、资金拆借、债务债权重组、股权投资等）
- [ ] 按会计准则分类归集（关联销售、关联采购、关联资金、关联担保、关联其他）

### B. 公允性核查

- [ ] 交易定价依据（市场公允价 / 成本加成 / 第三方评估 / 第三方同类交易参考价）
- [ ] 关联交易价格与独立第三方交易价格对比
- [ ] 是否存在显著高于或低于市场价的非公允安排
- [ ] 涉及资产类关联交易：评估报告合规（具备资质的评估机构 + 符合评估准则）
- [ ] 涉及房地产类关联交易：土地、房产评估基准日、方法、结果合理
- [ ] 涉及股权类关联交易：标的公司估值方法、参数合理
- [ ] 是否提供第三方市场参考价 / 行业价格信息作为辅助依据

### C. 必要性核查

- [ ] 关联交易是否因业务需要而发生
- [ ] 是否存在无交易必要的"凑数"交易
- [ ] 是否可通过独立第三方替代（为何选择关联方）
- [ ] 长期或大额关联交易的战略合理性

### D. 决策程序

- [ ] 按章程和交易所上市规则确定的金额阈值 / 占比阈值判断应履行的决策程序
- [ ] 董事会审议：关联董事是否回避表决
- [ ] 股东（大）会审议：关联股东是否回避表决
- [ ] 独立董事事前认可意见 / 独立意见（涉及关联交易、对外担保、变更募投等）
- [ ] 审计委员会 / 关联交易控制委员会意见（如章程要求）
- [ ] 监管披露（上市公司在交易所临时公告、招股书专章披露）

### E. 占比与重大依赖

- [ ] 关联销售占营业收入比例（分年度）
- [ ] 关联采购占营业成本比例（分年度）
- [ ] 关联方合计交易金额占营业收入 / 总资产比例
- [ ] 是否构成对关联方的重大依赖——触发"业务独立性"问题（与 `ecm-dd-independence` 协同）
- [ ] 报告期末（或上市前）是否已清理非必要关联交易

### F. 非经营性资金占用与违规担保（IPO 红线）

- [ ] 报告期内是否存在控股股东、实控人及其关联方非经营性资金占用
- [ ] 报告期内是否存在为控股股东 / 实控人 / 关联方违规提供担保
- [ ] 若曾存在：清理时点、清理方式（还款 / 解除担保）、后续制度安排
- [ ] 清理后是否再次发生（监管高度敏感，零容忍）

## 三、同业竞争核查

### A. 竞争识别

- [ ] 列明控股股东、实控人及其控制的除发行人以外的其他企业
- [ ] 逐一比对每家关联企业的**实际业务**（不仅看经营范围，更看实际经营内容）
- [ ] 判断是否构成"相同"或"相似"业务
- [ ] 判断是否构成"重大不利影响"（《注册办法》第 12 条第 1 款）

### B. 重大不利影响判断维度

- [ ] 竞争双方业务的重叠程度
- [ ] 竞争方的业务规模（收入、利润、市场占有率）
- [ ] 竞争方的业务是否延伸至发行人的上下游
- [ ] 竞争是否限制发行人未来业务发展空间
- [ ] 竞争方是否与发行人客户 / 供应商有交集
- [ ] 板块规则：科创板 / 创业板 / 主板 / 北交所对同业竞争的具体尺度
- [ ] 审核实践案例中同类情形的处理结果（可调用 `ecm-research:case-search`）

### C. 解决方案

若存在同业竞争，核查以下解决方案的有效性：

- [ ] **注销**：竞争业务实体在申报前完成注销，核查注销登记 + 税务清算
- [ ] **转让**：转让给非关联第三方，核查定价公允性、款项支付、工商变更、后续无关联
- [ ] **置入**：注入发行人体内（涉及资产重组，需单独评估）
- [ ] **托管**：不推荐，监管对托管方案持审慎态度
- [ ] **业务分割**：产品 / 地域 / 客户维度的分工，核查协议的可执行性、是否存在交叉
- [ ] **承诺**：仅在"未来可能但当前未构成"的情况下作为辅助安排，不能作为主要解决方式

### D. 避免同业竞争承诺函

- [ ] 承诺主体：控股股东、实际控制人、持股 5% 以上股东（视情况）
- [ ] 承诺期限：覆盖锁定期 + 上市后合理期间（通常为"只要作为发行人控股股东 / 实控人"）
- [ ] 承诺范围：不直接或间接从事与发行人相同 / 相似业务；不投资从事该等业务的主体；
      若出现竞争机会优先提供给发行人
- [ ] 违约责任：明确违约赔偿机制
- [ ] 承诺函签署规范性：本人签字 / 法定代表人签字 + 盖章
- [ ] 承诺函的执行状况（报告期内是否已违反）

## 四、避免非必要关联交易承诺函

- [ ] 承诺内容：尽量减少、规范关联交易；若不可避免按市场公允价格签订
- [ ] 承诺主体：控股股东、实际控制人
- [ ] 承诺期限：覆盖上市后持续期
- [ ] 违约责任：明确违约赔偿

## 五、关联交易信息披露

### A. 招股说明书披露

- [ ] "关联交易"专章内容完整（关联方清单、认定依据、交易分类、金额、占比、决策、公允性）
- [ ] 重大关联交易的专项披露
- [ ] 同业竞争情况的专项披露
- [ ] 避免同业竞争 / 避免非必要关联交易承诺函的披露

### B. 上市公司定期报告披露（若发行人已为上市公司）

- [ ] 年度报告关联交易附注
- [ ] 半年度报告关联交易附注
- [ ] 重大关联交易的临时公告

## 六、关键文件清单（客户需提供）

- [ ] 关联方确认函（控股股东 / 实控人 / 董监高 / 主要股东签署）
- [ ] 关联方清单（含认定依据，加盖发行人公章）
- [ ] 报告期内所有关联交易协议
- [ ] 关联交易决策文件（董事会 / 股东（大）会决议、独董事前认可意见）
- [ ] 审计报告关联方附注
- [ ] 控股股东、实控人控制的其他企业清单（含业务范围、注册资本、是否在存续）
- [ ] 避免同业竞争承诺函 / 避免非必要关联交易承诺函
- [ ] 同业竞争解决方案相关文件（注销 / 转让 / 分割协议、款项支付凭证、工商变更文件）
- [ ] 关联交易定价公允性支持文件（第三方参考价 / 评估报告 / 询价函）
- [ ] 报告期内非经营性资金占用、违规担保核查证明

## 七、特别关注事项

- **IPO 红线**：控股股东、实控人非经营性资金占用和违规担保，一旦发生未清理即构成高风险，
  监管持零容忍态度
- **同业竞争的"相似"判断**：不能仅以"经营范围不完全相同"就认定不构成竞争，要看**实际业务**
- **关联方遗漏的连锁反应**：一旦漏列关联方，其相关交易则可能被认定为未披露的关联交易，
  构成信息披露违规
- **12 个月回溯期**：过去 12 个月内曾为关联方的主体，其期间交易仍按关联交易核查
- **股权投资型关联交易**：涉及发行人对关联方的投资或关联方对发行人的投资，应独立评估其
  合规性和商业合理性
- **2024 年新《公司法》第 140 条**：关联关系范围在新法下的细化（控股股东 / 实控人 /
  董监高 / 董监高近亲属以及近亲属直接或间接控制的企业）
- **上市公司关联交易监管**：上市公司发行股份购买资产、募集配套资金时，向关联方收购的监管
  尺度（24 月间接 / 36 月直接等）与本 skill 协同
