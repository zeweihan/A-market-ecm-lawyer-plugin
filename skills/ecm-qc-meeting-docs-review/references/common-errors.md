# 会议文件 — 常见错误库

---

## A 类 — 基础信息类

| 编号 | 典型错误 |
|---|---|
| A1 | 公司简称前后不一致 |
| A2 | 届次编号错（董事会 / 监事会 / 股东会届次） |
| A3 | 召开日期 / 时间 / 地点在通知 / 决议 / 记录不一致 |
| A4 | 召集人 / 主持人不一致 |
| A5 | 总股本 / 总表决权股份跨文件不一致 |
| A6 | 股权登记日跨文件不一致 |
| A7 | 跨年错（2025 年会议文件出现 2024 年字样） |

---

## B 类 — 议案结构类

| 编号 | 典型错误 |
|---|---|
| B1 | 议案名在通知 / 议案文本 / 决议 / 表决统计表 不一致 |
| B2 | 议案编号断号 / 重号 |
| B3 | 议案顺序颠倒 |
| B4 | 总议案 vs 子议案混淆 |
| B5 | 议案文本缺"决策依据"段（法规 / 章程引用） |
| B6 | 议案文本第四段未标表决方式 |

---

## C 类 — 特别决议类（**高频**）

| 编号 | 典型错误 |
|---|---|
| C1 | 应特别决议但未标注 |
| C2 | 标注用语不规范（未使用 `meeting-docs-format §三` 标准话术） |
| C3 | 误标非特别决议为特别决议 |
| C4 | 章程附件（三会议事规则）修订未标特别决议（深交所要求） |
| C5 | 特别决议议案 / 通知 / 决议 的标注不同步 |

---

## D 类 — 关联回避类

| 编号 | 典型错误 |
|---|---|
| D1 | 关联股东名单不全（漏隐性关联如亲属 / 前员工控制企业） |
| D2 | 关联回避未在议案 / 决议 / 表决统计中同步标注 |
| D3 | 关联回避后基数未扣除回避股东表决权股份 |
| D4 | 关联方自认但起草人误以为非关联（或反之） |

---

## E 类 — 中小投资者单独计票类（上市公司）

| 编号 | 典型错误 |
|---|---|
| E1 | 应单独计票的议案未标注（关联交易 / 重组 / 业绩补偿 / 章程修改等） |
| E2 | 单独计票的决议表述未并列"整体表决 + 中小投资者表决" |
| E3 | 中小投资者计票基数错（应为"中小投资者实际出席表决权股份"） |

---

## F 类 — 通知期限类（**高频；可能导致决议效力瑕疵**）

| 编号 | 典型错误 |
|---|---|
| F1 | 年度股东大会通知不足 20 日 |
| F2 | 临时股东大会通知不足 15 日 |
| F3 | 董事会通知不足章程约定 / 10 日 |
| F4 | 以"工作日"代替"日历日"计算（除非法规 / 章程明确写工作日） |
| F5 | 章程约定通知期更长，但按法定最低执行 |

---

## G 类 — 累积投票类

| 编号 | 典型错误 |
|---|---|
| G1 | 累积投票议案仍写"同意 / 反对 / 弃权" |
| G2 | 累积投票议案未列"当选人"结论 |
| G3 | 累积投票议案票数计算错（未乘以应选人数） |

---

## H 类 — 网络投票类（上市公司）

| 编号 | 典型错误 |
|---|---|
| H1 | 不涉及网络投票的会议保留"网络投票"表述（模板残留） |
| H2 | 网络投票系统地址张冠李戴（深交所用沪交所地址） |
| H3 | 网络投票时段超出交易所规则 |
| H4 | 决议未单独列示网络投票部分的股东数和股数 |

---

## I 类 — 会议记录 / 签到 / 签字类

| 编号 | 典型错误 |
|---|---|
| I1 | 会议记录缺监票人 / 唱票人签字（股东大会） |
| I2 | 缺见证律师签字（有见证业务时） |
| I3 | 签到表 "占总股本比例" 基数错 |
| I4 | 授权委托书议案清单与通知不一致 |
| I5 | 授权委托书未写"未勾选时的代理意愿" |
| I6 | 授权委托书未写"复印有效" |
| I7 | 决议 / 记录日期与签字页日期不一致 |

---

## 执行建议

优先扫描 **C 类（特别决议）+ D 类（关联回避）+ F 类（通知期限）+ E 类（中小投资者）** —— 这 4 类是最常触发效力瑕疵或监管关注的问题。
