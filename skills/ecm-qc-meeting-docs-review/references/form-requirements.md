# 会议文件形式要件审查清单

依据 `shared/templates/meeting-docs-format.md`、《公司法》（2023 修订、2024 施行）、《上市公司股东会规则》（2025 修订）、交易所业务规则、公司章程。

11 项形式要件：

---

## 审查项 1 — 通知期限（**硬校验**）

**要求**（`meeting-docs-format §2.3`）：

| 会议类型 | 最少通知期 | 法规依据 |
|---------|:---------:|---------|
| 上市公司股东大会（年会） | **20 日**（现场会议前） | 《股东会规则》 |
| 上市公司股东大会（临时） | **15 日**（现场会议前） | 同上 |
| 非上市股份公司股东大会 | 20 日 | 《公司法》§ 112 |
| 有限责任公司股东会 | 章程约定；无约定 15 日 | 《公司法》§ 60 |
| 董事会 | 章程约定；无约定 10 日 | 《公司法》§ 119 |

**核对**：
- 扫描通知的"出具日期"
- 扫描通知的"召开时间"
- 计算 会议日 - 通知日（按日历日，除非法规明确写"工作日"）
- 比对最少通知期

**偏差处理**：
- 不足：通知文件顶部打 ⚠️ + 【必改】【风险提示】批注指出具体天数，可能导致决议效力瑕疵
- 章程约定更严格：以章程为准，不足同样【必改】【风险提示】

---

## 审查项 2 — 会议通知文件结构（股东（大）会）

**要求**（`meeting-docs-format §2.1`）：6 段必备：

1. 一、本次会议召开的基本情况（时间 / 股权登记日 / 地点 / 召集人 / 召开方式 / 主持人）
2. 二、会议审议事项（议案清单，含特别决议 / 关联回避 / 中小投资者单独计票标注）
3. 三、出席会议对象（股东 / 董监高 / 见证律师）
4. 四、会议登记方法（登记时间 / 地点 / 方式 / 授权委托 / 其他）
5. 五、参加网络投票的具体操作流程（上市公司必备）
6. 六、联系方式（联系人 / 电话 / 邮箱 / 地址）

**常见错误**：
- 漏"股权登记日"（上市公司必备）
- 漏"网络投票操作流程"（上市公司）
- 漏"授权委托说明"（股份公司）

**修订方式**：【必改】批注。

---

## 审查项 3 — 会议通知文件结构（董事会 / 监事会）

**要求**（`meeting-docs-format §2.2`）：7 段简化版：

1. 会议时间
2. 会议地点
3. 会议方式（现场 / 通讯 / 视频）
4. 会议召集人
5. 议案清单
6. 会议资料发送方式及时间
7. 回复方式及截止时间

---

## 审查项 4 — 议案文本结构

**要求**（`meeting-docs-format §三`）：4 段必备：

1. 一、议案背景（事实背景、决策必要性）
2. 二、议案内容（具体方案）
3. 三、决策依据（公司章程条款、法律法规引用）
4. 四、本议案需经 xxx 审议批准；表决方式：{普通 / 特别决议}；{关联回避情况}；{中小投资者单独计票}

**常见错误**：
- 议案无"决策依据"段
- 议案第四段未标注表决方式（是否特别决议 / 关联回避）

---

## 审查项 5 — 会议决议结构

**要求**（`meeting-docs-format §四`）：

```
XX 公司 {YYYY 年第 N 次}（临时）股东（大）会 / {第 X 届第 Y 次} 董事会决议

{基本情况段：召开时间 / 应出席 / 实际出席 / 合法有效性认定}

会议审议通过 / 否决 / 搁置议案如下：

一、审议通过 / 否决《关于 xxx 的议案》
  表决结果：同意 xxx 股（占 xx%）、反对 xxx 股（占 xx%）、弃权 xxx 股（占 xx%）
  （特别决议：已获得 2/3 以上通过）
  （关联回避：xxx 回避表决）
  （中小投资者单独计票：整体 / 中小 分列）

二、……

本决议须报 {备案 / 登记机关} 备案 / 登记。

{公司盖章 + 日期}
```

**常见错误**：
- 基本情况段缺少"合法有效性认定"（"会议符合《公司法》《公司章程》规定，会议所作决议合法有效"）
- 表决结果未列"占比"
- 特别决议 / 关联回避 / 中小投资者标注缺失

---

## 审查项 6 — 会议记录结构

**要求**（`meeting-docs-format §五`）：

股东大会记录必备：
- 会议召开基本情况
- 列席人员
- 议程（主持人宣布会议开始 / 推选唱票人 / 宣读议案 / 投票 / 结果 / 结束）
- **五种签字**：记录人 / 主持人 / 监票人 / 唱票人 / 见证律师

董事会记录简化版（去掉唱票 / 监票）。

**常见错误**：
- 缺监票人 / 唱票人签字（股东（大）会）
- 缺"记录股东发言要点"（议程中）
- 缺见证律师签字（有见证业务时）

---

## 审查项 7 — 签到表列定义

**要求**（`meeting-docs-format §六`）：

股东（大）会签到表列：
`序号 | 股东姓名 / 名称 | 身份证 / 统一社会信用代码 | 持股数 | 占总股本比例 | 是否亲自出席 | 代理人 | 到场时间 | 签字`

签到表末尾：出席股东合计 / 代表股份合计 / 占总股本 / 监督人签字 / 日期

**常见错误**：
- 签到表"占总股本比例"使用了"占出席表决权股份"（基数错）
- 列数不全

---

## 审查项 8 — 授权委托书结构（股份公司 / 上市公司必备）

**要求**（`meeting-docs-format §七`）：

- 委托人信息
- 委托人持股数和比例
- 受托人信息
- **议案清单 + 同意 / 反对 / 弃权勾选**（议案清单必须与通知一致）
- 未打勾时的代理意愿说明
- 委托日期和有效期
- 复印有效声明

**常见错误**：
- 授权委托书的议案清单与通知不一致（M1 问题）
- 未写"复印有效"
- 未写"未打勾时按某意愿行使"

---

## 审查项 9 — 表决统计表列定义

**要求**：

```
议案编号 | 议案名 | 议案类型 | 总表决权股数 | 同意 | 占比 | 反对 | 占比 | 弃权 | 占比 | 表决结果
```

**常见错误**：
- 议案类型列缺失（普通决议 / 特别决议）
- 占比基数错（混用"占总股本" vs "占出席表决权股份"）
- 表决结果未列（"通过 / 未通过"）

---

## 审查项 10 — 代理投票计票细则（上市公司必备）

**要求**：
- 网络投票规则（投票账号 / 时段 / 默认值）
- 整体计票 vs 中小投资者单独计票
- 适用交易所规则（上交所 / 深交所 / 北交所）

**常见错误**：
- 上市公司会议文件集缺代理投票计票细则
- 交易所地址张冠李戴

---

## 审查项 11 — 字体 / 字号 / 日期格式

**要求**（`meeting-docs-format §八 / §九`）：
- 主标题：二号黑体加粗
- 正文：小四宋体
- 一级小节："一、"黑体；"（一）"黑体不加粗；"1、"宋体
- 日期：中文全格式"2026 年 4 月 24 日"
- 股数：带千分位
- 比例：两位小数

---

## 11 项勾对执行流程

先做硬校验（审查项 1 通知期限），再做结构完整性（2-10），字体字号（11）交 format-adjust 统一处理或【建议】批注。
