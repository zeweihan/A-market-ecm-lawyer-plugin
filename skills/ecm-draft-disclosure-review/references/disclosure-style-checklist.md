# 披露风格审查清单

本文件是 `ecm-draft-disclosure-review` 做 Step 4 "披露风格审查" 的明细条目清单。

---

## 一、夸大表述（禁用宣传语）

信披文件禁用无客观依据的宣传性措辞。典型违规词：

- "行业领先" / "行业第一" / "龙头"（除非有权威第三方数据支持）
- "独家" / "独有" / "唯一"（除非有专利 / 垄断性资质）
- "国际先进" / "国内领先" / "填补空白"（需专业鉴定证明）
- "最好" / "最佳" / "最优" / "最大"（绝对化用语）
- "顶尖" / "一流" / "优秀"（主观评价）
- "效果最佳" / "性能最优"
- "畅销产品" / "受欢迎"（需销售数据支撑且有定义）

**检查规则**：正则扫描上述关键词；命中后检查上下文是否有权威依据（专利号、第三方报告编号、政府发文）；无依据则标记为 ⚠️ "夸大表述"。

---

## 二、避重就轻（风险披露不充分）

识别风险披露段过于笼统、未具体说明的情形：

| 不合格表述 | 合格的表述 |
|-----------|---------|
| "公司存在一定的经营风险" | "公司报告期内对前五大客户的销售占比为 65%、72%、78%，客户集中度较高；若其中任一主要客户流失，可能对公司营收造成不低于 10% 的影响" |
| "公司面临一定的法律风险" | "公司与 X 公司的诉讼案件（案号：XXX）涉及金额 1,200 万元，若败诉，可能影响净利润约 700 万元（占最近一期净利润的 15%）" |
| "公司可能面临监管政策变化的影响" | "若国家发改委将本公司主营业务纳入 {产业政策目录} 的限制类，公司新增项目立项可能受阻，具体表现为 ..." |
| "公司部分资产存在权利受限情形" | "公司持有的 {具体地块}（不动产权证号：XXX）已抵押给 {银行名称}，担保公司 {借款合同编号} 项下的 5,000 万元借款，该笔借款将于 YYYY 年 MM 月到期" |

**检查规则**：识别风险因素章节的段落；检查每段是否包含（1）具体数字、（2）具体事件、（3）具体影响量化；缺一则标记为 ⚠️ "避重就轻"。

---

## 三、措辞模糊（信披禁用词）

以下措辞在信披文件中**禁用**或**严格限制使用**：

| 禁用 / 限用措辞 | 原因 | 建议替换 |
|--------------|------|--------|
| "基本符合 xx 规定" | 定性模糊（符合 or 不符合？） | 明确"符合"或"除 xxx 外符合"+ 具体瑕疵 |
| "大致遵循"、"基本做到" | 同上 | 明确状态 + 具体详情 |
| "相关规定" / "有关法律" | 未指明具体条款 | 明确具体法律名称 + 条款号 |
| "近年来"、"近期"、"最近" | 时间模糊 | 明确具体时间（YYYY 年 MM 月 / 报告期内 / 最近 N 年） |
| "一般情况下"、"通常" | 事实陈述应确定 | 改为陈述事实，或删除"一般""通常" |
| "应该"、"可能"、"或许" | 用于过去 / 现在时的事实陈述不当 | 改为确定表述（未来预测除外） |
| "据了解"、"据估计" | 非律师应有用语 | 改为事实陈述 + 来源；无法确认的不披露 |
| "xxx 前后" | 时间模糊 | 明确具体日期 |

**检查规则**：全文正则扫描；命中即标记。

---

## 四、结论与事实不符（逻辑一致性）

识别信披文件中**结论性表述**与**详情段**矛盾的情形：

典型模式：

1. "公司不存在重大诉讼"（结论） vs "公司诉讼案件涉及金额超 5,000 万元"（详情）→ 结论与事实矛盾
2. "公司股权清晰"（结论） vs "股东 A 持股 15% 目前尚未完成代持清理"（详情） → 结论与事实矛盾
3. "不存在违规担保"（结论） vs 担保清单中存在未经股东大会审议的担保 → 矛盾
4. "董监高变动符合程序"（结论） vs "监事 X 于 YYYY 年 MM 月辞任，辞任未经监事会审议"（详情） → 矛盾

**检查规则**：
- 扫描含"不存在"、"符合"、"不违反"、"未发生"等肯定性陈述
- 在同章节或交叉章节搜索与该陈述相关的具体事实
- 若具体事实与陈述定性冲突，标记为 🔴 "结论与事实不符"

---

## 五、未定义缩写

首次出现的缩写必须在**释义段**或**首次出现处的括号内**定义。

**检查规则**：
- 识别 2-6 个汉字的缩写（如"发行人"、"上市公司"、"保荐人"）
- 识别英文大写字母缩写（如 "NDA"、"MOU"）
- 在"释义"段 / 括号内定义 / 首次出现的上下文中搜索定义
- 未定义的标记为 ⚠️ "未定义缩写"

---

## 六、引用过时法规

识别信披文件引用的法律法规是否为**现行有效版本**：

**检查规则**：
- 扫描信披文件引用的所有法规（《xxx 法》第 yy 条）
- 在 `shared/regulations/` 下搜索对应法规
- 若本地 shared/regulations/ 无该法规，调用 `ecm-research:reg-search` 核验时效
- 若该法规已被修订且引用的是旧版本，标记为 ⚠️ "引用过时法规"
- 若该法规已废止，标记为 🔴 "引用已废止法规"

典型需关注的法规更新（2024-2026 年）：
- 《公司法》2023 年修订（2024 年 7 月 1 日施行）
- 《上市公司股东会规则》2025 年修订（原名《股东大会规则》）
- 《首次公开发行股票注册管理办法》2023 年（证监会令第 205 号）
- 《上市公司重大资产重组管理办法》2023 年修订（证监会令第 196 号）

## 七、数据 / 日期格式不规范

识别信披文件中不符合披露惯例的格式：

| 不规范 | 规范 |
|-------|------|
| "1234567 元" | "1,234,567 元"（千分位） |
| "50%" / "50.5%" 混用 | 统一为两位小数（"50.00%" / "50.50%"） |
| "2026.4.24" | "2026 年 4 月 24 日" 或 "2026-04-24" |
| "一千二百万元" | "1,200 万元" 或 "12,000,000 元"（金额一般用小写数字） |
| 金额无币种 | "1,000 万元" / "RMB 1,000 万元"（明确币种） |

**检查规则**：正则扫描；命中即标记 ⚠️ "格式不规范"。

---

## 八、关联关系披露不充分

识别信披文件关联交易 / 关联方段落是否充分披露：

- 关联方身份穿透是否到最终自然人 / 国资
- 关联交易价格的"定价依据"是否充分（市场公允价、评估价、成本加成）
- 是否披露决策程序（董事会 / 股东会审议、关联方回避）
- 关联交易金额 / 占比 / 占净利润比例是否披露
- 是否与工作报告"关联交易汇总表"完全对应

**检查规则**：对照工作报告的"关联交易汇总表"和"关联方清单表"逐行比对；缺失披露项标记为 🔴 "关联披露不充分"。

---

## 九、承诺事项披露不完整

识别信披文件中的承诺事项段是否完整：

- 承诺方身份（控股股东 / 实控人 / 董监高 / 核心技术人员）
- 承诺内容（股份锁定、减持限制、避免同业竞争、关联交易规范、业绩承诺、补偿安排等）
- 承诺期限
- 违约责任

**检查规则**：对照工作报告中的承诺事项清单；逐条核对是否已披露。

---

## 十、涉外 / 跨境披露遗漏

识别涉外项目（跨境并购、红筹、VIE、境外上市备案）是否披露：

- ODI / FDI 审批文件
- 外汇登记凭证
- 境外上市备案结果（证监会备案通知书）
- 反垄断申报 / 审查结论
- 国家安全审查结论（若触发）
- 数据出境合规性（若涉及个人信息出境）

**检查规则**：基于项目类型是否涉外判断；涉外但未披露上述项标记为 🔴 "涉外披露遗漏"。

---

## 十一、告知性披露不准确

一些常见的告知性披露需注意准确性：

- "发行人于 YYYY 年 MM 月 DD 日召开 XX 会议审议通过" —— 日期、届次必须精确
- "发行人控股股东为 XXX，实际控制人为 YYY" —— 与股东穿透资料一致
- "本次发行拟募集资金不超过 XXX 元" —— 金额与募投项目测算表一致
- "最近一期末净资产为 XXX 元" —— 数字与经审计财务报告一致

**检查规则**：对照同项目的其他基准文件（工作报告 / 审计报告 / 评估报告 / 招股书的不同章节）逐一比对。

---

## 十二、综合建议

disclosure-review 执行 Step 4 风格审查时，建议按以下顺序：

1. 先扫第一节（重大事项提示 / 风险因素） —— 披露风格最容易出问题
2. 再扫第四节（发行人基本情况） —— 事实性描述最多
3. 然后扫关联交易 / 同业竞争 / 诉讼仲裁 / 其他重要事项 —— 信披高频问题区
4. 最后扫附录（承诺函、委托书） —— 形式错误高发区

每一条问题按"🔴（必改） / ⚠️（建议改） / ℹ️（可选改）"三级标记，详见 SKILL.md Step 5 输出格式。
