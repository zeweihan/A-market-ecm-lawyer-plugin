# 常见议案文本模板

本文件列示 A 股拟 IPO 公司 / 上市公司常见的董事会、监事会、股东（大）会议案文本模板。每个模板含：背景、内容、决策依据、表决方式标注。实际使用时由 `ecm-draft-meeting-docs` 按用户提供的细节填充占位。

---

## 一、董事会议案模板

### 1.1 关于公司年度财务决算报告的议案

```
议案 1：关于 YYYY 年度财务决算报告的议案

各位董事：

公司 YYYY 年度财务决算报告已经 {会计师事务所} 审计，形成《{审计报告文号}》。
本议案拟提请董事会审议通过上述财务决算报告，并提交公司 YYYY 年度股东（大）会审议。

决策依据：《公司法》、《公司章程》第 {X} 条。
表决方式：普通决议。

特此议案，请审议。

{公司全称} 董事会
YYYY 年 MM 月 DD 日
```

### 1.2 关于年度利润分配预案的议案

```
议案 N：关于 YYYY 年度利润分配预案的议案

公司 YYYY 年度经审计的归属于母公司股东的净利润为 {xxx} 元，加上年初未分配利润
{xxx} 元，截至 YYYY 年 MM 月 DD 日，母公司可供分配利润为 {xxx} 元。

结合公司经营情况、发展规划及资金需求，拟定 YYYY 年度利润分配预案：
1. 每 10 股派发现金红利人民币 {x.xx} 元（含税）；
2. 合计派发现金红利 {xxx} 元；
3. 本次分配不以资本公积金转增股本；
4. 本次分配不送红股。

本预案拟提请董事会审议后，提请股东（大）会审议。

决策依据：《公司法》、《公司章程》第 {X} 条。
表决方式：普通决议。

特此议案，请审议。
```

### 1.3 关于申请首次公开发行股票并上市的议案

```
议案 N：关于申请首次公开发行人民币普通股（A 股）股票并在 {板块} 上市的议案

为满足公司持续发展的资金需求，推动公司长远发展，本公司拟申请首次公开发行
人民币普通股（A 股）股票并在上海证券交易所 / 深圳证券交易所 / 北京证券交易所
{主板 / 科创板 / 创业板 / 北交所} 上市。

发行方案要点：
1. 股票种类：人民币普通股（A 股）
2. 每股面值：人民币 1.00 元
3. 发行数量：不超过 {xxx} 万股，占发行后总股本的比例不低于 25%
4. 发行方式：{网下询价配售与网上定价发行相结合}
5. 发行对象：符合条件的询价对象及已开立 A 股账户的投资者
6. 价格确定方式：根据市场询价结果确定
7. 承销方式：主承销商包销
8. 发行决议有效期：自股东大会审议通过之日起 24 个月
9. 募集资金用途：{具体募投项目}

本议案拟提请董事会审议后，提请股东（大）会以**特别决议**方式审议。

决策依据：《公司法》、《证券法》、《首次公开发行股票注册管理办法》、《公司章程》。
表决方式：**特别决议（需 2/3 以上表决权股东通过）**。

特此议案，请审议。
```


## 三、股东（大）会议案模板

### 3.1 关于修改《公司章程》的议案（特别决议）

```
议案 N：关于修改《公司章程》的议案

为适应公司发展需要 / 符合《公司法》（2023 年修订）相关要求 / 配合本次 {IPO /
再融资} 的相关规定，拟对公司现行《公司章程》进行修订。主要修订内容如下：

| 条款编号 | 修订前 | 修订后 | 修订理由 |
|---------|-------|-------|---------|
| 第 X 条 | ... | ... | ... |
| ... | ... | ... | ... |

修订后的《公司章程》全文详见公司公告 / 附件。

决策依据：《公司法》、《上市公司章程指引》、《公司章程》。
表决方式：**特别决议（需 2/3 以上表决权股东通过）**。
{若为上市公司且涉及股东权益的条款修改：中小投资者单独计票议案。}

特此议案，请审议。
```

### 3.2 关于与 XX 公司签订《XX 协议》暨关联交易的议案（关联议案）

```
议案 N：关于与 XX 公司签订《XX 协议》暨关联交易的议案

一、关联交易背景

二、关联关系说明
{交易对方} 与公司 {关联关系描述，如持有公司股份比例、共同实际控制人等}，
根据《上海证券交易所股票上市规则》/《深圳证券交易所股票上市规则》的规定，
本次交易构成关联交易。

三、交易内容
{具体交易条款}

四、定价依据
{市场公允价 / 评估价 / 成本加成等，说明公允性}

五、影响分析
本次交易 {不/需} 经公司股东（大）会审议批准（依《公司章程》第 X 条）。
本次交易对公司的影响为：{影响分析}

六、审议程序
本议案已经公司 {届次} 董事会第 {X} 次会议审议通过，独立董事已就本次交易
出具事前认可意见和独立意见。关联董事 {姓名} 已回避表决。

**本议案需经股东（大）会审议，关联股东 {名单} 需回避表决。**

表决方式：普通决议，**关联股东回避**。
{若为上市公司：中小投资者单独计票议案。}

```

## 四、模板使用说明

1. meeting-docs 使用本模板时，用实际信息替换 `{xxx}` 占位；未提供的字段留 `{xxx}` 并在生成文件顶部标注"假设字段，需用户确认"
2. 模板的"决策依据"段需由 meeting-docs 运行时填入具体条款号（结合 `shared/regulations/` 中的公司法、证券法等）
3. 特别决议议案必须在议案末尾明确标注"**特别决议（需 2/3 以上表决权股东通过）**"
4. 关联议案必须明确列出"关联股东 {名单} 需回避表决"
5. 上市公司的中小投资者单独计票议案必须明确标注"**中小投资者单独计票议案**"

## 五、未列入模板的议案类型

以下议案类型较复杂、需项目定制，不提供标准模板，meeting-docs 运行时会主动追问用户：

- 重大资产重组议案（涉及标的资产、发行股份、业绩补偿等）
- 收购议案
- 员工持股计划议案
- 股权激励计划议案
- 对外投资 / 对外担保 / 对外提供财务资助议案（金额较大时需股东大会审议）
- 公司合并、分立、解散、清算议案

对于上述议案，meeting-docs 仅生成议案文本**框架**（背景 / 内容 / 决策依据 / 表决方式），具体内容由用户提供或由专项律师起草。
