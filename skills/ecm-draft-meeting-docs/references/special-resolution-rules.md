## 一、特别决议的识别

### 1.1 法律明确规定的特别决议（《公司法》2023 年修订）

依《公司法》需特别决议（通常要求 2/3 以上表决权股东通过）的事项：

| 事项 | 法律依据 | 表决比例要求 |
|------|---------|-------------|
| 公司合并 | 《公司法》第 219 条 | 2/3 以上 |
| 公司分立 | 《公司法》第 219 条 | 2/3 以上 |
| 公司解散 | 《公司法》第 217 条 | 2/3 以上 |
| 公司清算 | 《公司法》第 219 条 | 2/3 以上 |
| 变更公司形式 | 《公司法》第 219 条 | 2/3 以上 |
| 修改公司章程 | 《公司法》第 66 条（有限）/ 第 112 条（股份） | 2/3 以上 |
| 增加注册资本 | 《公司法》第 66 条 / 112 条 | 2/3 以上 |
| 减少注册资本 | 《公司法》第 66 条 / 112 条 | 2/3 以上 |
| 发行公司债券（股份公司） | 《公司法》第 112 条 | 2/3 以上 |
| 公司章程约定的其他特别事项 | 《公司章程》 | 按章程约定 |

### 1.2 监管规则规定的特别决议（证监会 / 交易所）

| 事项 | 法律依据 | 表决比例 |
|------|---------|---------|
| 首次公开发行股票并上市 | 《公司法》+ 《首次公开发行股票注册管理办法》 | 2/3 以上 |
| 上市公司再融资（配股 / 定增 / 发行可转债 / 发行优先股） | 《上市公司证券发行注册管理办法》 | 2/3 以上 |
| 重大资产重组 | 《上市公司重大资产重组管理办法》 | 2/3 以上 |
| 重大资产出售 | 同上 | 2/3 以上 |
| 发行股份购买资产 | 同上 | 2/3 以上 |
| 涉及管理层权益变动的重大股权投资 | 部分交易所规则 | 2/3 以上 |
| 回购股份（涉及实控人收购的特殊情形） | 《上市公司股份回购规则》 | 2/3 以上 |
| 股权激励计划 | 《上市公司股权激励管理办法》 | 2/3 以上 |
| 员工持股计划 | 证监会规范 | 2/3 以上 |

### 1.3 《公司章程》约定的特别决议

实务中常见的章程约定特别决议事项：

- 公司对外担保超过净资产一定比例
- 单笔对外投资超过净资产一定比例
- 聘任 / 解聘公司总经理（部分公司）
- 设立分公司 / 子公司（部分公司）
- 年度利润分配方案的重大变更

**meeting-docs 识别策略**：在 Step 1 追问时向用户问询"贵公司《公司章程》是否有额外的特别决议约定"；用户提供则按章程识别，用户未提供则仅按法律明确规定识别。

### 1.4 识别实现

```python
def is_special_resolution(motion_text, company_charter):
    # 法律特别决议模式匹配
    legal_triggers = [
        "合并", "分立", "解散", "清算", "变更公司形式",
        "修改公司章程", "章程修订", "修订章程",
        "增加注册资本", "减少注册资本", "增资", "减资",
        "发行公司债", "发行可转换债", "发行优先股",
        "首次公开发行", "IPO",
        "非公开发行", "定向增发", "定增",
        "配股", "重大资产重组", "发行股份购买资产",
        "股权激励", "员工持股计划",
    ]
    for trigger in legal_triggers:
        if trigger in motion_text:
            return True
    # 章程约定的特别决议
    if company_charter and has_charter_special_resolution(motion_text, company_charter):
        return True
    return False
```

## 二、关联股东 / 董事 回避识别

### 2.1 构成关联关系的情形（《上市公司关联交易实施指引》及交易所规则）

| 关联方类型 | 识别要点 |
|-----------|--------|
| 关联法人 | 公司控股股东、实际控制人及其控制的其他企业 |
| 关联法人 | 持有公司 5% 以上股份的股东 |
| 关联法人 | 与公司共同成立合资企业、共同投资的企业 |
| 关联自然人 | 公司董事、监事、高级管理人员 |
| 关联自然人 | 公司董监高的配偶、父母、子女及其配偶 |
| 关联自然人 | 控股股东、实际控制人的近亲属 |
| 关联方（回溯） | 过去 12 个月内或未来可预见的 12 个月内曾是 / 将是关联方的实体 |

### 2.2 需要回避的场景

| 议案情形 | 需回避的主体 |
|---------|------------|
| 关联股东参与的交易 | 该关联股东（含一致行动人） |
| 涉及公司对关联方的担保 | 该关联股东 |
| 涉及公司向关联方借款 / 提供资金 | 该关联股东 |
| 涉及选举关联董事的议案 | 与候选董事有关联关系的股东 |
| 涉及公司向关联方转让 / 购买资产 | 该关联股东 |
| 涉及关联董事自身利益的议案（董事会层面） | 该关联董事 |

### 2.3 董事会层面的关联回避

董事会审议关联交易议案时：
- 与关联方有关联关系的董事**必须**回避表决
- 关联董事回避后，若剩余董事人数不足法定 / 章程规定的开会人数，则该议案需**直接提交股东（大）会审议**
- 独立董事对关联交易议案**必须**出具事前认可意见和独立意见

### 2.4 识别策略

meeting-docs 运行时：
1. 从用户提供的议案内容识别交易对方
2. 与关联方清单对比（若用户提供）
3. 若无关联方清单，从 DD Memo（`02-07-关联交易与同业竞争/`）自动加载
4. 识别出的关联股东 / 董事在议案末尾标注"**关联股东 {名单} 需回避表决**"

## 三、中小投资者单独计票

### 3.1 上市公司需要单独计票的议案类型（按证监会《上市公司章程指引》及交易所规则）

| 议案类型 | 单独计票依据 |
|---------|------------|
| 关联交易议案 | 上交所上市规则、深交所上市规则、北交所规则 |
| 重大资产重组议案 | 《上市公司重大资产重组管理办法》 |
| 业绩补偿协议议案 | 重组相关规则 |
| 利润分配议案（涉及重大利润分配方案调整时） | 《上市公司监管指引第 3 号 - 现金分红》 |
| 股权激励议案 | 《上市公司股权激励管理办法》 |
| 员工持股计划议案 | 相关规则 |
| 选举独立董事议案 | 部分交易所规则 |
| 章程修改议案（涉及中小投资者权益时） | 《上市公司章程指引》 |
| 董事、监事薪酬议案（涉及重大变化时） | 公司治理准则 |

### 3.2 中小投资者的定义

- 除单独或合计持有公司 5% 以上股份的股东之外的其他股东
- 董事、监事、高级管理人员持有的股份**不计入**中小投资者

### 3.3 单独计票的呈现方式

在决议中同时列出：
- **整体表决结果**：同意 / 反对 / 弃权股份数及比例（含所有出席股东）
- **中小投资者表决结果**：同意 / 反对 / 弃权股份数及比例（仅中小投资者）

格式示例：
```
议案 N：关于 XX 事项的议案
整体表决结果：同意 XXX,XXX,XXX 股，占出席股东所持有表决权股份总数的 XX.XX%；
  反对 XXX,XXX 股，占 XX.XX%；弃权 XXX 股，占 XX.XX%。
中小投资者表决结果：同意 XXX,XXX 股，占出席中小投资者所持有表决权股份总数的 XX.XX%；
  反对 XXX,XXX 股，占 XX.XX%；弃权 XXX 股，占 XX.XX%。
表决结果：通过。
```

## 五、常见识别难点

1. **"重大资产重组" vs "一般资产出售"**：《上市公司重大资产重组管理办法》定义"重大"为资产总额 / 营业收入 / 净利润任一指标达到 50%+ 或金额超过 5,000 万元且超过净资产 50%；不达标的不是重大资产重组，不走特别决议。
2. **"关联交易" vs "日常经营"**：日常经营的采购销售若对方非关联方，不构成关联交易；若对方是关联方但金额较小（上市公司一般 120 万 / 2,000 万 / 3,000 万阈值），可能无需股东大会审议（董事会即可）。
3. **"独立董事事前认可" vs "表决回避"**：两者并存不矛盾——独立董事事前认可是出具独立意见，董事层面不回避；关联董事表决回避是表决阶段不投票。
4. **"中小投资者" 的范围**：具体到某次股东大会，中小投资者的 5% 阈值以**股权登记日**的持股数计算；临近日期的增减持不影响本次。
5. **"一致行动人" 的穿透**：关联股东回避时，与该股东签订一致行动协议的其他股东**也应一并回避**（按关联规则穿透）。
