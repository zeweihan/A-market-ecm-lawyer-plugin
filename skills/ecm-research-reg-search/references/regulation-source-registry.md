# 法规数据源登记表（Regulation Source Registry）

本文件登记 `ecm-research-reg-search` 在检索时会用到的**所有权威法规数据源**，按五级效力
层级分类：入口 URL、覆盖范围、可靠性等级、查询方式、已知坑。

---

## 一、法律层

### 1.1 国家法律法规数据库

- **入口 URL**：https://flk.npc.gov.cn
- **覆盖范围**：现行有效的全部法律、行政法规、监察法规、司法解释、地方性法规、部门规章
- **可靠性等级**：A（官方原始）
- **维护方**：全国人大常委会办公厅
- **查询方式**：
  - 关键词全文检索
  - 按制定机关、法规类别筛选
  - 按发布日期、施行日期排序
  - 每部法规都有**效力状态**标注和**历次修订记录**
- **优势**：
  - 条款级引用方便（每一条都有稳定 URL 锚点）
  - 版本对照清晰（修订前 / 修订后条款并列）
  - 效力状态权威（已废止的法规会显示"废止"并指向替代法规）
- **已知坑**：
  - 部分 2010 年以前的地方性法规收录不全
  - 改版后条款的稳定 URL 格式变化，但法规编号稳定
- **调用原则**：**法律、行政法规、司法解释、部门规章的首选 P0 源**

### 1.2 全国人大常委会官网

- **入口 URL**：http://www.npc.gov.cn
- **覆盖范围**：立法动态、法律草案、征求意见稿、立法工作报告
- **可靠性等级**：A
- **用途**：追踪**在途立法**（征求意见稿、草案一审/二审/三审），判断制度变动趋势
- **注意**：**征求意见稿不具备法律效力**，不得作为正式依据

### 1.3 中国政府网

- **入口 URL**：http://www.gov.cn → 政策
- **覆盖范围**：行政法规、国务院决定、国务院文件
- **可靠性等级**：A
- **调用原则**：查找"国务院决定"类非严格法律形式的文件时使用

## 二、部门规章层（证监会系统）

### 2.1 中国证监会规章库

- **入口 URL**：http://www.csrc.gov.cn → 信息公开 → 规章库
- **覆盖范围**：证监会发布的全部部门规章（"证监会令"系列）
- **可靠性等级**：A
- **查询方式**：
  - 按文号（证监会令第 X 号）
  - 按规章类别（发行类 / 上市监管类 / 信披类 / 市场类 / 会计类 / 稽查类……）
  - 按发布日期
- **配套**：证监会官网有**单独的"废止公告"**，更新比规章库稍快
- **已知坑**：
  - 改版后部分旧规章 PDF 链接失效，但规章名称和文号稳定——**引用时以文号为准**
  - 规章库与 flk.npc.gov.cn 偶有版本不同步，以**更晚发布日期的版本**为准

### 2.2 证监会规范性文件

- **入口 URL**：证监会官网 → 信息公开 → 部门规范性文件
- **覆盖范围**：非"证监会令"的规范性文件，包括公告、指引、办法（不带文号的）、通知
- **可靠性等级**：A
- **典型**：
  - 证监会公告（年份+序号格式）
  - 《监管规则适用指引——发行类》《——上市类》《——会计类》《——律师事务所类》等系列
  - 窗口指导通知、问答文件
- **调用原则**：在部门规章之外常被作为审核依据的"指引"、"问答"、"通知"均在此查找

### 2.3 证监会发行监管部 / 上市公司监管部 / 并购重组部的专项公告

- **入口**：证监会官网对应业务条线栏目
- **可靠性等级**：A
- **特点**：偶有不被归入"规章库"或"规范性文件"的**内部业务指引**，但对实务有约束力
- **调用原则**：研究具体业务审核口径时作为补充源

## 三、部门规章层（其他部委）

### 3.1 财政部

- **入口**：http://www.mof.gov.cn
- **ECM 相关重点**：
  - 企业会计准则（CAS 系列）及解释
  - 国有股份管理相关文件
  - 会计师事务所业务监管

### 3.2 国务院国资委

- **入口**：http://www.sasac.gov.cn
- **ECM 相关重点**：
  - 企业国有产权转让管理办法
  - 企业国有资产评估管理暂行办法
  - 国有股东标识管理
  - 中央企业境外投资监督管理办法

### 3.3 商务部

- **入口**：http://www.mofcom.gov.cn
- **ECM 相关重点**：
  - 外国投资者对上市公司战略投资管理办法
  - 境外投资管理办法（联合发改委发布）
  - 10 号文（关于外国投资者并购境内企业的规定，2006 + 2009 修订）

### 3.4 国家发改委

- **入口**：http://www.ndrc.gov.cn
- **ECM 相关重点**：
  - 企业境外投资管理办法
  - 外商投资准入特别管理措施（负面清单）
  - 产业结构调整指导目录

### 3.5 人民银行 / 外汇管理局

- **入口**：http://www.pbc.gov.cn / http://www.safe.gov.cn
- **ECM 相关重点**：
  - 外汇管理办法
  - 跨境证券投资相关（QFII / QDII / RQFII / 互联互通）

### 3.6 市场监督管理总局

- **入口**：http://www.samr.gov.cn
- **ECM 相关重点**：
  - 市场主体登记管理条例 / 注册资本登记管理制度配套规定
  - 经营者集中反垄断审查

## 四、规范性文件层

### 4.1 最高人民法院司法解释

- **入口**：http://www.court.gov.cn → 司法政策 → 司法解释
- **可靠性等级**：A
- **ECM 相关重点**：
  - 关于审理证券市场虚假陈述侵权民事赔偿案件的若干规定（2022 最新修订）
  - 关于适用《中华人民共和国公司法》若干问题的规定（一）至（五）
  - 关于适用《中华人民共和国民法典》有关担保制度的解释
  - 全国法院民商事审判工作会议纪要（"九民纪要"，2019，虽为会议纪要但有类司法解释效力）

### 4.2 最高人民检察院公布案例

- **入口**：http://www.spp.gov.cn
- **ECM 相关重点**：证券犯罪典型案例（虚假陈述、内幕交易、操纵市场、背信损害上市公司利益罪）

### 4.3 两委（上市委、重组委）的审议规则和程序规定

- **入口**：各交易所官网 + 证监会并购重组部
- **性质**：属于组织规则，指导审议流程

## 五、交易所业务规则层

### 5.1 上海证券交易所

- **入口 URL**：http://www.sse.com.cn → 规则专区
- **板块细分**：
  - 主板：http://www.sse.com.cn → 规则 → 主板业务规则
  - 科创板：http://www.sse.com.cn → 科创板 → 规则专区
  - 债券：http://bond.sse.com.cn
- **可靠性等级**：A
- **重点规则**（以**主板/科创板**为主）：
  - 上交所股票上市规则（主板，最新 2024 修订）
  - 科创板股票上市规则
  - 科创板股票发行上市审核规则
  - 上市委员会管理办法
  - 自律监管指引系列（第 1-15 号，覆盖各专题）
  - 交易规则、会员管理、信息披露规则等
- **已知坑**：
  - 规则修订频繁，每次修订都有"新旧条款对照表"，必看
  - 征求意见稿单独页面，不要混入现行规则

### 5.2 深圳证券交易所

- **入口 URL**：http://www.szse.cn → 业务规则
- **板块**：主板、创业板
- **可靠性等级**：A
- **重点规则**：
  - 深交所股票上市规则（主板）
  - 创业板股票上市规则
  - 创业板股票发行上市审核规则
  - 上市公司自律监管指引系列

### 5.3 北京证券交易所

- **入口 URL**：http://www.bse.cn → 规则专区
- **可靠性等级**：A
- **重点规则**：
  - 北交所股票上市规则
  - 向不特定合格投资者公开发行股票并上市审核规则
  - 上市公司持续监管办法
  - 股票交易规则

### 5.4 全国中小企业股份转让系统（新三板）

- **入口 URL**：http://www.neeq.com.cn → 业务规则
- **可靠性等级**：A
- **重点规则**：
  - 业务规则（试行）
  - 挂牌公司信息披露细则
  - 挂牌公司治理细则
  - 自律监管措施和纪律处分细则

## 六、商业数据库（B 级）

### 6.1 北大法宝

- **入口**：https://www.pkulaw.com
- **覆盖范围**：全面覆盖上述五级+地方性法规+司法解释+审判参考+学术评注
- **可靠性等级**：B（编辑加工，但源头可追溯）
- **特色功能**：
  - 新旧对照
  - 条文关联（法律 → 下位规章 → 司法解释 → 相关判例）
  - 效力状态实时更新
- **调用原则**：用户若订阅，可作为 P0 的**补充验证源**；未订阅不默认使用

### 6.2 威科先行 / 法信 / 万律

- 功能类似北大法宝

## 七、非权威来源（不作为直接引用）

以下数据源仅作为**线索发现**，不得作为正式引用：

- 百度文库、道客巴巴
- 个人公众号、非官方律所文章
- 社交媒体（微博、知乎）
- 新闻报道（可作背景，但具体条款以官方源核实）

**硬性规则**：若某部法规仅在以上来源出现而在官方数据库未查到，**不列入法规清单**，
在"3. 时效性与局限性"中说明"有线索但未能在官方源核实"。
