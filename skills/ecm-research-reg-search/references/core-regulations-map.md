# A 股 ECM 高频法规速查表（Core Regulations Map）

本文件是 `ecm-research-reg-search` 在常见 ECM 主题上的**高频法规清单速查**。用户检索
常见主题时，本表提供**第一轮命中**的法规候选（减少从头检索的工作量）；实际输出仍须经过
Phase 3 的时效性核验后列入正式清单。

> ⚠️ 本表维护日期：2026-04-24。法规持续更新，实际使用时必须通过 Phase 3 核验现行版本。

## 使用说明

本表作为**检索起点**而非最终清单：

1. 用户提出主题后，本 skill **先扫描本表**命中的候选法规
2. 对候选法规通过 Phase 3 核验**时效性**（可能有部分已废止 / 已修订）
3. 经核验后列入正式输出的"2.1 法规清单"
4. 本表未列入的法规（地方性法规、特定行业监管、新发布法规）需通过 Phase 2 的五级分层
   检索发现

本表**不涵盖**：
- 行业监管特殊规定（如医药、金融、游戏等行业的特殊准入）
- 地方性法规（涉及地方国企、地方产业时才会用到）
- 历史上已整体废止的旧规章（如需要历史研究，通过 `include_repealed=true` 配置追加）
