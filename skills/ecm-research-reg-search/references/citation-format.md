# 法规引证规范格式（Citation Format）

本文件规定 `ecm-research-reg-search` 及所有 `ecm-research-*` skill 在正文中引用法规时
应遵循的**规范引证格式**，供 Claude 在抽取和编写时对齐。

---

## 一、总体原则

1. **首次引用用全称**；后续同一文件可使用简称，但**首次出现**必须写全称
2. **附文号**：有文号的规章必须附文号（如证监会令第 X 号）
3. **附生效日期**：所有法规第一次被引用时，附"YYYY 年 M 月 D 日公布"+ "YYYY 年 M 月 D 日起施行"
4. **附效力状态**：紧跟法规后用方括号标注 [现行有效]、[已修订]、[已废止]
5. **引用具体条款**：必须精确到"第 X 条"，不得只说"该办法规定"
6. **原文引用加引号**：引用原文时用 `>` 引用块，不得自行转述后仍声称是原文

---

## 二、法律的引证

### 基础格式

```
《中华人民共和国公司法》（2023 年修订，2024 年 7 月 1 日起施行）[现行有效] 第 112 条
```

### 含章节的引用

```
《中华人民共和国证券法》（2019 年修订，2020 年 3 月 1 日起施行）[现行有效] 第一章 总则 第 9 条
```

### 多次修订的法律

```
《中华人民共和国公司法》（1993 年 12 月 29 日公布，历经 1999、2004、2005、2013、2018、2023
年修订，2024 年 7 月 1 日起施行最新版本）[现行有效] 第 112 条
```

如果引用的是**历史版本**（例如研究某一历史事件时），明确标注：

```
《中华人民共和国公司法（2018 年修订版）》[已被 2023 修订版替代] 第 97 条（对应现行公司法第 X 条）
```

## 八、条款原文引用

**原文必须用 `>` 引用块**。不得口述或转述后仍使用引号。

正确：
```markdown
《公司法》第 112 条 [现行有效]：
> 董事会会议应当由董事本人出席；董事因故不能出席，可以书面委托其他董事代为出席……
```

错误（不得将转述伪装为原文）：
```
《公司法》第 112 条规定"董事会要求本人出席"  ← 错误，这是转述不是原文
```

原文引用长度建议：
- 与论点直接相关的**核心条款**完整引用
- 长条款可节选，节选处用 `……` 替代
- 节选不得断章取义

## 九、脚注格式（用于意见书 / 工作报告）

当 `ecm-research-reg-search` 的输出被 `ecm-draft-opinion-letter` 或 `ecm-draft-report-assembly`
采用作为意见书脚注时，建议如下标准格式：

```
脚注格式示例：
¹ 参见《中华人民共和国证券法》（2019 年修订）第 9 条。
² 参见中国证监会《首次公开发行股票注册管理办法》（证监会令第 205 号，2023 年 2 月 17 日起
  施行）第 13 条第 1 款。
³ 参见上海证券交易所《上海证券交易所科创板股票上市规则》（2024 年修订）第 2.1.2 条。
```

|---------|--------|

## 十一、输出正文中的法规引证编号规则

本 skill 输出中，每条法规条目用 **[L-XX]** 编号（L = Legal/Regulation）。编号原则：

- 按效力层级从高到低连续编号（L-01, L-02, ...）
- 法律 → 行政法规 → 部门规章 → 规范性文件 → 交易所规则
- 同一层级内按发布日期从早到晚排序
- 编号供"1. 核心结论"段反向引用使用

示例：
```
## 1. 核心结论

1. 科创板发行上市的上位法由《证券法》[L-01] 提供 ...
2. 具体发行条件见《首发注册管理办法》[L-03] 第 13 条 ...
3. 科创板特殊要求见《科创板上市规则》[L-06] 第 2.1 条 ...

## 2.1 法规清单

### 2.1.1 法律

**[L-01]** 《中华人民共和国证券法》（2019 年修订）[现行有效]
...

**[L-02]** 《中华人民共和国公司法》（2023 年修订）[现行有效]
...

### 2.1.3 部门规章

**[L-03]** 《首次公开发行股票注册管理办法》（证监会令第 205 号）[现行有效] 第 13 条
...

### 2.1.5 交易所业务规则

**[L-06]** 《上海证券交易所科创板股票上市规则》（2024 年修订）[现行有效] 第 2.1 条
...
```
