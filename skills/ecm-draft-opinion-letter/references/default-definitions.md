# 法律意见书释义段 — 默认 15 条基础释义

`ecm-draft-opinion-letter` 起草"二、释义" 段时，若同项目律师工作报告未生成、未提供释义表，则以本清单为起点，再扫描 DD Memo 补充项目专有名词。

---

## 默认基础释义

```markdown
**本所** 指 {律所全称}

**本所律师** / **本所经办律师** 指 本所指派的为发行人本次{发行/交易}提供法律服务的专项律师

**发行人** / **{客户简称}** 指 {客户全称}

**本次发行** / **本次交易** 指 {发行人拟通过向中国证券监督管理委员会/上海/深圳/北京证券
交易所申请首次公开发行人民币普通股（A 股）并在{板块}上市之行为 / 本次并购重组 / 本次定增……}

**A 股** 指 在中华人民共和国境内（不包括香港特别行政区、澳门特别行政区和台湾地区）发行的、
供境内的机构和自然人以人民币认购和交易、以人民币标明面值、以人民币标明价格、
以人民币支付的股份

**中国证监会** / **证监会** 指 中国证券监督管理委员会

**交易所** 指 {上海证券交易所 / 深圳证券交易所 / 北京证券交易所}（按项目指定）

**保荐人** / **主承销商** 指 {具体机构全称}

**会计师** / **报告期** 指 {会计师事务所全称} / 指 最近三年零一期（{YYYY 年 - YYYY 年 及 YYYY 年 Q 期}）

**报告期末** 指 {YYYY 年 MM 月 DD 日}

**《公司法》** 指 《中华人民共和国公司法》（2023 年修订）

**《证券法》** 指 《中华人民共和国证券法》（2019 年修订）

**《注册办法》** 指 《首次公开发行股票注册管理办法》（中国证监会令第 205 号）

**《编报规则第 12 号》** 指 《公开发行证券公司信息披露的编报规则第 12 号——公开发行证券
的法律意见书和律师工作报告》

**元** 指 人民币元

**本所律师工作报告** 指 {律所全称}就本次{发行/交易}出具的《{律所全称}关于 {发行人}
{项目名}之律师工作报告》

**本法律意见书** 指 本文所指的《{律所全称}关于 {发行人} {项目名}之法律意见书》
```

## 补充释义建议（项目类型相关）

### 并购 / 重组项目

```markdown
**上市公司** / **收购人** 指 {上市公司全称}

**标的公司** / **被收购方** 指 {标的公司全称}

**业绩承诺期** 指 {具体期间}

**承诺补偿方** 指 {承诺方全称}
```

### 再融资项目

```markdown
**发行对象** 指 {符合条件的特定投资者 / 原股东 / 社会公众投资者}

**配股基准日** 指 {YYYY 年 MM 月 DD 日}

**募集资金** 指 发行人本次发行所募集的人民币资金
```

### 新三板挂牌

```markdown
**全国股转系统** 指 全国中小企业股份转让系统

**主办券商** 指 {主办券商全称}
```

## opinion-letter 运行时扩展规则

1. 本模板的占位符（如 `{律所全称}`）由 opinion-letter 运行时用项目元信息替换
2. opinion-letter 扫描整份意见书正文抽取以下类型的专有名词自动追加释义：
   - 以"XX 协议" / "XX 决议" / "XX 批复" 命名的文件
   - 以 2 个或 2 个以上汉字缩写的实体（如"北交所"、"上证所"）
   - 报告期内出现的主要关联方、主要客户、主要供应商
3. 自动追加的释义会在释义段末尾用注释标注"（本 skill 自动抽取，请律师复核）"
4. 释义条目数 < 15 或 > 35 时，opinion-letter 在首页编制说明中提示："释义条目数 {N}，低于 / 高于惯例区间（15-30），请复核"

## 常见释义错误

1. **"发行人"与"本公司"混用**：意见书中只用"发行人"，不用"本公司"（公司章程才用"本公司"）
2. **"本次发行"的范围不清**：补充说明发行品种、发行数量、发行对象
3. **报告期定义不准**：IPO 报告期为最近三年零一期；再融资一般为最近一年或最近一个完整会计年度；需依规严格界定
4. **法规释义用旧版**：《公司法》《证券法》《注册办法》须用最新生效版本；opinion-letter 启动时按 `shared/regulations/` 下的版本注入
