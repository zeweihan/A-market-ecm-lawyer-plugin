# 法律意见书 "法律意见" 小节 — 标准句式模板

本文件列示 `ecm-draft-opinion-letter` 起草"三、正文"各章节"法律意见"小节时的标准句式模板，分 **肯定 / 有条件肯定 / 保留** 三级。每级配套触发条件（基于 DD Memo 的"风险分级汇总"结果）。

---

## 一、结论三级与触发条件

| 级别 | 触发条件（依 DD Memo） | 意见书措辞模板 |
|------|-----------------------|--------------|
| **肯定（绿）** | 本章 Memo 风险汇总表**仅**有"低"级或无风险项 | "本所认为，……合法有效，符合……的规定" |
| **有条件肯定（黄）** | 本章 Memo 风险汇总表出现"中"级风险（无"高"级） | "除 {具体瑕疵} 外，本所认为，……合法有效；针对 {瑕疵}，本所建议 {整改措施}" |
| **保留 / 限定（红）** | 本章 Memo 风险汇总表出现"高"级风险 | "本所对……保留意见 / 暂无法发表肯定性意见，理由是 {具体原因}；{是否构成本次发行的实质性障碍}" |

> **硬性要求**：意见书结论的级别必须与 DD Memo 的最高级风险对应。若不一致，opinion-letter 在形式配套校验报告中起红旗，提示律师人工复核。

---

## 二、通用句式模板（各章节）

### 2.1 肯定结论

```
经核查，本所认为：

（1）{事项 1} {动作，如"经过了公司董事会、股东大会审议通过"}，符合《{具体法律}》第 {X} 条
的规定；
（2）{事项 2} {动作}，符合《{具体法律}》第 {Y} 条的规定；
……
（N）综上，本所认为，{章节主题} 合法有效，符合{相关法律法规}的有关规定。
```

### 2.2 有条件肯定结论

```
经核查，本所认为：

（1）{事项 1} {动作}，符合 {法律依据}；但 {具体瑕疵描述}；该瑕疵 {是否影响有效性}：
{瑕疵影响分析}；
（2）针对上述瑕疵，{具体整改措施或补救方案}已经 / 拟由 {整改主体} 采取，预计于
{时间节点} 完成 / 已完成；
（3）除上述瑕疵外，{章节主题} 合法有效，符合 {相关法律法规} 的有关规定。
```

### 2.3 保留 / 限定结论

```
经核查，本所律师注意到 {具体问题描述}。

（1）该事项的具体情况为：{详细事实描述}；
（2）法规依据：{具体法律条文引用}；
（3）本所暂时无法就 {事项} 发表肯定性意见，理由如下：
    ① {具体理由 1}；
    ② {具体理由 2}；
（4）上述事项 {是 / 否} 构成本次发行的实质性障碍；
（5）本所建议发行人 {具体整改 / 补救 / 补充披露措施}；
（6）在上述事项得到妥善解决之前，本所仅就 {章节主题的有效部分} 发表意见，对 {保留事项}
保留意见。
```

## 三、针对 17 章的分章节标准措辞

### 第一部分  关于本次发行的批准和授权（对应 ecm-dd-approval）

**肯定句式**：
```
综上，本所认为，发行人本次发行已经履行了《公司法》《证券法》《首次公开发行股票注册管理办法》
及发行人《公司章程》规定的必要内部决策程序，获得了本次发行所需的全部有效批准和授权，
本次发行的决议内容及授权范围合法有效。
```

**有条件肯定 / 保留句式**（示例）：
```
……发行人召集通知期限不足（股份公司临时股东大会应于会议前 15 日通知），该程序瑕疵
{已经/尚未} {通过 {补正措施} 整改}；{若已整改} 本所认为该瑕疵不构成本次发行的实质性障碍，
{若未整改} 本所建议发行人 {具体整改措施}。
```

### 第二部分  关于发行人的主体资格（对应 ecm-dd-entity）

**肯定句式**：
```
……发行人是依法设立并有效存续的股份有限公司，不存在《公司法》及《首次公开发行股票注册
管理办法》规定的终止情形，亦不存在《注册办法》第 10 条规定的不得公开发行股票并在
{板块} 上市的情形，具备本次发行的主体资格。
```

### 第三部分  关于发行人的设立（对应 ecm-dd-establishment）

**肯定句式**：
```
……发行人的设立（含整体变更）程序符合《公司法》的规定，发起人股东出资合法有效，
整体变更时的净资产折股符合会计准则，设立事项不存在重大历史遗留问题。
```

### 第四部分  关于发行人的独立性（对应 ecm-dd-independence）

**肯定句式**：
```
……发行人与控股股东、实际控制人及其控制的其他企业在人员、财务、业务、资产、机构方面
均保持独立，符合《注册办法》第 12 条规定的独立性要求。
```

**限定句式**（存在关联依赖时）：
```
……发行人在 {资产 / 业务 / 人员 / 财务 / 机构} 某一方面存在对 {关联方} 的依赖，
依赖程度为 {具体依赖比例或表现}。本所意见：该依赖 {是/否} 影响发行人的独立性判断；
建议发行人 {解决依赖的具体方案}。
```

### 第五部分  关于发行人的股东（含实际控制人）（对应 ecm-dd-shareholders）

**肯定句式**：
```
……发行人股东持股合法有效；实际控制人为 {姓名/企业名}，系 {单独控制 / 共同控制 /
一致行动控制}；最近 2 年实际控制人未发生变化；不存在影响发行人控制权稳定性的重大情形。
```

### 第六部分  关于发行人的股本及其演变（对应 ecm-dd-history）

**肯定句式**：
```
……发行人的股本及其演变过程中，历次增资、股权转让等股本变动行为合法有效；出资真实到位；
代持（如有）已完成清理；不存在影响本次发行的股本瑕疵。
```

### 第七部分  关于发行人的业务（对应 ecm-dd-business）

**肯定句式**：
```
……发行人的业务经营范围、业务资质、重大业务合同均符合法律法规的规定；前五大客户、供应商
集中度 {在合理范围内 / 符合行业惯例}；业务独立、完整、持续稳定。
```

### 第八部分  关于发行人的关联交易和同业竞争（对应 ecm-dd-related-party）

**肯定句式**：
```
……发行人报告期内发生的关联交易履行了必要的决策程序；关联交易定价公允，具备商业合理性；
发行人与控股股东、实际控制人及其控制的其他企业 {不存在同业竞争 / 已通过 {具体方案}
解决同业竞争}；不存在《注册办法》第 13 条禁止的情形。
```

### 第九部分  关于发行人的主要财产（对应 ecm-dd-assets）

**肯定句式**：
```
……发行人的主要财产（不动产、知识产权、重大设备、承租资产）权属清晰，取得合法，
{不存在权利受限情形 / 权利受限情形已如实披露}；不存在可能影响发行人持续经营或本次发行
的重大资产瑕疵。
```

### 第十部分  关于发行人的重大债权债务（对应 ecm-dd-debt）

**肯定句式**：
```
……发行人的重大债权债务合法有效；对外担保决策程序齐备；不存在控股股东、实际控制人
及其关联方违规占用发行人资金或违规担保的情形；不存在影响发行人持续经营能力的或有负债。
```

### 第十一部分  关于发行人的公司章程及组织机构（对应 ecm-dd-charter）

**肯定句式**：
```
……发行人的现行《公司章程》合法有效，内容符合《公司法》的强制性规定；三会（股东大会/
董事会/监事会或审计委员会）议事规则健全；内控制度完备；三会召开程序规范。
```

### 第十二部分  关于发行人的董事、监事及高级管理人员（对应 ecm-dd-directors）

**肯定句式**：
```
……发行人现任董事、监事、高级管理人员的任职资格均符合《公司法》第 146 条及证监会
相关规定的要求；不存在被市场禁入、最近 36 个月被中国证监会处罚、最近 12 个月被证券
交易所公开谴责的情形；报告期内董监高变动符合《公司章程》规定的程序。
```

### 第十三部分  关于发行人的税务（对应 ecm-dd-tax）

**肯定句式**：
```
……发行人按照中华人民共和国有关税收法律法规的规定履行了纳税义务，报告期内不存在被
税务机关重大处罚的情形，享受的税收优惠均合法有效。
```

### 第十四部分  关于发行人的环境保护、产品质量、安全生产（对应 ecm-dd-environment）

**肯定句式**：
```
……发行人的生产经营活动符合国家有关环境保护、产品质量、安全生产的法律法规的规定；
报告期内不存在因违反环保/产品质量/安全生产法律法规而被重大行政处罚的情形。
```

### 第十五部分  关于本次发行的募集资金运用（对应 ecm-dd-fundraising）

**肯定句式**：
```
……发行人本次发行募集资金拟投资的项目已履行必要的备案/核准程序（如涉及），募投项目
合法合规，符合国家产业政策和发行人主营业务发展方向；募投项目用地、用房等资产权属
清晰。
```

### 第十六部分  关于发行人的诉讼、仲裁或行政处罚（对应 ecm-dd-litigation）

**肯定句式**：
```
……发行人及其控股股东、实际控制人、董事、监事、高级管理人员报告期内不存在对发行人
有重大影响的未决诉讼、仲裁或行政处罚情形；或：发行人的 {具体未决诉讼/仲裁} 已如实
在《招股说明书》中披露，{是/否} 构成本次发行的实质性障碍。
```

### 第十七部分  关于发行人的其他合规事项（对应 ecm-dd-compliance）

**肯定句式**：
```
……发行人在社会保险、住房公积金、海关、外汇、劳动用工等其他合规事项方面，整体上
符合有关法律法规的规定，不存在构成本次发行的实质性障碍的合规瑕疵。
```

## 四、特别事项提示段 — 标准句式

按 `dd-output-schema.md` §5 规则，每条特别事项按以下句式组织：

```markdown
### 特别事项 {N}：{章节名}

经核查，发行人 {问题简述}。
法规依据：{法规条文引用}。
本所意见：{是否构成发行障碍；是否需整改 / 补充披露}。
```

"本所意见"的典型措辞：

| 情形 | 典型措辞 |
|------|---------|
| 不构成实质性障碍，已整改 | 本所认为上述事项已通过 {具体措施} 整改，不构成本次发行的实质性障碍 |
| 不构成实质性障碍，可通过披露解决 | 本所认为上述事项不构成本次发行的实质性障碍，建议发行人在《招股说明书》中充分披露 |
| 构成实质性障碍，需整改 | 本所认为上述事项构成本次发行的实质性障碍，发行人需在本次发行申报前 {具体整改措施} |
| 构成实质性障碍，目前无法解决 | 本所对上述事项保留意见；在上述事项得到妥善解决之前，本所对发行人本次发行的合规性暂无法发表肯定性意见 |

## 五、关于"经过了审议"的标准措辞细化

- **董事会审议**："本次发行已经发行人 {届次} 董事会第 {X} 次会议审议通过，审议程序符合《公司章程》第 {Y} 条的规定"
- **股东大会审议**："本次发行已经发行人 {YYYY 年第 N 次}（临时）股东大会审议通过，{同意/反对/弃权} 股份数占出席会议股东所持有表决权股份总数的 {XX.XX%}，符合《公司章程》第 {Y} 条的规定"
- **特别决议**："本次发行涉及的 {事项} 属于特别决议事项，已获得出席会议股东所持有表决权股份总数的 2/3 以上通过，{具体通过比例 XX.XX%}"
- **关联股东回避**："本次发行涉及关联交易，关联股东 {名单} 已回避表决；剔除关联股东后，同意 {xxx} 股，占 {XX.XX%}"
- **独立董事事前认可**："本次发行中涉及关联交易的议案已经独立董事事前认可；{独立董事姓名} 发表了同意 / 无异议的独立意见"

## 六、语言风格硬性要求（回顾）

- 第一人称统一为"**本所**"
- 禁用 **显然** / **明显** / **必然** / **毫无疑问** / **应当无疑**
- 结论句用 "**合法有效**" / "**符合……的规定**" / "**不存在需要披露的违法违规情形**"
- 保留 / 限定句用 "**除……外**" / "**在……前提下**" / "**暂无法发表肯定性意见**"（不用"基本符合"）
- 对事实的引用必须有精确出处（具体文件名 / 具体日期 / 具体条款号）
